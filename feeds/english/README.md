## 📁 Folders

| Folder | What It Covers | When to Use |
|:-------|:--------------|:------------|
| [technical/](./technical/) | Writing docs, commit messages, PR descriptions, code comments, README files | Before opening a PR or writing documentation |
| [professional/](./professional/) | Emails, Slack messages, meetings, presentations, feedback | Before sending a message to a client or manager |
