## бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ„бҹ’бһҒбҹҒбһ” (Summary)

бһҜбһҖбһҹбһ¶бһҡбһ“бҹҒбҹҮбһҖбҹҶбһҺбһҸбҹӢбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡ Jira бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһёбһ–бһёбһҜбһҖбһҹбһ¶бһҡ Functional Spec (FS) бһҡбһ”бһҹбҹӢ PO бһҡбһ бһјбһҸбһҠбһӣбҹӢ Deployed бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҗбһ¶ бһўбҹ’бһ“бһҖбһҺбһ¶бһҮбһ¶бһўбҹ’бһ“бһҖбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ (бһўбһ“бһ»бһҳбҹҗбһҸ) бһ“бҹ…бһҮбҹҶбһ бһ¶бһ“бһ“бһёбһҳбһҪбһҷбҹ—бҹ” бһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бһ¶бһҸбҹӢбһ‘бһ»бһҖбһҮбһ¶ **бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– (state machine)**бҹ– бһңбһ¶бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бҹ…бһҳбһ»бһҒбһҸбһ¶бһҳбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢ бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ“бһёбһҳбһҪбһҷбҹ—бһҳбһ¶бһ“бһҳбҹ’бһ…бһ¶бһҹбҹӢбһҸбҹӮбһҳбҹ’бһ“бһ¶бһҖбҹӢ бһ бһҫбһҷбһҳбһ¶бһ“бһҸбҹӮбһҳбҹ’бһ…бһ¶бһҹбҹӢбһ“бҹ„бҹҮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһҠбҹӮбһӣбһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ…бҹҒбһүбһ–бһёбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ“бҹ„бҹҮбһ”бһ¶бһ“бҹ” бһҡбһ¶бһӣбҹӢбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһўбһ“бһ»бһҳбҹҗбһҸ (sign-off) вҖ” бһҳбҹ’бһ…бһ¶бһҹбҹӢбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһ“бҹғбһ…бҹ’бһҡбһҖ (gate) бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹҶбһ–бҹҒбһү бһҳбһ»бһ“бһ“бһ№бһ„бһ”бҹ’бһҡбһӮбһӣбҹӢбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ‘бҹ…бһҳбҹ’бһ…бһ¶бһҹбҹӢбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ”

бһ“бһёбһҸбһ·бһңбһ·бһ’бһёбһ“бҹҒбҹҮбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһӣбһҫбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһӣбҹҶбһ бһјбһҡбһ‘бҹ…бһҳбһ»бһҒ бҹЁ бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ…бҹҶбһ бҹҖбһ„ бҹҘ (Fails, Paused, On Hold, To Discuss, Redo) бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– Pending Fix бһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡ FS бһҮбһ¶бһ’бһ¶бһҸбһ»бһ…бһјбһӣбһҠбҹҶбһ”бһјбһ„ вҖ” бһҹбһҡбһ»бһ”бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ бҹЎбҹҘ бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бҹ” бһңбһ¶бһҖбҹҶбһҺбһҸбҹӢбһҳбҹ’бһ…бһ¶бһҹбҹӢбһ“бҹғбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ“бһёбһҳбһҪбһҷбҹ— бһ–бҹ’бһҡбһҳбһ‘бһ¶бҹҶбһ„бһ…бҹ’бһ”бһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһҡбһ¶бһҮбҹҗбһҷ бһҖбһ¶бһҡбһ•бҹ’бһўбһ¶бһҖ бһ“бһ·бһ„бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһЎбһҫбһ„бһңбһ·бһүбҹ” бһҸбһ¶бһҡбһ¶бһ„бһ бһҸбҹ’бһҗбһӣбҹҒбһҒбһ¶бһ“бҹ…бһҒбһ¶бһ„бһ…бһ»бһ„ бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҖбһ¶бһҡбһ–бҹ’бһҡбһҳбһ–бҹ’бһҡбҹҖбһ„бһҡбһ”бһҹбҹӢбһҖбҹ’бһҡбһ»бһҳбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳбҹ”

бһҖбһ¶бһҡбһ„бһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҮбһ¶бһҳбһҪбһҷ **бһҜбһҖбһҹбһ¶бһҡ Functional Spec (FS)** бһҠбҹӮбһӣбһҹбһҡбһҹбҹҒбһҡ бһ“бһ·бһ„бһҮбһ¶бһҖбһҳбҹ’бһҳбһҹбһ·бһ‘бҹ’бһ’бһ·бһҡбһ”бһҹбҹӢ POбҹ” PO бһҮбһ¶бһҳбҹ’бһ…бһ¶бһҹбҹӢбһҸбҹӮ FS бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ вҖ” бһңбһ¶бһҖбҹҶбһҺбһҸбҹӢбһҗбһ¶ бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҹбһ¶бһ„бһҹбһ„бҹӢ бһ“бһ·бһ„бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбҹ” бһ–бһё FS бһҠбҹӮбһӣбһ”бһ¶бһ“бһўбһ“бһ»бһҳбҹҗбһҸ **PM** бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ…бһјбһӣбһ‘бҹ… **Backlogs** (Gate 0) бһ бһҫбһҷбһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһҳбҹ’бһ…бһ¶бһҹбҹӢбһ…бһ¶бһ”бҹӢбһ–бһёбһ–бҹҒбһӣбһ“бҹ„бҹҮбҹ” бһӮбҹ’бһҳбһ¶бһ“бһўбҹ’бһңбһёбһўбһ¶бһ…бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһӣбҹҶбһ бһјбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбһ”бһ¶бһ“бһЎбһҫбһҷ бһ”бһҫбһӮбҹ’бһҳбһ¶бһ“ FS бһҠбҹӮбһӣбһ”бһ¶бһ“бһўбһ“бһ»бһҳбҹҗбһҸбҹ”

**Gate 0 вҖ” FS Document вҶ’ Backlogs (PM, бһ–бһё FS бһҡбһ”бһҹбҹӢ PO)бҹ”** PO бһҹбһҡбһҹбҹҒбһҡ бһ“бһ·бһ„бһўбһ“бһ»бһҳбҹҗбһҸ Functional Spec вҖ” бһ“бҹҒбҹҮбһҮбһ¶бһҜбһҖбһҹбһ¶бһҡбһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢбһҠбҹӮбһӣ PO бһҮбһ¶бһҳбҹ’бһ…бһ¶бһҹбҹӢбһ“бҹ…бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһ“бҹҒбҹҮбҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ PM бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ–бһё FS бһҠбҹӮбһӣбһ”бһ¶бһ“бһўбһ“бһ»бһҳбҹҗбһҸ бһ…бһјбһӣбһ‘бҹ… Backlogs бһ бһҫбһҷбһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһҳбҹ’бһ…бһ¶бһҹбҹӢбһ…бһ¶бһ”бҹӢбһ–бһёбһ–бҹҒбһӣбһ“бҹ„бҹҮбҹ” бһ“бҹҒбҹҮбһҮбһ¶бһ…бҹ’бһҡбһҖбһ…бһјбһӣбһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢбһ“бҹғбһӣбҹҶбһ бһјбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбҹ” FS бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһҳбһ»бһ“бһҳбһҪбһҷ sprint вҖ” бһҠбһ¶бһҖбҹӢбһұбҹ’бһҷбһҡбһҪбһ…бһҸбҹ’бһҡбһ№бһҳ бһҗбҹ’бһ„бҹғбһ‘бһё бҹ§ (бһҗбҹ’бһ„бҹғбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ‘бһё бҹ§) бһ“бҹғ sprint бһҳбһ»бһ“ вҖ” бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҳбҹ’бһўбһ·бһҸ бһ“бһ·бһ„бһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһҳбһ»бһ“бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһ•бҹӮбһ“бһҖбһ¶бһҡ sprint бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ”

**Gate 1 вҖ” Backlogs вҶ’ Ready for Devs (PO)бҹ”** PO бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶бһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ”бҹҶбһ–бҹҒбһү Definition of Readyбҹ– бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ…бҹ’бһ”бһ¶бһҹбҹӢ бһ“бһ·бһ„бһўбһ¶бһ…бһҸбһ¶бһҳбһҠбһ¶бһ“бһ‘бҹ… FS бһ”бһ¶бһ“ бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһ‘бһ‘бһҪбһӣбһҷбһҖ (AC) бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҡбһҹбҹҒбһҡ бһ“бһ·бһ„бһўбһ¶бһ…бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ”бһ¶бһ“ бһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһңбһ¶бһҹбҹӢбһ‘бҹҶбһ бҹҶ бһ бһҫбһҷбһӮбҹ’бһҳбһ¶бһ“бһ—бһ¶бһ–бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһҺбһ¶бһҳбһҪбһҷбһҡбһ¶бһҡбһ¶бҹҶбһ„бҹ” бһӣбһ»бҹҮбһҸбҹ’бһҡбһ¶бһҸбҹӮбһ”бҹӮбһ”бһ“бҹҒбҹҮ бһ‘бһҫбһ”бһңбһ¶бһўбһ¶бһ…бһұбҹ’бһҷ Dev бһ‘бһ¶бһүбһҷбһҖбһ”бһ¶бһ“бҹ”

**Pull вҖ” Ready for Devs вҶ’ In Progress (PM вҶ’ Dev)бҹ”** PM бһҮбһ¶бһҳбҹ’бһ…бһ¶бһҹбҹӢ Ready for Devsбҹ– бһӮбһ¶бһҸбҹӢбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶бһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҡбһҪбһ…бһҡбһ¶бһӣбҹӢ бһ“бһ·бһ„бһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹ’бһҡбһ»бһҳбҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ Dev бһҳбҹ’бһ“бһ¶бһҖбҹӢбһ‘бһ‘бһҪбһӣбһҒбҹ’бһӣбһҪбһ“бһҜбһ„ бһ бһҫбһҷбһ‘бһ¶бһүбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһўбһ¶бһ‘бһ·бһ—бһ¶бһ–бһҖбҹҶбһ–бһјбһӣбҹ” бһ…бһ¶бһ”бҹӢбһ–бһёбһ–бҹҒбһӣбһ“бҹ„бҹҮ бһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҳбһ¶бһ“бһҳбҹ’бһ…бһ¶бһҹбҹӢбһҸбҹӮбһҳбҹ’бһ“бһ¶бһҖбҹӢбҹ” бһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢ WIP бһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ– Dev бһҳбһ·бһ“бһӮбһҪбһҡбһ‘бһ¶бһүбһҖбһ¶бһҡбһ„бһ¶бһҡбһҗбҹ’бһҳбһёбһҒбһҺбҹҲбһҠбҹӮбһӣбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһ…бһ”бҹӢбһЎбһҫбһҷбҹ”

**Gate 2 вҖ” In Progress вҶ’ To Reviews (Dev бһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡ)бҹ”** Dev бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶бһҖбһјбһҠбһ”бҹҶбһ–бҹҒбһүбһӮбҹ’бһҡбһ”бҹӢбһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһ‘бһ‘бһҪбһӣбһҷбһҖ unit test бһҮбһ¶бһ”бҹӢ бһ бһҫбһҷбһҖбһ¶бһҡбһ„бһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҠбҹ„бһҷбһҒбҹ’бһӣбһҪбһ“бһҜбһ„бҹ” бһӮбһ¶бһҸбҹӢбһ”бһҫбһҖ Pull Request бһҠбҹӮбһӣбһ•бҹ’бһӣбһ¶бһҹбҹӢбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ‘бҹ… To Reviewsбҹ” бһ…бһ¶бһ”бҹӢбһ–бһёбһ–бҹҒбһӣбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ (reviewer) бһҮбһ¶бһҳбҹ’бһ…бһ¶бһҹбҹӢбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡ бһҳбһ·бһ“бһҳбҹӮбһ“бһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбһ‘бҹҒбҹ”

**Gate 3 вҖ” To Reviews / In Reviews вҶ’ Pending Deploy (Peer Dev / бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ)бҹ”** бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ‘бһ‘бһҪбһӣбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ–бһё To Reviews бһ•бҹ’бһӣбһ¶бһҹбҹӢбһңбһ¶бһ‘бҹ… In Reviews бһ бһҫбһҷбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ PR бһӣбһҫбһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ—бһ¶бһ–бһ„бһ¶бһҷбһўбһ¶бһ“ бһ“бһ·бһ„бһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбҹ” CI бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһ”бҹғбһҸбһ„бҹ” бһ–бҹҒбһӣбһўбһ“бһ»бһҳбҹҗбһҸ бһ“бһ·бһ„ merge бһҡбһҪбһ… бһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бҹ… Pending Deployбҹ” бһ”бһҫбһҳбһ¶бһ“бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһҖбһ¶бһҡбһҖбҹӮбһ”бҹ’бһҡбҹӮ бһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбһҸбҹ’бһҡбһјбһңбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ; бһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҳбһ·бһ“бһӣбҹ„бһҸбһ‘бҹ…бһҳбһ»бһҒбһЎбһҫбһҷбҹ”

**Gate 4 вҖ” Pending Deploy вҶ’ Ready For Test (PM / DevOps)бҹ”** Build бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһӣбһҫбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“ staging/test бһ бһҫбһҷ smoke check бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ QA бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһҗбһ¶ build бһўбһ¶бһ…бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ”бһ¶бһ“ бһ бһҫбһҷбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бҹ… Ready For Testбҹ” бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҸбҹӮбһ„бһҸбҹӮбһ’бҹ’бһңбһҫбһӣбһҫ build бһҠбҹӮбһӣбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһұбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһҳбһ·бһ“бһҳбҹӮбһ“бһҳбһ»бһ“бһ–бҹҒбһӣбһҠбһ¶бһҖбҹӢбһЎбһҫбһҷбҹ”

> **бһҖбһ¶бһҡбһ–бһ“бҹ’бһҷбһ¶бһҡбһ–бҹҒбһӣбһ“бҹ… Pending Deployбҹ”** бһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҠбҹӮбһӣбһҮбһ¶бһ”бҹӢбһӮбһ¶бҹҶбһ„бһ“бҹ… Pending Deploy бһҡбһ¶бһҡбһ¶бҹҶбһ„ QA бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢ бһ“бһ·бһ„бһ”бһ“бҹ’бһҗбһҷбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҠбҹӮбһӣбһ“бҹ…бһҹбһӣбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһҮбһҪбһҹбһҮбһ»бһӣбһҳбһ»бһ“ sprint бһ…бһ”бҹӢбҹ” бһ”бһҫбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ“бҹ… Pending Deploy **бһӣбһҫбһҹбһ–бһё бҹЎ бһҗбҹ’бһ„бҹғбһ’бҹ’бһңбһҫбһҖбһ¶бһҡ** бһҳбҹ’бһ…бһ¶бһҹбҹӢ (PM / DevOps) бһҸбҹ’бһҡбһјбһңбһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбһҸбһ·бһ“бҹ…бһӣбһҫбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡ бһ–бһ“бҹ’бһҷбһӣбҹӢбһ–бһёбһ”бһүбҹ’бһ бһ¶бһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһҮбһҪбһ”бһ”бҹ’бһҡбһ‘бҹҮ бһ“бһ·бһ„бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҠбҹӮбһӣбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖ бһ бһҫбһҷ CC бһҠбһӣбҹӢбһҹбһҳбһ¶бһҮбһ·бһҖбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’ (QA бһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡ Dev бһ“бһ·бһ„ PO/PM) бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһ“бһ¶бһҠбһ№бһ„ бһ“бһ·бһ„бһўбһ¶бһ…бһҖбҹӮбһҸбһҳбҹ’бһҡбһјбһңбҹ” бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ“бҹҒбҹҮбһҠбҹӮбһҡ бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ…бҹҶбһ–бҹ„бҹҮбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҺбһ¶бһҳбһҪбһҷбһҠбҹӮбһӣбһҖбһ¶бһҡбһ–бһ“бҹ’бһҷбһ¶бһҡбһ–бҹҒбһӣбһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҳбһҪбһҷ бһҡбһ¶бһҡбһ¶бҹҶбһ„бһҖбһ¶бһҡбһ„бһ¶бһҡбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ”
>
> бһҳбһјбһӣбһ бҹҒбһҸбһ»бһ‘бһјбһ‘бҹ… бһ“бһ·бһ„бһўбҹ’бһңбһёбһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫбҹ–
> - **бһ”бһүбҹ’бһ бһ¶ Server / бһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бҹ”** бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“ staging бһҠбһ¶бһ…бҹӢ бһ¬ deploy pipeline бһ”бһҡбһ¶бһҮбҹҗбһҷбҹ” бһ”бһүбҹ’бһ…бҹҒбһүбһҳбһҸбһ·бһ–бһёбһҳбһјбһӣбһ бҹҒбһҸбһ» бһ“бһ·бһ„бһ–бҹҒбһӣбһҡбҹҶбһ–бһ№бһ„бһҮбһҪбһҹбһҮбһ»бһӣ; бһҖбһ»бҹҶбһ•бҹ’бһӣбһ¶бһҹбҹӢбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ‘бҹ… Ready For Test бһҡбһ бһјбһҸбһҠбһӣбҹӢ build бһҹбҹ’бһўбһ¶бһҸбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ (бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһӣбһҫбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһҒбһјбһ… бһҒбҹ’бһҮбҹҮбһҒбҹ’бһҮбһ¶бһҷбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶ QA)бҹ”
> - **бһ—бһ¶бһ–бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ•бҹ’бһҹбҹҒбһ„бҹ”** бһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ“бҹҒбҹҮбһҳбһ·бһ“бһўбһ¶бһ…бһҠбһ¶бһҖбҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ¬бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһұбҹ’бһҷбһҳбһ¶бһ“бһ“бҹҗбһҷбһ”бһ¶бһ“бһ‘бҹҒ бһҡбһ бһјбһҸбһҠбһӣбҹӢбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбҹ” **бһҖбһ»бҹҶ** бһ”бҹ’бһҡбһӮбһӣбҹӢбһңбһ¶бһ‘бҹ… QA бһ“бҹ…бһЎбһҫбһҷ вҖ” бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһңбһ¶бһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһҒбҹ’бһҮбҹҮбһҒбҹ’бһҮбһ¶бһҷбһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„ бһ–бҹ’бһҡбҹ„бҹҮбһңбһ¶бһ“бһ№бһ„бһ”бһҡбһ¶бһҮбҹҗбһҷ бһ¬бһҒбһјбһ…бһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸбһ–бҹҒбһӣбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ“бҹ„бҹҮбһҳбһҖбһҠбһӣбҹӢбҹ” бһ”бһүбҹ’бһ…бҹҒбһүбһҳбһҸбһ·бһҗбһ¶бһңбһ¶бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҺбһ¶ бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ–бһҪбһҖбһңбһ¶бһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶ бһ бһҫбһҷбһ‘бһ»бһҖбһңбһ¶бһ“бҹ… Pending Deploy (бһ¬бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бҹ… On Hold / To Discuss бһ”бһҫбһҖбһ¶бһҡбһҡбһ„бҹӢбһ…бһ¶бҹҶбһҷбһјбһҡ) бһҡбһ бһјбһҸбһҠбһӣбҹӢбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶бҹ”
>
> бһӮбҹ„бһӣбһҠбҹ…бһӮбһәбһҖбһ¶бһҡбһ–бһ¶бһҡбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһ”бһҹбҹӢ QAбҹ– бһ”бҹ’бһҡбһӮбһӣбҹӢбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ‘бҹ… Ready For Test бһҸбҹӮбһ“бҹ…бһ–бҹҒбһӣбһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһңбһ¶бһ“бһ№бһ„бһ•бҹ’бһҸбһӣбҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ бһ“бһ·бһ„бһўбһ¶бһ…бһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“бҹ”

**Gate 5 вҖ” Ready For Test вҶ’ Done бһ¬ Fails (QA)бҹ”** QA бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһҡбһҺбһёбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһӣбһҫ build бһҠбҹӮбһӣбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһұбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ бһҫбһҷбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһ‘бһ‘бһҪбһӣбһҷбһҖбҹ” бһ”бһҫбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҮбһ¶бһ”бҹӢ бһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҖбҹҶбһ бһ»бһҹ Sev-1/Sev-2 бһҠбҹӮбһӣбһ“бҹ…бһ”бһҫбһҖбһ…бҹҶбһ  бһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ‘бҹ… Doneбҹ” бһ”бһҫбһҳбһ¶бһ“бһўбҹ’бһңбһёбһ”бһҡбһ¶бһҮбҹҗбһҷ QA бһҡбһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһҖбһ¶бһҡбһ”бһҡбһ¶бһҮбҹҗбһҷбһҠбҹ„бһҷбһҠбһ¶бһҖбҹӢбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ‘бҹ… Fails вҖ” QA бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһҡбһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚ бһңбһ¶бһҳбһ·бһ“бһҹбһҳбҹ’бһҡбҹҒбһ…бһҗбһ¶бһўбҹ’бһңбһёбһҖбһҫбһҸбһЎбһҫбһ„бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһЎбһҫбһҷбҹ”

**Deploy вҖ” Done вҶ’ Deployed (PM)бҹ”** PM бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷ бһ•бҹӮбһ“бһҖбһ¶бһҡ rollback бһ“бһ·бһ„бһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһҠбһӣбҹӢбһ—бһ¶бһӮбһёбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’ бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҠбһ¶бһҖбҹӢбһ•бһӣбһ·бһҸбһ•бһӣбһұбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ… production бһ бһҫбһҷбһ’бҹ’бһңбһҫ smoke check бһҖбҹ’бһҡбҹ„бһҷбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ”

## бҹЈ. бһҳбҹүбһ¶бһ‘бҹ’бһҡбһёбһҹбһҳбҹ’бһ…бһ¶бһҹбҹӢ бһ“бһ·бһ„ RACI (Ownership & RACI Matrix)

бһҳбҹ’бһ…бһ¶бһҹбҹӢ (Owner) = бһҸбһҪбһ“бһ¶бһ‘бһёбһҸбҹӮбһҳбһҪбһҷбһҠбҹӮбһӣбһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ…бҹҒбһүбһ–бһёбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ“бҹҒбҹҮбҹ” бһўбҹ’бһ“бһҖбһҠбһ‘бҹғбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҳбһ·бһ“бһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢбһңбһ¶бһ”бһ¶бһ“бһЎбһҫбһҷбҹ” R = бһ‘бһ‘бһҪбһӣбһҒбһ»бһҹбһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫ (Responsible), A = бһ‘бһ‘бһҪбһӣбһҒбһ»бһҹбһҸбҹ’бһҡбһјбһңбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ/бһҳбҹ’бһ…бһ¶бһҹбҹӢ (Accountable), C = бһҸбҹ’бһҡбһјбһңбһ–бһ·бһӮбҹ’бһҡбҹ„бҹҮ (Consulted), I = бһҸбҹ’бһҡбһјбһңбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„ (Informed)бҹ”

бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ“бһёбһҳбһҪбһҷбҹ—бһҳбһ¶бһ“бһҳбҹ’бһ…бһ¶бһҹбҹӢбһ‘бһ‘бһҪбһӣбһҒбһ»бһҹбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҳбҹ’бһ“бһ¶бһҖбҹӢбһӮбһҸбҹӢбҹ” бһҮбһҪбһҡ В«бһҳбһ·бһ“бһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢВ» бһ”бһ„бҹ’бһ бһ¶бһүбһҸбһҪбһ“бһ¶бһ‘бһёбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бһ¶бһҳбһ•бҹ’бһӣбһ¶бһҹбҹӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ“бҹ„бҹҮ вҖ” бһ–бһҪбһҖбһӮбҹҒбһўбһ¶бһ…бһ”бһүбҹ’бһ…бҹҒбһүбһҳбһҸбһ· бһ¬бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Jira бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҳбҹ’бһҡбһ»бһ„бһ‘бһ»бһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹ’бһ…бһ¶бһҹбҹӢбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ”

## бҹӨ. бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ…бҹҶбһ бҹҖбһ„ (Side Statuses)

бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһҖбҹ’бһҡбҹ…бһӣбҹҶбһ бһјбһҡбһ‘бҹ…бһҳбһ»бһҒбҹ” бһҸбһҪбһ“бһ¶бһ‘бһёбһҺбһ¶бһҖбҹҸбһўбһ¶бһ…бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ…бһјбһӣбһ”бһ¶бһ“ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ¶бһ“бһҸбҹӮбһҳбҹ’бһ…бһ¶бһҹбҹӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбһ¶бһҷбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ бһҠбҹӮбһӣбһ•бҹ’бһӣбһ¶бһҹбҹӢбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҸбҹ’бһҡбһЎбһ”бҹӢбһ…бһјбһӣбһӣбҹҶбһ бһјбһҡбһңбһ·бһүбҹ” бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһҖбһҳбҹ’бһҳбһҹбһ·бһ‘бҹ’бһ’бһ· PO/PM; бһҖбһҡбһҺбһёбһӣбһҫбһҖбһӣбҹӮбһ„бһӮбһә Pending Fix бһ“бһ·бһ„ Paused бһҠбҹӮбһӣ Dev бһҮбһ¶бһҳбҹ’бһ…бһ¶бһҹбҹӢбҹ”

бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ Fails (бһҠбҹ„бһҷ Dev / PM)бҹ– бһҖбҹҶбһ бһ»бһҹбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ вҶ’ Pending Fix В· бһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһҒбһ»бһҹ вҶ’ бһҸбҹ’бһҡбһЎбһ”бҹӢбһ‘бҹ… Ready For Test В· бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶ вҶ’ To Discussбҹ”

Paused бһ’бҹҖбһ”бһ“бһ№бһ„ On Holdбҹ– Paused бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһҖбһ¶бһҡбһ„бһ¶бһҡбһ”бһ¶бһ“бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ бһҡбһҪбһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ»бһҖбһҠбҹ„бһҷбһҖбһ¶бһҡбһ”бҹ’бһҸбһјбһҡбһўбһ¶бһ‘бһ·бһ—бһ¶бһ– (Dev бһҡбһҖбҹ’бһҹбһ¶бһ—бһ¶бһ–бһҮбһ¶бһҳбҹ’бһ…бһ¶бһҹбҹӢ бһ бһҫбһҷбһ”бһ“бҹ’бһҸбһ–бҹҒбһӣ PM/PO бһҹбҹ’бһҸбһ¶бһҡбһўбһ¶бһ‘бһ·бһ—бһ¶бһ–)бҹ” On Hold бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ¶бһҡбһ¶бҹҶбһ„ бһ¬бһҖбҹҶбһ–бһ»бһ„бһҡбһ„бҹӢбһ…бһ¶бҹҶбһўбҹ’бһңбһёбһҳбһҪбһҷбһ–бһёбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ… (PO/PM бһҮбһ¶бһҳбҹ’бһ…бһ¶бһҹбҹӢ)бҹ” бһ”бҹ’бһҡбһҫ Paused бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ В«бһҷбһҫбһ„бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ’бҹ’бһңбһҫбһўбҹ’бһңбһёбһҠбҹӮбһӣбһ”бһ“бҹ’бһ‘бһ¶бһ“бҹӢбһҮбһ¶бһ„бһҹбһ·бһ“В» бһ”бҹ’бһҡбһҫ On Hold бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ В«бһңбһ¶бһҳбһ·бһ“бһўбһ¶бһ…бһ”бһ“бҹ’бһҸбһ–бҹҒбһӣбһ“бҹҒбҹҮбһ”бһ¶бһ“В»бҹ”

## бҹҘ. бһ…бҹ’бһ”бһ¶бһ”бҹӢбһ“бҹғбһҖбһ¶бһҡбһўбһ“бһ»бһҳбҹҗбһҸ (Sign-Off Rules)

1. **бһҳбҹ’бһ…бһ¶бһҹбҹӢбһҸбҹӮбһҳбҹ’бһ“бһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бҹ”** бһ“бҹ…бһ–бҹҒбһӣбһҺбһ¶бһҳбһҪбһҷ бһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҳбһ¶бһ“бһҸбһҪбһ“бһ¶бһ‘бһёбһ‘бһ‘бһҪбһӣбһҒбһ»бһҹбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҳбҹ’бһ“бһ¶бһҖбҹӢ; бһҳбһ·бһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бһ–бһёбһҡбһ“бһ¶бһҖбҹӢбһҮбһ¶бһҳбҹ’бһ…бһ¶бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷбһЎбһҫбһҷбҹ”
2. **бһҳбһ¶бһ“бһҸбҹӮбһҳбҹ’бһ…бһ¶бһҹбҹӢбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢбһңбһ¶бҹ”** бһҸбһҪбһ“бһ¶бһ‘бһёбһҠбһ‘бҹғбһўбһ¶бһ…бһ”бһүбҹ’бһ…бҹҒбһүбһҳбһҸбһ· бһ¬бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҳбҹ’бһҡбһ»бһ„бһ‘бһ»бһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹ’бһ…бһ¶бһҹбҹӢ вҖ” бһ‘бҹ„бҹҮбһ”бһёбһҳбһ¶бһ“бһҹбһ·бһ‘бҹ’бһ’бһ· admin бһҖбҹҸбһҠбҹ„бһҷбҹ”
3. **бһ‘бҹ…бһҳбһ»бһҒбһҸбһ¶бһҳбһ…бҹ’бһҡбһҖбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ”** бһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһӣбҹ„бһҸ (бһ§. In Progress бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ‘бҹ… Done) бһЎбһҫбһҷбҹ” бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһҗбһҷбһҖбҹ’бһҡбҹ„бһҷбһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢбһӮбһә QA бһҡбһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚ Fails (бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ Dev/PM бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ) бһ“бһ·бһ„ Redo бһҠбҹӮбһӣбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһҸбҹ’бһҡбһјбһңбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бҹ”
4. **бһҖбһ¶бһҡбһўбһ“бһ»бһҳбҹҗбһҸбһҸбҹ’бһҡбһјбһңбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбҹ”** бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһҳбҹҗбһҸ; бһҳбҹ’бһ…бһ¶бһҹбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбһҸбһ· Jira бһҒбҹ’бһӣбһёбҹ— бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶бһ…бҹ’бһҡбһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹҶбһ–бҹҒбһүбҹ”
5. **бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ…бҹҶбһ бҹҖбһ„бһҮбһ¶бһҖбһҳбҹ’бһҳбһҹбһ·бһ‘бҹ’бһ’бһ· PO/PM** (бһӣбһҫбһҖбһӣбҹӮбһ„ Pending Fix бһ“бһ·бһ„ Paused бһҠбҹӮбһӣ Dev бһҮбһ¶бһҳбҹ’бһ…бһ¶бһҹбҹӢ)бҹ” бһўбҹ’бһ“бһҖбһҺбһ¶бһҖбҹҸбһўбһ¶бһ…бһҹбҹ’бһ“бһҫ; бһҳбһ¶бһ“бһҸбҹӮбһҳбҹ’бһ…бһ¶бһҹбҹӢбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһҸбҹ’бһҡбһЎбһ”бҹӢбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ…бһјбһӣбһӣбҹҶбһ бһјбһҡбһңбһ·бһүбҹ”
6. **бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҶбҹ’бһӣбһ»бҹҮбһ”бһүбҹ’бһ…бһ¶бҹҶбһ„бһҖбһ¶бһҡбһ–бһ·бһҸбҹ”** бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– Jira бһҮбһ¶бһ‘бһёбһҸбһ¶бҹҶбһ„бһ–бһ·бһҸбһ“бҹғбһҖбһ¶бһҡбһ„бһ¶бһҡ; бһ”бһҫбһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ¶бһҡбһ¶бҹҶбһ„ бһ•бҹ’бһӣбһ¶бһҹбҹӢбһңбһ¶бһ‘бҹ… On Hold бһ¬бһҠбһ¶бһҖбҹӢбһҹбһүбҹ’бһүбһ¶бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбҹ”
7. **бһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһ‘бһ¶бһүбһҗбһҷбһҖбҹ’бһҡбҹ„бһҷбһҠбҹ„бһҷбһҹбҹ’бһ„бһ¶бһҸбҹӢбҹ—бҹ”** бһҡбһ¶бһӣбҹӢбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһҗбһҷбһҖбҹ’бһҡбҹ„бһҷ бһ¬бһ…бҹҶбһ бҹҖбһ„ бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶ бһ“бһ·бһ„бһҳбһҸбһ·бҹ”
8. **бһ…бҹ’бһ”бһ¶бһ”бҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҖбҹӢбһҖбҹ’бһ“бһ»бһ„бһўбҹҶбһЎбһ»бһ„ sprintбҹ”** бһӮбҹ’бһҳбһ¶бһ“бһ“бһҡбһҺбһ¶бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ“бһёбһҸбһ·бһңбһ·бһ’бһёбһ“бҹҒбҹҮбһ“бҹ…бһ–бһ¶бһҖбҹӢбһҖбһҺбҹ’бһҸбһ¶бһӣ sprint бһЎбһҫбһҷ; бһ”бһүбҹ’бһ бһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶ бһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһ“бҹ… retrospective бһ бһҫбһҷбһҖбһ¶бһҡбһҖбҹӮбһҸбһҳбҹ’бһҡбһјбһңбһ…бһјбһӣбһҮбһ¶бһ’бһҡбһҳбһ¶бһ“бһ“бҹ… sprint бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ (бһҹбһјбһҳбһҳбһҫбһӣ бһҖбһҡбһҺбһёбһӣбһҫбһҖбһӣбҹӮбһ„ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһјбһ“бһ”бһ“бҹ’бһҸ)бҹ”
9. **бһҠбһ¶бһҖбҹӢбһҹбһүбҹ’бһүбһ¶бһҖбһ¶бһҡбһ–бһ“бҹ’бһҷбһ¶бһҡбһ–бҹҒбһӣбһҠбҹӮбһӣбһҡбһ¶бһҡбһ¶бҹҶбһ„бһўбҹ’бһ“бһҖбһҠбһ‘бҹғбҹ”** бһ”бһҫбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҺбһ¶бһҳбһҪбһҷбһӣбһҫбһҹбһ–бһё бҹЎ бһҗбҹ’бһ„бҹғбһ’бҹ’бһңбһҫбһҖбһ¶бһҡ бһ бһҫбһҷбһҖбһ¶бһҡбһ–бһ“бҹ’бһҷбһ¶бһҡбһ“бҹ„бҹҮбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһҖбһ¶бһҡбһ„бһ¶бһҡбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ (бһ§. Pending Deploy бһҷбһәбһҸ бһҡбһ¶бһҡбһ¶бҹҶбһ„ QA) бһҳбҹ’бһ…бһ¶бһҹбҹӢбһҸбҹ’бһҡбһјбһңбһ”бһүбҹ’бһ…бҹҒбһүбһҳбһҸбһ·бһ“бҹ…бһӣбһҫбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡ бһ–бһ“бҹ’бһҷбһӣбҹӢбһ–бһёбһ”бһүбҹ’бһ бһ¶ бһ бһҫбһҷ CC бһҠбһӣбҹӢбһҹбһҳбһ¶бһҮбһ·бһҖбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’ бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһ“бһ¶бһҠбһ№бһ„бҹ” бһ§бһ”бһҹбһӮбҹ’бһӮбһҠбҹӮбһӣбһ”бһҫбһҖбһ…бҹҶбһ бһӣбһҫбһҹбһ–бһё бҹЎ бһҗбҹ’бһ„бҹғбһ’бҹ’бһңбһҫбһҖбһ¶бһҡ бһҸбҹ’бһҡбһјбһңбһӣбһҫбһҖбһЎбһҫбһ„бһ“бҹ… Daily Standup бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ•бһ„бһҠбҹӮбһҡбҹ”

## бҹҰ. бһҖбһҡбһҺбһёбһӣбһҫбһҖбһӣбҹӮбһ„ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһјбһ“бһ”бһ“бҹ’бһҸ (Exceptions & Escalation)

бһӮбҹ’бһҳбһ¶бһ“бһ“бһёбһҸбһ·бһңбһ·бһ’бһёбһҺбһ¶бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһӮбҹ’бһҡбһ”бҹӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһЎбһҫбһҷбҹ” бһ–бҹҒбһӣбһӣбҹҶбһ бһјбһҡбһ’бһҳбҹ’бһҳбһҸбһ¶бһҳбһ·бһ“бһўбһ¶бһ…бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһ¶бһ“бҹ–

- **бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҸбһ¶бһҳбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһўбһҸбһ·бһҗбһ·бһҮбһ“ (бһҹбһ·бһ‘бҹ’бһ’бһ· PO/PM)бҹ”** бһҠбҹ„бһҷбһҹбһ¶бһҡ PO/PM бһ’бҹ’бһңбһҫбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҮбһ¶бһҳбһҪбһҷбһўбһҸбһ·бһҗбһ·бһҮбһ“ бһ–бһҪбһҖбһӮбҹҒбһўбһ¶бһ…бһ‘бһ¶бһүбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡ **бһҺбһ¶бһҳбһҪбһҷ** вҖ” бһҠбҹ„бһҷбһҳбһ·бһ“бһӮбһ·бһҸбһ–бһёбһҳбҹ’бһ…бһ¶бһҹбҹӢбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ вҖ” бһ…бһјбһӣбһ‘бҹ…бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ…бҹҶбһ бҹҖбһ„ (On Hold, To Discuss, Paused, бһ¬ Redo) бһ–бҹҒбһӣбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһўбһҸбһ·бһҗбһ·бһҮбһ“бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбҹ” бһ–бһҪбһҖбһӮбҹҒбһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫбһҠбҹ„бһҷбһ”бҹ’бһҡбһ»бһ„бһ”бҹ’бһҡбһҷбҹҗбһҸбҹ’бһ“ бһ“бһ·бһ„бһҳбһ¶бһ“бһҖбһ¶бһҡбһ‘бһ‘бһҪбһӣбһҒбһ»бһҹбһҸбҹ’бһҡбһјбһңбҹ– бһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбһҸбһ· Jira бһҮбһ¶бһҳбһҪбһҷбһ бҹҒбһҸбһ»бһ•бһӣ бһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡбһҷбҹ„бһ„бһўбһҸбһ·бһҗбһ·бһҮбһ“ бһ бһҫбһҷбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһҠбһӣбҹӢбһҳбҹ’бһ…бһ¶бһҹбҹӢбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бҹ” бһҹбһ·бһ‘бҹ’бһ’бһ·бһ“бҹҒбҹҮ **бһҳбһ·бһ“** бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһ–бһҪбһҖбһӮбҹҒбһҡбһ»бһүбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡ *бһ‘бҹ…бһҳбһ»бһҒ* бһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢбһ…бҹ’бһҡбһҖбһЎбһҫбһҷ (бһ§. бһӣбҹ„бһҸ QA бһ‘бҹ… Done) вҖ” бһ…бҹ’бһҡбһҖбһӮбһ»бһҺбһ—бһ¶бһ–бһ“бҹ…бһҸбҹӮбһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ” PO/PM бһ‘бһ‘бһҪбһӣбһҒбһ»бһҹбһҸбҹ’бһҡбһјбһңбһӣбһҫбһ•бһӣбһңбһ·бһ”бһ¶бһҖбһ“бҹғбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҺбһ¶бһҳбһҪбһҷбһҠбҹӮбһӣбһӮбҹҒбһ•бҹ’бһӣбһ¶бһҹбҹӢбҹ”
- **бһҖбһ¶бһҡбһҳбһ·бһ“бһҷбһӣбҹӢбһҹбҹ’бһҡбһ”бһӣбһҫбһ…бҹ’бһҡбһҖбҹ”** бһ”бһҫбһҸбһҪбһ“бһ¶бһ‘бһёбһ–бһёбһҡбһҳбһ·бһ“бһҷбһӣбҹӢбһҹбҹ’бһҡбһ”бһҗбһ¶бһ…бҹ’бһҡбһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹҶбһ–бҹҒбһүбһ¬бһўбһҸбҹӢ (бһ§. QA бһҗбһ¶бһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ”бһҡбһ¶бһҮбҹҗбһҷ бһҸбҹӮ Dev бһҗбһ¶бһңбһ¶бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң) бһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ“бҹ…бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ бһ бһҫбһҷбһ”бһүбҹ’бһ бһ¶бһҸбҹ’бһҡбһјбһңбһ”бһүбҹ’бһҮбһјбһ“бһ‘бҹ… PMбҹ” PM бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…; бһ”бһҫбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„бһңбһ·бһҹбһ¶бһӣбһ—бһ¶бһ– бһ¬бһҸбһҳбҹ’бһӣбҹғ PO бһҹбһҳбҹ’бһҡбҹҒбһ…бҹ” бһӣбһ‘бҹ’бһ’бһ•бһӣбһҸбҹ’бһҡбһјбһңбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҮбһ¶бһҳбһҸбһ· Jiraбҹ”
- **бһҖбһ¶бһҡбһҮбһҪбһҹбһҮбһ»бһӣбһ”бһ“бҹ’бһ‘бһ¶бһ“бҹӢбһ“бҹ… production (hotfix)бҹ”** бһ”бһүбҹ’бһ бһ¶ production бһ’бҹ’бһ„бһ“бҹӢбһ’бҹ’бһ„бһҡбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ•бҹ’бһӣбһјбһңбһҡбһ бҹҗбһҹбһҠбҹӮбһӣбһ”бһ¶бһ“бһ–бҹ’бһҡбһҳбһ–бҹ’бһҡбҹҖбһ„бһҮбһ¶бһҳбһ»бһ“бһҠбҹ„бһҷ PM бһ“бһ·бһ„ POбҹ” бһ‘бҹ„бҹҮбһ”бһёбһ”бҹӮбһ”бһ“бҹҒбҹҮбһҖбҹҸбһҠбҹ„бһҷ бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһјбһҠ (Gate 3) бһ“бһ·бһ„бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸ QA (Gate 5) бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбҹҶбһӣбһ„бһЎбһҫбһҷ вҖ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһҖбҹҶбһҺбһҸбҹӢбһ–бҹҒбһӣ бһҳбһ·бһ“бһҳбҹӮбһ“бһӣбһ»бһ”бһ…бҹ„бһӣбҹ” бһ…бҹ’бһҡбһҖбһҺбһ¶бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҡбһҪбһҳбһҸбҹ’бһҡбһјбһңбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ“бҹ…бһӣбһҫбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбҹ”
- **бһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһҖбһҡбһҺбһёбһӣбһҫбһҖбһӣбҹӮбһ„бҹ”** бһҸбһҪбһ“бһ¶бһ‘бһёбһҺбһ¶бһҖбҹҸбһўбһ¶бһ…бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһҖбһҡбһҺбһёбһӣбһҫбһҖбһӣбҹӮбһ„бһ–бһёбһ“бһёбһҸбһ·бһңбһ·бһ’бһёбһ“бҹҒбҹҮ бһҠбҹ„бһҷбһӣбһҫбһҖбһЎбһҫбһ„бһ‘бҹ…бһҳбҹ’бһ…бһ¶бһҹбҹӢбһҜбһҖбһҹбһ¶бһҡ бһ¬ PMбҹ” бһҖбһҡбһҺбһёбһӣбһҫбһҖбһӣбҹӮбһ„бһҠбҹӮбһӣбһ”бһ¶бһ“бһўбһ“бһ»бһҳбҹҗбһҸбһҸбҹ’бһҡбһјбһңбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ“бҹ…бһӣбһҫбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡ; бһңбһ¶бһҳбһ·бһ“бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ“бһёбһҸбһ·бһңбһ·бһ’бһёбһҒбҹ’бһӣбһҪбһ“бһҜбһ„бһЎбһҫбһҷбҹ”
- **бһҖбһ¶бһҡбһҳбһ·бһ“бһӮбҹ„бһҡбһ–бһҠбһҠбҹӮбһӣбҹ—бҹ”** бһ”бһҫбһ“бһёбһҸбһ·бһңбһ·бһ’бһёбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбҹҶбһӣбһ„бһҮбһ¶бһүбһ№бһҖбһүбһ¶бһ”бҹӢ (бһ§. бһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһӣбһ¶бһҹбҹӢбһҠбҹ„бһҷбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһҳбҹӮбһ“бһҳбҹ’бһ…бһ¶бһҹбҹӢ бһ¬бһ…бҹ’бһҡбһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбҹҶбһӣбһ„) PM бһӣбһҫбһҖбһңбһ¶бһЎбһҫбһ„бһ“бҹ… retrospective бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷбһҖбҹ’бһҡбһ»бһҳбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҳбһјбһӣбһ бҹҒбһҸбһ» вҖ” бһҗбһ¶бһҸбһҫбһ…бҹ’бһ”бһ¶бһ”бҹӢбһҒбһ»бһҹ бһ¬бһӮбҹҒбһҳбһ·бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳбҹ”

### бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ“бһёбһҸбһ·бһңбһ·бһ’бһёбһ“бҹҒбҹҮ вҖ” бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҖбҹӢбһҖбҹ’бһ“бһ»бһ„бһўбҹҶбһЎбһ»бһ„ sprint

бһ“бһёбһҸбһ·бһңбһ·бһ’бһёбһ“бҹҒбҹҮ (ISOP вҖ” Internal Sign-Off Procedure) бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“ **бһ”бһ„бҹ’бһҖбһҖбҹӢ (frozen) бһ–бҹҒбһүбһҳбһҪбһҷбһҡбһҷбҹҲбһ–бҹҒбһӣбһ“бҹғ sprint**бҹ” бһӮбҹ’бһҳбһ¶бһ“бһ“бһҡбһҺбһ¶бһҳбҹ’бһ“бһ¶бһҖбҹӢбһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ…бҹ’бһ”бһ¶бһ”бҹӢбһ“бҹ…бһ–бһ¶бһҖбҹӢбһҖбһҺбҹ’бһҸбһ¶бһӣ sprint бһ”бһ¶бһ“бһЎбһҫбһҷ бһҠбҹ„бһҷбһҳбһ·бһ“бһӮбһ·бһҸбһ–бһёбһҸбһҪбһ“бһ¶бһ‘бһёбҹ”

- бһҡбһ¶бһӣбҹӢбһ”бһүбҹ’бһ бһ¶ бһ…бһ“бҹ’бһӣбҹ„бҹҮбһҒбҹ’бһңбҹҮбһҒбһ¶бһҸ бһ¬бһҖбһ¶бһҡбһҳбһ·бһ“бһҷбһӣбҹӢбһҹбҹ’бһҡбһ”бһҮбһ¶бһҳбһҪбһҷбһ“бһёбһҸбһ·бһңбһ·бһ’бһё бһҠбҹӮбһӣбһҡбһҖбһғбһҫбһүбһҖбҹ’бһ“бһ»бһ„бһўбҹҶбһЎбһ»бһ„ sprint бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“ **бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶** (бһ“бҹ…бһӣбһҫбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡ бһ¬бһ”бһүбҹ’бһҮбһёбһҡбһҪбһҳ) бһҳбһ·бһ“бһҳбҹӮбһ“бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҠбҹ„бһҷбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ…бҹ’бһ”бһ¶бһ”бҹӢбһ—бҹ’бһӣбһ¶бһҳбҹ—бһЎбһҫбһҷбҹ”
- бһ”бһүбҹ’бһ бһ¶бһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“ **бһӣбһҫбһҖбһЎбһҫбһ„ бһ“бһ·бһ„бһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһ“бҹ… Sprint Retrospective**бҹ”
- бһҖбҹ’бһҡбһ»бһҳ **бһҹбһҳбҹ’бһҡбҹҒбһ…** бһ“бҹ…бһ‘бһёбһ“бҹ„бҹҮбһҗбһ¶ бһҸбһҫбһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ– ISOP бһ¬бһўбһҸбҹӢ бһ бһҫбһҷбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҠбҹӮбһӣбһ”бһ¶бһ“бһ–бҹ’бһҡбһҳбһ–бҹ’бһҡбҹҖбһ„ бһ…бһјбһӣбһҮбһ¶бһ’бһҡбһҳбһ¶бһ“бһ“бҹ… **sprint бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ** вҖ” бһҳбһ·бһ“бһҳбҹӮбһ“ sprint бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһЎбһҫбһҷбҹ”
- бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһҸбҹ’бһҡбһјбһңбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҖбҹ’бһ“бһ»бһ„ Change Log бһҮбһ¶бһҳбһҪбһҷбһӣбҹҒбһҒбһҖбҹҶбһҺбҹӮбһҗбҹ’бһҳбһё бһ“бһ·бһ„бһҖбһ¶бһӣбһ”бһҡбһ·бһ…бҹ’бһҶбҹҒбһ‘бһ…бһјбһӣбһҮбһ¶бһ’бһҡбһҳбһ¶бһ“бҹ”

бһ“бҹҒбҹҮбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҡбһ¶бһӣбҹӢбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ sprint бһҹбҹ’бһҗбһ·бһҸбһҖбҹ’бһҡбҹ„бһҳбһ…бҹ’бһ”бһ¶бһ”бҹӢбһҹбҹ’бһҗбһ·бһҡбһ—бһ¶бһ–бһҸбҹӮбһҳбһҪбһҷ бһ–бһёбһҠбһҫбһҳбһҠбһӣбҹӢбһ…бһ”бҹӢбҹ” бһ“бһёбһҸбһ·бһңбһ·бһ’бһёбһ“бҹҒбҹҮбһҮбһ¶бһҖбһ·бһ…бҹ’бһ…бһ–бҹ’бһҡбһҳбһ–бҹ’бһҡбҹҖбһ„бһҖбһ¶бһҡбһ„бһ¶бһҡбҹ– бһҹбҹ’бһҗбһ·бһҡбһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„ sprint бһҖбҹӮбһӣбһҳбҹ’бһўбһ“бҹ…бһ…бһ“бҹ’бһӣбҹ„бҹҮ sprintбҹ”

## бҹ§. бһңбһ…бһ“бһ¶бһ“бһ»бһҖбҹ’бһҡбһҳ (Glossary)

| бһ–бһ¶бһҖбҹ’бһҷ | бһўбһҸбҹ’бһҗбһ“бҹҗбһҷ |
|------|---------|
| FS | Functional Spec вҖ” бһҜбһҖбһҹбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҠбҹ„бһҷ PO бһҠбҹӮбһӣбһҖбҹҶбһҺбһҸбҹӢбһҗбһ¶ бһўбҹ’бһңбһёбһҸбҹ’бһҡбһјбһңбһҹбһ¶бһ„бһҹбһ„бҹӢ бһ“бһ·бһ„бһ бҹҒбһҸбһ»бһўбҹ’бһңбһё |
| AC | Acceptance Criteria вҖ” бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһўбһ¶бһ…бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ”бһ¶бһ“ бһҠбҹӮбһӣбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҸбҹ’бһҡбһјбһңбһ”бҹҶбһ–бҹҒбһү бһҠбһҫбһҳбҹ’бһ”бһёбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ‘бһ‘бһҪбһӣбһҷбһҖ |
| DoR | Definition of Ready вҖ” бһ”бһүбҹ’бһҮбһёбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҠбҹӮбһӣбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҸбҹ’бһҡбһјбһңбһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢбһҳбһ»бһ“бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ |
| DoD | Definition of Done вҖ” бһ”бһүбҹ’бһҮбһёбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһӮбһ»бһҺбһ—бһ¶бһ–бһҠбҹӮбһӣбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҸбҹ’бһҡбһјбһңбһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢбһҳбһ»бһ“бһңбһ¶бһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶ Done |
| PR | Pull Request вҖ” бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҖбһјбһҠбһҠбҹӮбһӣбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ |
