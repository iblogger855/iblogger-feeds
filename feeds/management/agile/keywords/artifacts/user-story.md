## бҹЎ. бһҸбһҫвҖӢбһўбҹ’бһңбһёвҖӢбһ‘бҹ…вҖӢбһҮбһ¶ User Story? (What is User Story?)

**бһҡбһҝбһ„бһҡбҹүбһ¶бһңбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ (User Story)** бһӮбһәбһҮбһ¶вҖӢбһҖбһ¶бһҡвҖӢбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһўбҹҶвҖӢбһ–бһёвҖӢбһҳбһ»бһҒбһ„бһ¶бһҡвҖӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёвҖӢбһҳбһҪбһҷвҖӢбһҷбҹүбһ¶бһ„вҖӢбһҹбһ¶бһҳбһүбҹ’бһү бһҒбҹ’бһӣбһё бһ“бһ·бһ„вҖӢбһ„бһ¶бһҷбһҷбһӣбҹӢ бһ•бҹ’бһўбҹӮбһҖвҖӢбһӣбһҫвҖӢбһ‘бһҹбҹ’бһҹбһ“бҹҲвҖӢбһҡбһ”бһҹбҹӢвҖӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢвҖӢбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ (End-User)бҹ” бһңбһ¶вҖӢбһҮбһ¶вҖӢбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбҹ’бһ“бһјбһӣвҖӢбһҖбҹ’бһ“бһ»бһ„вҖӢбһҖбһ¶бһҡвҖӢбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҸбһҳбҹ’бһҡбһјбһңвҖӢбһҖбһ¶бһҡ (Requirements) бһҖбҹ’бһ“бһ»бһ„вҖӢбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ Agile бһҠбҹ„бһҷвҖӢбһ•бҹ’бһҸбҹ„бһҸвҖӢбһӣбһҫ В«бһўбҹ’бһ“бһҖвҖӢбһҺбһ¶В» В«бһ…бһ„бҹӢвҖӢбһ”бһ¶бһ“вҖӢбһўбҹ’бһңбһёВ» бһ“бһ·бһ„ В«бһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһўбҹ’бһңбһёВ»бҹ”

User Story бһҳбһ·бһ“вҖӢбһҳбҹӮбһ“вҖӢбһҮбһ¶вҖӢбһӣбһҖбҹ’бһҒбһҺбҹҲвҖӢбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһӣбһҳбҹ’бһўбһ·бһҸ (Specification) бһЎбһҫбһҷ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮвҖӢбһңбһ¶вҖӢбһҮбһ¶вҖӢбһ…бҹҶбһҺбһ»бһ…бһ…бһ¶бһ”бҹӢбһ•бҹ’вҖӢбһҸбһҫвҖӢбһҳвҖӢбһ“бҹғвҖӢбһҖбһ¶бһҡвҖӢбһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶вҖӢбһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһҷбһӣбҹӢвҖӢбһ–бһёвҖӢбһҸбһҳбҹ’бһҡбһјбһңвҖӢбһҖбһ¶бһҡвҖӢбһўбһ¶бһҮбһёбһңбһҖбһҳбҹ’бһҳвҖӢбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбҹ”

## бҹў. бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡвҖӢбһ“бҹғ User Story (Standard Format)

бһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһ’бһ¶бһ“бһ¶вҖӢбһҖбһ¶бһҡвҖӢбһ•бҹ’бһҸбҹ„бһҸвҖӢбһӣбһҫвҖӢбһҸбһҳбҹ’бһӣбҹғвҖӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһӮбҹҒвҖӢбһҹбһҡбһҹбҹҒбһҡвҖӢбһҸбһ¶бһҳвҖӢбһ‘бһҳбҹ’бһҡбһ„бҹӢбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбҹ–

> **бһҖбҹ’бһ“бһ»бһ„вҖӢбһ“бһ¶бһҳвҖӢбһҮбһ¶ (As a)** `[бһ”бҹ’бһҡбһ—бҹҒбһ‘бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ / Role]` 
> **бһҒбҹ’бһүбһ»бҹҶвҖӢбһ…бһ„бҹӢвҖӢбһ”бһ¶бһ“ (I want to)** `[бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– / Action]` 
> **бһҠбһҫбһҳбҹ’бһ”бһё (So that)** `[бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚ / Benefit]` 

*бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ–* 
> **бһҖбҹ’бһ“бһ»бһ„вҖӢбһ“бһ¶бһҳвҖӢбһҮбһ¶** бһўбһҸбһ·бһҗбһ·бһҮбһ“бһ‘бһ·бһүбһ‘бҹҶбһ“бһ·бһү 
> **бһҒбҹ’бһүбһ»бҹҶвҖӢбһ…бһ„бҹӢ** бһ”бһ“бҹ’бһҗбҹӮвҖӢбһҳбһҖвҖӢбһ¶бһҸбһ’бһ“бһ¶бһӮбһ¶бһҡвҖӢбһҡбһ”бһҹбҹӢвҖӢбһҒбҹ’бһүбһ»бҹҶвҖӢбһ‘бҹ…вҖӢбһҖбҹ’бһ“бһ»бһ„вҖӢбһӮбһҺбһ“бһё 
> **бһҠбһҫбһҳбҹ’бһ”бһё** бһҒбҹ’бһүбһ»бҹҶбһўбһ¶бһ…бһ‘бһјбһ‘бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһҖбҹӢвҖӢбһ”бһ¶бһ“вҖӢбһҡбһ бҹҗбһҹвҖӢбһ“бҹ…вҖӢбһ–бҹҒбһӣвҖӢбһ”бһүбҹ’вҖӢбһҮбһ¶вҖӢбһ‘бһ·бһүвҖӢбһӣбһҫвҖӢбһҖвҖӢбһҖбҹ’бһҡбҹ„бһҷ

## бҹӨ. бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢвҖӢбһ“бҹғвҖӢбһҖбһ¶бһҡвҖӢбһ‘бһ‘бһҪбһӣбһҷбһҖвҖӢбһҖбһ¶бһҡвҖӢбһ„бһ¶бһҡ (Acceptance Criteria)

бһҡбһ¶бһӣбҹӢ User Story бһҸбҹ’бһҡбһјбһңбһҸбҹӮвҖӢбһ—бҹ’вҖӢбһҮбһ¶вҖӢбһ”бҹӢвҖӢбһҳбһҖвҖӢбһҮбһ¶вҖӢбһҳбһҪбһҷбһ“бһјбһң **бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢвҖӢбһ“бҹғвҖӢбһҖбһ¶бһҡвҖӢбһ‘бһ‘бһҪбһӣбһҷбһҖвҖӢбһҖбһ¶бһҡвҖӢбһ„бһ¶бһҡ (Acceptance Criteria - AC)**бҹ” AC бһӮбһәбһҮбһ¶вҖӢбһ”бһүбҹ’бһҮбһёбһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ бһ“бһ·бһ„вҖӢбһҳбһ»бһҒбһ„бһ¶бһҡвҖӢбһҠбҹӮбһӣвҖӢбһҸбҹ’бһҡбһјбһңвҖӢбһ”бҹҶбһ–бҹҒбһү бһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһ”бһүбҹ’вҖӢбһҮбһ¶вҖӢбһҖбҹӢбһҗбһ¶бһҖбһ·бһ…бҹ’бһ…вҖӢбһҖбһ¶бһҡвҖӢбһ“бҹ„бҹҮвҖӢбһҸбҹ’бһҡбһјбһңвҖӢбһ”бһ¶бһ“вҖӢбһ”бһүбҹ’бһ…бһ”бҹӢвҖӢбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ (бһ”бҹҶбһ–бҹҒбһүвҖӢбһҸбһ¶бһҳ DoD)бҹ”

AC бһҮбһ¶вҖӢбһҡбһҝбһҷ бҹ— бһҸбҹ’бһҡбһјбһңвҖӢбһ”бһ¶бһ“вҖӢбһҹбһҡбһҹбҹҒбһҡвҖӢбһҸбһ¶бһҳвҖӢбһ‘бһҳбҹ’бһҡбһ„бҹӢ **Given-When-Then** (Behavior-Driven Development - BDD)бҹ–
* **Given (бһҠбҹ„бһҷбһҹбһ¶бһҡвҖӢбһҸбҹӮ):** бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҠбҹҶбһ”бһјбһ„ (бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ– бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢвҖӢбһҹбҹ’бһҗбһ·бһҸбһ“бҹ…вҖӢбһӣбһҫвҖӢбһ‘бҹҶбһ–бҹҗбһҡвҖӢбһ‘бһјбһ‘бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһҖбҹӢ)
* **When (бһ“бҹ…вҖӢбһ–бҹҒбһӣвҖӢбһҠбҹӮбһӣ):** бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–вҖӢбһҠбҹӮбһӣвҖӢбһҖбһҫбһҸбһЎбһҫбһ„ (бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ– бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢвҖӢбһ”бһүбҹ’бһ…бһјбһӣбһӣбҹҒбһҒбһҖбһ¶бһҸвҖӢбһҳбһ·бһ“вҖӢбһҸбҹ’бһҡбһ№бһҳвҖӢбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„вҖӢбһ…бһ»бһ…бһ‘бһјбһ‘бһ¶бһҸбҹӢ)
* **Then (бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢвҖӢбһҳбһҖ):** бһӣбһ‘бҹ’бһ’бһ•бһӣбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖ (бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ– бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’вҖӢбһ”бһ„бҹ’бһ бһ¶бһүвҖӢбһҹбһ¶бһҡбһ–бҹ’бһҡвҖӢбһҳбһ¶бһ“вҖӢбһҗбһ¶ "бһӣбҹҒбһҒбһҖбһ¶бһҸвҖӢбһҳбһ·бһ“вҖӢбһҸбҹ’бһҡбһ№бһҳвҖӢбһҸбҹ’бһҡбһјбһң")
