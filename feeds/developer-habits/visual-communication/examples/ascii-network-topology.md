## Network Diagram Character Conventions

| Symbol | Meaning |
| :--- | :--- |
| `в”Җв”Җв–ә` / `в—„в”Җв”Җ` | Directional data flow |
| `в”Җв”Җв”Җв”Җв”Җ` | Connection / link |
| `в”Ӯ` | Vertical connection |
| `в”Ңв”Җв”җ в””в”Җв”ҳ` | Zone or server boundary box |
| `[ ]` | Lightweight component label |
| `AZ-1 / AZ-2` | Availability zone annotation |
