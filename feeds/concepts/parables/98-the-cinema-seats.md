# Array Data Structure (бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһ¶бһҡбҹҒ)бҹ– бһҖбҹ…бһўбһёбһҡбҹ„бһ„бһҖбһ»бһ“ бһ“бһ·бһ„бһҒбҹ’бһҹбҹӮбһ—бһ¶бһ–бһҷбһ“бҹ’бһҸ (Arrays & The Cinema Seats)

## рҹ“Ң бһҳбһ¶бһҸбһ·бһҖбһ¶ (Table of Contents)
- [бһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢбһ•бҹ’бһӣбһјбһңбһ…бһ·бһҸбҹ’бһҸ (The Trap)](#0)
- [бҹЎ. бһҡбһҝбһ„бһ–бҹ’бһҡбҹҒбһ„бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбҹ– бһҳбһ·бһҸбҹ’бһҸбһ—бһҖбҹ’бһҸбһ·бһ‘бһ¶бҹҶбһ„бһ”бһёбһ“бһ¶бһҖбҹӢ бһ“бһ·бһ„бһҖбҹ…бһўбһёбһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶бһҖбҹ’бһ“бһ»бһ„бһҡбҹ„бһ„бһҖбһ»бһ“ (The Legend of cinema contiguous seats)](#1)
  - [бһҳбһ·бһҸбҹ’бһҸбһ—бһҖбҹ’бһҸбһ·бһҷбһәбһҸбһҷбҹүбһ¶бһң бһ“бһ·бһ„бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҮбһҪбһҡбһҠбҹҸбһӮбһҪбһҡбһұбҹ’бһҷбһ’бһ»бһүбһ‘бҹ’бһҡбһ¶бһ“бҹӢ (Shifting and Moving Array Resizing)](#1-1)
- [бҹў. бһ”бһүбҹ’бһ бһ¶бҹ– бһҖбһ¶бһҡбһ…бһ„бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ…бһ„бһ…бһ¶бҹҶбһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶ бһ“бһ·бһ„бһҖбһ¶бһҡбһ…бҹҶбһҺбһ¶бһҷбһӣбһҫбһҖбһ¶бһҡбһҡбһ»бһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (The Issue: Contiguous Memory Allocations and Shifting Overheads)](#2)
- [бҹЈ. бһ§бһ‘бһ¶бһ бһҳбһҺбҹҚбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҖбҹ’бһ“бһ»бһ„бһ–бһ·бһ—бһ–бһ–бһ·бһҸ (Real World Examples)](#3)
  - [бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹЎ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбҹ’бһҡбһ¶бһӣ (бһӮбҹ’бһҡбһҪбһҹбһ¶бһҡ)бҹ– бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһҹбҹҖбһңбһ—бҹ…бһҸбһ¶бһҳбһӣбҹҶбһҠбһ¶бһ”бҹӢбһӣбһҫбһ’бҹ’бһ“бһҫ (Sequential Bookshelf Organization)](#3-1)
  - [бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹў вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ’бҹ’бһҷбһҳ (бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ)бҹ– бһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбһ¶бһҳбһҡбһҷбҹҲ Indexes (Fast Index Lookup in Ram Arrays)](#3-2)
  - [бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹЈ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ’бҹ’бһҷбһҳ (бһ’бһ»бһҡбһҖбһ·бһ…бҹ’бһ…)бҹ– бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҮбһҪбһҡбһҖбҹ…бһўбһёбһўбҹ’бһ“бһҖбһҠбҹҶбһҺбһҫбһҡбһӣбһҫбһҷбһ“бҹ’бһҸбһ бҹ„бҹҮ (Contiguous Aircraft Seating Assignments)](#3-3)
  - [бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹӨ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ’бҹ’бһҷбһҳ (бһҹбһ„бҹ’бһӮбһҳ/бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„)бҹ– бһҮбһҪбһҡбһҖбҹ’бһ”бһҪбһ“бһҡбһҗбһҷбһ“бҹ’бһҸбһӮбһҺбҹҲбһ”бҹ’бһҡбһҸбһ·бһ—бһјбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡ (Sequential Diplomatic Motorcade Formations)](#3-4)
  - [бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹҘ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһ’бҹ’бһ„бһ“бҹӢ (бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„)бҹ– бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһўбһ“бһ»бһҹбҹ’бһҹбһ¶бһңбһҡбһёбһҷбҹҚбһ”бҹ’бһҡбһ…бһ¶бҹҶбһҗбҹ’бһ„бҹғбһҸбһ¶бһҳбһҖбһ¶бһӣбһ”бһҡбһ·бһ…бҹ’бһҶбҹҒбһ‘ (Rigid Chronological Memory Books)](#3-5)
- [бҹӨ. бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бһјбһ‘бҹ…бҹ– бһҸбһ»бһӣбҹ’бһҷбһ—бһ¶бһ–бһҡбһңбһ¶бһ„ Arrays бһ“бһ·бһ„ Dynamic Resizing Mechanics (The General Solution: Optimizing Array Allocations and Dynamic Resizing)](#4)

## бһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢбһ•бҹ’бһӣбһјбһңбһ…бһ·бһҸбҹ’бһҸ (The Trap)

бһҸбһҫбһўбҹ’бһ“бһҖбһ’бҹ’бһӣбһ¶бһ”бҹӢбһҮбһҪбһ”бһ”бһүбҹ’бһ бһ¶бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһҹбҹҶбһҺбһ»бҹҶбһӣбҹҶбһҠбһ¶бһ”бҹӢбһӣбҹҶбһҠбҹ„бһҷ бһ бһҫбһҷбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Array бһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶбһ…бһ¶бһҖбҹӢбһҹбҹ„бһҡ (Fixed Size Array) бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһўбһ¶бһ…бһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҗбҹ’бһҳбһёбһ”бһ¶бһ“бһ“бҹ…бһ–бҹҒбһӣбһңбһ¶бһ–бҹҒбһү бһ¬бһҸбҹ’бһҡбһјбһңбһ…бҹҶбһҺбһ¶бһҷбһҗбһ¶бһҳбһ–бһӣбһӮбһҺбһ“бһ¶бһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„ (Performance Penalty) бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбһ»бһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҗбһҷбһҖбҹ’бһҡбҹ„бһҷ (Shifting) бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹ…бһ…бҹҶбһҖбһҺбҹ’бһҸбһ¶бһӣбһҠбҹӮбһҡбһ¬бһ‘бҹҒ?

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ–
* **бһҷбһҫбһ„бһ„бһ¶бһҷбһ“бһ№бһ„бһ’бҹ’бһӣбһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„бһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢ** бһ“бҹғбһҖбһ¶бһҡбһӮбһ·бһҸбһҗбһ¶ Array бһӮбһәбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ’бһҡбһ”бҹӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҠбҹ„бһҷбһҳбһҫбһӣбһҡбҹҶбһӣбһ„бһҖбһ¶бһҡбһҒбһ¶бһҸбһ”бһ„бҹӢбһҗбһ¶бһҳбһ–бһӣ CPU бһӣбһҫбһҖбһ¶бһҡбһ…бһҳбҹ’бһӣбһ„ бһ“бһ·бһ„бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһёбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбҹ’бһ“бһ»бһ„ Memory (O(N) operations for insertion/deletion) бһ“бҹ…бһ–бҹҒбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бҹҶбһ бҹҶбһүбһ№бһҖбһүбһ¶бһ”бҹӢбҹ”
* **бһҷбһҫбһ„бһҳбһҫбһӣбһҡбҹҶбһӣбһ„** бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бһҳбҹ’бһҡбһ»бһ„ Memory бһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶ (Contiguous Allocation) бһҠбҹӮбһӣбһўбһ¶бһ…бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһҡбһ¶бһҮбҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ”бһ“бҹ’бһ‘бһ”бҹӢбһ‘бҹҶбһ“бҹҒбһҡ бһ”бһҫбһ‘бҹ„бҹҮбһҮбһ¶бһ‘бҹҶбһ бҹҶ Memory бһҹбһҡбһ»бһ”бһ“бҹ…бһҹбһӣбҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҖбҹҸбһҠбҹ„бһҷ (Out of Memory due to fragmentation)бҹ”

бһҖбһ¶бһҡбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫ Array бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бҹҶбһ бҹҶ бһ“бһ·бһ„бһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһ“бҹ’бһҗбҹӮбһҳ/бһӣбһ»бһ”бһүбһ№бһҖбһүбһ¶бһ”бҹӢ бһ бҹ…бһҗбһ¶ **бһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҗбҹҒбһҡ (Fixed Contiguous Array Allocation Trap)**бҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһ…бҹҶбһҺбһ»бһ…бһҒбҹ’бһӣбһ¶бҹҶбһ„ бһ“бһ·бһ„бһҒбҹ’бһҹбҹ„бһҷбһҡбһ”бһҹбҹӢ Array бһ“бҹҒбҹҮбһҮбһ¶бһ•бҹӮбһ“бһ‘бһёбһ”бһ„бҹ’бһ бһ¶бһүбһ•бҹ’бһӣбһјбһңбҹ–
1. **бһҡбһҝбһ„бһ–бҹ’бһҡбҹҒбһ„бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ (The Historic Legend)** вҖ” бһҡбһҝбһ„бһҡбҹүбһ¶бһңбһҡбһ”бһҹбҹӢбһҳбһ·бһҸбҹ’бһҸбһ—бһҖбҹ’бһҸбһ·бһҠбҹӮбһӣбһ…бһ„бҹӢбһ‘бһ·бһүбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҖбҹ…бһўбһёбһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶бһҖбҹ’бһ“бһ»бһ„бһҡбҹ„бһ„бһҖбһ»бһ“ бһ“бһ·бһ„бһ—бһ¶бһ–бһңбһ№бһҖбһңбһҡбһ“бҹғбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҖбһ“бҹ’бһӣбҹӮбһ„бһ“бҹ…бһ–бҹҒбһӣбһҳбһ¶бһ“бһҳбһ·бһҸбҹ’бһҸбһ—бһҖбҹ’бһҸбһ·бһҳбһҖбһҷбһәбһҸбҹ”
2. **бһ”бһүбҹ’бһ бһ¶ (The Issue)** вҖ” бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһҷбһ“бҹ’бһҸбһҖбһ¶бһҡ Contiguous Allocation бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ RAM бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбһҺбһ“бһ¶ Time Complexity O(N) бһӣбһҫбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣ/бһӣбһ»бһ”бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ”
3. **бһ§бһ‘бһ¶бһ бһҳбһҺбҹҚбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҖбҹ’бһ“бһ»бһ„бһ–бһ·бһ—бһ–бһ–бһ·бһҸ (Real World Examples)** вҖ” бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһ”бһүбҹ’бһ бһ¶бһ“бҹҒбҹҮбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҡбһ·бһҸбһӮбҹ’бһҡбһҪбһҹбһ¶бһҡ бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶ бһ’бһ»бһҡбһҖбһ·бһ…бҹ’бһ… бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бҹ”
4. **бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бһјбһ‘бҹ… (The General Solution)** вҖ” бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҠбҹ„бһҷбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ»бһӣбҹ’бһҷбһ—бһ¶бһ–бһҡбһңбһ¶бһ„ static allocation, dynamic scaling, бһ“бһ·бһ„ hybrid data structuresбҹ”

## бҹЎ. бһҡбһҝбһ„бһ–бҹ’бһҡбҹҒбһ„бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбҹ– бһҳбһ·бһҸбҹ’бһҸбһ—бһҖбҹ’бһҸбһ·бһ‘бһ¶бҹҶбһ„бһ”бһёбһ“бһ¶бһҖбҹӢ бһ“бһ·бһ„бһҖбҹ…бһўбһёбһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶бһҖбҹ’бһ“бһ»бһ„бһҡбҹ„бһ„бһҖбһ»бһ“ (The Legend of cinema contiguous seats)

бһҖбһ¶бһӣбһ–бһёбһ–бҹ’бһҡбҹҒбһ„бһ“бһ¶бһҷ бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ‘бһёбһҖбҹ’бһҡбһ»бһ„бһҳбһҪбһҷ бһҳбһ¶бһ“бһҳбһ·бһҸбҹ’бһҸбһ—бһҖбҹ’бһҸбһ·бһҮбһ·бһҸбһҹбҹ’бһ“бһ·бһ‘бҹ’бһ’ бҹЈ бһ“бһ¶бһҖбҹӢбһ…бһ„бҹӢбһ‘бҹ…бһҳбһҫбһӣбһҒбҹ’бһҹбҹӮбһ—бһ¶бһ–бһҷбһ“бҹ’бһҸбһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶бҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ“бһ·бһҷбһ¶бһҷбһӣбҹҒбһ„ бһ“бһ·бһ„бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһ–бҹ„бһҸбһӣбһёбһ„бһүбҹүбһ¶бҹҶбҹ–
* бһ–бһҪбһҖбһӮбҹҒбһ…бһ„бҹӢбһўбһ„бҹ’бһӮбһ»бһҷ **бһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶бһӣбҹ’бһўбһҮбһ¶бһҮбһҪбһҡбһҸбҹӮбһҳбһҪбһҷ**бҹ” бһ–бһҪбһҖбһӮбҹҒбһ”бһ¶бһ“бһ‘бҹ…бһ”бһүбҹ’бһҮбһҡбһ‘бһ·бһүбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҖбҹ…бһўбһёбһӣбҹҒбһҒ бҹў, бһӣбҹҒбһҒ бҹЈ, бһ“бһ·бһ„бһӣбҹҒбһҒ бҹӨ бһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶ (Contiguous Memory Structure)бҹ”
* бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҠбҹҸбһўбһҹбҹ’бһ…бһ¶бһҡбҹ’бһҷбһӮбһәбҹ– бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҠбһ№бһ„бһҗбһ¶ Trang (бһҳбһ·бһҸбҹ’бһҸбһ‘бһёбһҳбһҪбһҷ) бһўбһ„бҹ’бһӮбһ»бһҷбһ“бҹ…бһҖбҹ…бһўбһёбһӣбҹҒбһҒ бҹў бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһӮбһҺбһ“бһ¶бһҡбһҖбһҖбҹ…бһўбһёбһҳбһ·бһҸбҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ—бҹ’бһӣбһ¶бһҳбҹ—бһҠбҹ„бһҷбһҳбһ·бһ“бһ”бһ¶бһ…бҹӢбһҠбһҫбһҡбһҹбһҪбһҡбһ“бһ¶бҹҶбҹ– бһӮбһә Sok бһ“бҹ…бһҖбҹ…бһўбһёбһӣбҹҒбһҒ бҹЈ (бһӣбҹҒбһҒ бҹў + бҹЎ) бһ“бһ·бһ„ Sao бһ“бҹ…бһҖбҹ…бһўбһёбһӣбҹҒбһҒ бҹӨ (бһӣбҹҒбһҒ бҹў + бҹў)бҹ” бһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһң **бһӣбҹ’бһ”бһҝбһ“бһҡбһҖбһғбһҫбһүбһӣбһҝбһ“бһ”бҹҶбһ•бһ»бһҸ (O(1) Direct Lookup)**бҹ”
* бһ–бһҪбһҖбһӮбҹҒбһўбһ¶бһ…бһ‘бһ¶бһҖбҹӢбһ‘бһ„бһӮбҹ’бһ“бһ¶ бһ“бһ·бһ„бһ бһ»бһ…бһ–бҹ„бһҸбһӣбһёбһ„бһұбҹ’бһҷбһӮбҹ’бһ“бһ¶бһ”бһ¶бһ“бһ—бҹ’бһӣбһ¶бһҳбҹ— бһҠбҹ„бһҷбһҳбһ·бһ“бһ”бһ¶бһ…бҹӢбһ„бһҫбһ”бһҠбһҫбһҡбһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢбһ“бһҡбһҺбһ¶бһҳбҹ’бһ“бһ¶бһҖбҹӢбһЎбһҫбһҷ (High Spatial Cache Locality)бҹ”

### бһҳбһ·бһҸбҹ’бһҸбһ—бһҖбҹ’бһҸбһ·бһҷбһәбһҸбһҷбҹүбһ¶бһң бһ“бһ·бһ„бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҮбһҪбһҡбһҠбҹҸбһӮбһҪбһҡбһұбҹ’бһҷбһ’бһ»бһүбһ‘бҹ’бһҡбһ¶бһ“бҹӢ (Shifting and Moving Array Resizing)

бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮ бһҳбһ»бһ“бһ–бҹҒбһӣбһҒбҹ’бһҹбҹӮбһ—бһ¶бһ–бһҷбһ“бҹ’бһҸбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ”бһүбҹ’бһ…бһ¶бҹҶбһ„ бҹҘ бһ“бһ¶бһ‘бһё бһҳбһ·бһҸбҹ’бһҸбһ—бһҖбҹ’бһҸбһ·бһ‘бһё бҹӨ бһҲбҹ’бһҳбҹ„бҹҮ Rith бһ”бһ¶бһ“бһҳбһҖбһҠбһӣбҹӢбһҷбҹүбһ¶бһ„бһ”бҹ’бһҡбһүбһ¶бһ”бҹӢбһ”бҹ’бһҡбһүбһ¶бһӣбҹӢбҹ” бһӮбһ¶бһҸбҹӢбһ…бһ„бҹӢбһўбһ„бҹ’бһӮбһ»бһҷбһҹбһҳбҹ’бһҡбһ¶бһҖбһҮбһ¶бһҳбһҪбһҷбһ–бһҪбһҖбһӮбҹҒбһҠбҹӮбһҡбҹ”
* Rith бһҹбһ»бҹҶбһўбһ„бҹ’бһӮбһ»бһҷбһ…бһ“бҹ’бһӣбҹ„бҹҮбһҖбһҺбҹ’бһҸбһ¶бһӣбһҡбһңбһ¶бһ„ Trang (бһӣбҹҒбһҒ бҹў) бһ“бһ·бһ„ Sok (бһӣбҹҒбһҒ бҹЈ)бҹ”
* бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷ Rith бһўбһ¶бһ…бһўбһ„бҹ’бһӮбһ»бһҷбһ”бһ¶бһ“ Sok бһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҒбҹҶбһ…бһ·бһҸбҹ’бһҸбһ„бһҫбһ”бһЎбһҫбһ„ бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһёбһ‘бҹ…бһўбһ„бҹ’бһӮбһ»бһҷбһҖбҹ…бһўбһёбһӣбҹҒбһҒ бҹӨ бһ бһҫбһҷ Sao бһҸбҹ’бһҡбһјбһңбһ„бһҫбһ”бһЎбһҫбһ„бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһёбһ‘бҹ…бһўбһ„бҹ’бһӮбһ»бһҷбһҖбҹ…бһўбһёбһӣбҹҒбһҒ бҹҘ (O(N) Shifting Action)бҹ” бһҳбһ“бһ»бһҹбҹ’бһҹбһҠбһ‘бҹғбһ‘бҹҖбһҸбһҳбһҫбһӣбһҳбһҖбһ–бһҪбһҖбһӮбҹҒбһҠбҹ„бһҷбһҖбһ¶бһҡбһҡбҹҶбһҒбһ¶бһ“бҹ”
* бһҡбһ№бһҸбһҸбҹӮбһўбһ¶бһҖбҹ’бһҡбһҖбҹӢбһҮбһ¶бһ„бһ“бҹҒбҹҮбһ‘бҹ…бһ‘бҹҖбһҸ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҖбҹ…бһўбһёбһӣбҹҒбһҒ бҹҘ бһҳбһ¶бһ“бһўбҹ’бһ“бһҖбһ•бҹ’бһҹбҹҒбһ„бһ‘бһ·бһүбһўбһ„бҹ’бһӮбһ»бһҷбһ”бһ¶бһҸбҹӢбһ‘бҹ…бһ бһҫбһҷ бһ–бһҪбһҖбһӮбҹҒбһҳбһ·бһ“бһўбһ¶бһ…бһҡбһ»бһүбһӮбҹ’бһ“бһ¶бһ”бһ¶бһ“бһ‘бҹҖбһҸбһЎбһҫбһҷбҹ”
* бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢбһӮбһәбҹ– **бһ–бһҪбһҖбһӮбҹҒбһ‘бһ¶бҹҶбһ„ бҹЈ бһ“бһ¶бһҖбҹӢбһҸбҹ’бһҡбһјбһңбһ„бһҫбһ”бһҠбһҫбһҡбһ…бҹҒбһүбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһӮбҹ’бһ“бһ¶** бһҠбһҫбһҡбһ‘бҹ…бһҡбһҖбһҮбһҪбһҡбһҗбҹ’бһҳбһёбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„бһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбҹ…бһўбһёбһ‘бҹҶбһ“бҹҒбһҡ бҹӨ бһҖбһ“бҹ’бһӣбҹӮбһ„бһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶ бһҡбһҪбһ…бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһўбһ„бҹ’бһӮбһ»бһҷбһҡбҹҖбһ”бһҮбһҪбһҡбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸбһ–бһёбһҠбһҫбһҳ (O(N) Reallocation and Copying)бҹ”

бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҖбҹ…бһўбһёбһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶бһӮбһәбһўбһҹбҹ’бһ…бһ¶бһҡбҹ’бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ„бҹ’бһӮбһ»бһҷбһҳбһҫбһӣ бһ“бһ·бһ„бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһӣбһҝбһ“ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһңбһ¶бһҮбһ¶бһҹбһ»бһ”бһ·бһ“бҹ’бһҸбһўбһ¶бһҖбҹ’бһҡбһҖбҹӢбһ“бҹ…бһ–бҹҒбһӣбһҳбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҖбҹ…бһўбһё бһ¬бһҳбһ¶бһ“бһҹбһҳбһ¶бһҮбһ·бһҖбһҗбҹ’бһҳбһёбһ…бһјбһӣбһҡбһҪбһҳбҹ”

## бҹў. бһ”бһүбҹ’бһ бһ¶бҹ– бһҖбһ¶бһҡбһ…бһ„бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ…бһ„бһ…бһ¶бҹҶбһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶ бһ“бһ·бһ„бһҖбһ¶бһҡбһ…бҹҶбһҺбһ¶бһҷбһӣбһҫбһҖбһ¶бһҡбһҡбһ»бһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (The Issue: Contiguous Memory Allocations and Shifting Overheads)

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ Array бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ RAM бһҮбһ¶бһ”бҹ’бһӣбһ»бһҖ Memory бһҮбһ¶бһ”бҹӢбҹ—бһӮбҹ’бһ“бһ¶ (Contiguous Block)бҹ” бһ“бҹҒбҹҮбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһўбһ¶бһ…бһӮбһҺбһ“бһ¶бһўбһ¶бһҹбһҷбһҠбҹ’бһӢбһ¶бһ“бһҡбһ”бһҹбҹӢ Element бһ‘бһё `i` бһ”бһ¶бһ“бһ—бҹ’бһӣбһ¶бһҳбҹ—бҹ–

$$\text{Address} = \text{BaseAddress} + i \times \text{ElementSize}$$

бһҹбһҳбһёбһҖбһ¶бһҡбһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҳбһүбҹ’бһүбһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһӣбҹ’бһ”бһҝбһ“ Direct Lookup O(1) бһҠбҹҸбһӣбһҝбһ“бһ”бҹҶбһ•бһ»бһҸбҹ” бһ‘бҹ„бҹҮбһҮбһ¶бһҷбҹүбһ¶бһ„бһҺбһ¶ бһ—бһ¶бһ–бһҮбҹҶбһ–бһ¶бһҖбҹӢбһҮбҹҶбһ–бһ·бһ“бһҖбһҫбһҸбһЎбһҫбһ„бһ“бҹ…бһ–бҹҒбһӣбһҷбһҫбһ„бһ”бһүбҹ’бһ…бһјбһӣ (Insert) бһ¬бһӣбһ»бһ” (Delete) бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ–

```java
// бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹ…бһҖбһҺбҹ’бһҸбһ¶бһӣ Array бһӮбһәбһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶ бһ“бһ·бһ„бһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„ CPU бһҒбҹ’бһӣбһ¶бҹҶбһ„
public void insertAt(int[] array, int index, int value) {
    // бһҸбҹ’бһҡбһјбһңбһҡбһ»бһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһҖбҹ’бһҡбҹ„бһҷ index бһҗбһҷбһҖбҹ’бһҡбҹ„бһҷбһҳбҹ’бһҸбһ„бһҳбһҪбһҷбҹ—
    for (int i = array.length - 1; i > index; i--) {
        array[i] = array[i - 1]; // O(N) Shifting overhead
    }
    array[index] = value;
}
```

* **бһҖбһ¶бһҡбһ…бҹҶбһҺбһ¶бһҷбһӣбһҫбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһё (Shifting Cost O(N))бҹ–** бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбһ¶бһҡбҹҒбһҳбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бҹЎ бһӣбһ¶бһ“бһ’бһ¶бһҸбһ» бһ бһҫбһҷбһҷбһҫбһ„бһ…бһ„бҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ’бһ¶бһҸбһ»бһҗбҹ’бһҳбһёбһ“бҹ…бһ…бҹҶбһҖбһҺбҹ’бһҸбһ¶бһӣ бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ„бһ¶бһҡбһ…бһҳбҹ’бһӣбһ„ бһ“бһ·бһ„бһҡбһ»бһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бҹҶбһ“бһҪбһ“ бҹҘбҹ  бһҳбҹүбһәбһ“бһҠбһ„ бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбһҸбҹӢбһҷбһәбһҸбһҷбҹүбһ¶бһңбһ—бҹ’бһӣбһ¶бһҳбҹ—бҹ”
* **бһ”бһүбҹ’бһ бһ¶бһ‘бҹҶбһ бҹҶбһҳбһ·бһ“бһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ”бһ¶бһ“ (Fixed Size Limit)бҹ–** Array бһҳбһ·бһ“бһўбһ¶бһ…бһ–бһ„бҹ’бһҡбһёбһҖбһ‘бҹҶбһ бҹҶбһҒбҹ’бһӣбһҪбһ“бһҜбһ„бһ”бһ¶бһ“бһЎбһҫбһҷбҹ” бһ“бҹ…бһ–бҹҒбһӣбһңбһ¶бһ–бҹҒбһү бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҖбһҫбһҸ Array бһҗбҹ’бһҳбһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶбһ’бҹҶбһҮбһ¶бһ„бһҳбһ»бһ“ бһҡбһҪбһ…бһ…бһҳбҹ’бһӣбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ–бһё Array бһ…бһ¶бһҹбҹӢбһ‘бҹ…бһҗбҹ’бһҳбһё (Reallocation & Copying) бһҠбҹӮбһӣбһҹбҹҠбһёбһ’бһ“бһ’бһ¶бһ“бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҷбҹүбһ¶бһ„бһ’бҹ’бһ„бһ“бҹӢбһ’бҹ’бһ„бһҡбҹ”

### бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹЎ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбҹ’бһҡбһ¶бһӣ (бһӮбҹ’бһҡбһҪбһҹбһ¶бһҡ)бҹ– бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһҹбҹҖбһңбһ—бҹ…бһҸбһ¶бһҳбһӣбҹҶбһҠбһ¶бһ”бҹӢбһӣбһҫбһ’бҹ’бһ“бһҫ (Sequential Bookshelf Organization)

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ’бҹ’бһ“бһҫбһҹбҹҖбһңбһ—бҹ…бһҡбһ”бһҹбҹӢбһӮбҹ’бһҡбһҪбһҹбһ¶бһҡ бһҹбҹҖбһңбһ—бҹ…бһҡбһҝбһ„бһ—бһ¶бһӮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶбһҸбһҳбҹ’бһҡбҹҖбһ”бһҸбһ¶бһҳбһӣбҹҒбһҒбһҡбҹҖбһ„бһ–бһё бҹЎ бһҠбһӣбҹӢ бҹЎбҹ  бһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶бһҷбҹүбһ¶бһ„бһҹбҹ’бһўбһ¶бһҸбҹ” бһҮбҹҶбһ“бһҪбһҹбһұбҹ’бһҷбһҖбһ¶бһҡбһҡбһ»бһүбһҹбҹҖбһңбһ—бҹ…бһҡбһүбҹүбҹҒбһҡбһүбҹүбҹғ (Direct Mix Trap) бһ–бһҪбһҖбһӮбҹҒбһ”бһ¶бһ“бһҡбҹҖбһ”бһңбһ¶бһұбҹ’бһҷбһ“бҹ…бһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҷбһҫбһ„бһ…бһ„бҹӢбһҡбһҖбһҳбһҫбһӣбһҹбҹҖбһңбһ—бҹ…бһ—бһ¶бһӮ бҹҘ бһҷбһҫбһ„бһўбһ¶бһ…бһӣбһјбһҖбһҠбҹғбһ‘бҹ…бһ‘бһ¶бһүбһҷбһҖбһ”бһ¶бһ“бһ—бҹ’бһӣбһ¶бһҳбҹ— (O(1) Lookup)бҹ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ”бһҫбһҳбһ¶бһ“бһҹбһҳбһ¶бһҮбһ·бһҖбһӮбҹ’бһҡбһҪбһҹбһ¶бһҡбһҳбҹ’бһ“бһ¶бһҖбҹӢбһ‘бһ·бһү "бһҹбҹҖбһңбһ—бҹ…бһ—бһ¶бһӮбһ–бһ·бһҹбҹҒбһҹ бҹӨ.бҹҘ" бһҳбһҖ бһ бһҫбһҷбһ…бһ„бҹӢбһүбһ¶бһҸбҹӢбһңбһ¶бһ…бһјбһӣбһ…бһ“бҹ’бһӣбҹ„бҹҮбһ—бһ¶бһӮ бҹӨ бһ“бһ·бһ„бһ—бһ¶бһӮ бҹҘ бһӮбһ¶бһҸбҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҒбҹҶбһ…бһ·бһҸбҹ’бһҸбһҡбһ»бһүбһ—бһ¶бһӮ бҹҘ бһҠбһӣбҹӢбһ—бһ¶бһӮ бҹЎбҹ  бһ‘бҹ…бһҹбҹ’бһҸбһ¶бҹҶбһҳбҹ’бһҸбһ„бһҳбһҪбһҷбһҖбҹ’бһ”бһ¶бһӣ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ…бһ“бҹ’бһӣбҹ„бҹҮбһ‘бҹҶбһ“бҹҒбһҡбҹ”

### бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹЈ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ’бҹ’бһҷбһҳ (бһ’бһ»бһҡбһҖбһ·бһ…бҹ’бһ…)бҹ– бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҮбһҪбһҡбһҖбҹ…бһўбһёбһўбҹ’бһ“бһҖбһҠбҹҶбһҺбһҫбһҡбһӣбһҫбһҷбһ“бҹ’бһҸбһ бҹ„бҹҮ (Contiguous Aircraft Seating Assignments)

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҮбһҫбһ„бһ бҹ„бҹҮбһ бһҫбһҡ бһҖбҹ…бһўбһёбһӣбһҫбһҷбһ“бҹ’бһҸбһ бҹ„бҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҸбһҳбҹ’бһҡбҹҖбһ”бһҮбһ¶бһҮбһҪбһҡбҹ—бһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶бҹ” бһ“бҹ…бһ–бҹҒбһӣбһҖбҹ’бһҡбһ»бһҳбһӮбҹ’бһҡбһҪбһҹбһ¶бһҡбһҳбһҪбһҷбһ‘бһ·бһүбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһ бҹ„бҹҮбһ бһҫбһҡ бһ–бһҪбһҖбһӮбҹҒбһ…бһ„бҹӢбһ”бһ¶бһ“бһҖбҹ…бһўбһёбһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶ (Contiguous Seats) бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһҗбҹӮбһӮбҹ’бһ“бһ¶бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҷбһ“бҹ’бһҸбһ бҹ„бҹҮбһ–бҹҒбһүбһўбһҹбҹӢбһҖбҹ…бһўбһёбһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶ бһ–бһҪбһҖбһӮбҹҒбһҳбһ·бһ“бһўбһ¶бһ…бһўбһ„бҹ’бһӮбһ»бһҷбһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶бһ”бһ¶бһ“бһЎбһҫбһҷ бһ‘бҹ„бҹҮбһ”бһёбһӣбһҫбһҷбһ“бҹ’бһҸбһ бҹ„бҹҮбһҳбһ¶бһ“бһҖбҹ…бһўбһёбһ‘бҹҶбһ“бҹҒбһҡбһҡбһ¶бһҷбһ”бҹүбһ¶бһҷ (Fragmented Seats) бһ…бҹҶбһ“бһҪбһ“ бҹЎбҹ  бһҖбһ“бҹ’бһӣбҹӮбһ„бһҖбҹҸбһҠбҹ„бһҷбҹ” бһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһўбһ¶бһҖбһ¶бһҹбһ…бһҡбһҺбҹҚбһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҲбһҳбһ“бһ№бһ„бһҖбһ¶бһҡбһҸбһңбҹүбһ¶ бһ¬бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҸбһјбһҡбһҖбһ“бҹ’бһӣбҹӮбһ„бһўбһ„бҹ’бһӮбһ»бһҷбһўбҹ’бһ“бһҖбһҠбҹҶбһҺбһҫбһҡбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҷбҹүбһ¶бһ„бһ…бҹ’бһҡбһ”бһјбһҖбһ…бҹ’бһҡбһ”бһӣбҹӢбҹ”

### бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹӨ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ’бҹ’бһҷбһҳ (бһҹбһ„бҹ’бһӮбһҳ/бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„)бҹ– бһҮбһҪбһҡбһҖбҹ’бһ”бһҪбһ“бһҡбһҗбһҷбһ“бҹ’бһҸбһӮбһҺбҹҲбһ”бҹ’бһҡбһҸбһ·бһ—бһјбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡ (Sequential Diplomatic Motorcade Formations)

### бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹҘ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһ’бҹ’бһ„бһ“бҹӢ (бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„)бҹ– бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһўбһ“бһ»бһҹбҹ’бһҹбһ¶бһңбһҡбһёбһҷбҹҚбһ”бҹ’бһҡбһ…бһ¶бҹҶбһҗбҹ’бһ„бҹғбһҸбһ¶бһҳбһҖбһ¶бһӣбһ”бһҡбһ·бһ…бҹ’бһҶбҹҒбһ‘ (Rigid Chronological Memory Books)

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҹбҹ’бһ“бҹҒбһ бһ¶ бһӮбһјбһҹбҹ’бһ“бҹҒбһ бҹҚбһҒбҹ’бһӣбҹҮбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһўбһ“бһ»бһҹбҹ’бһҹбһ¶бһңбһҡбһёбһҷбҹҚбһҸбһ¶бһҳбһӣбҹҶбһҠбһ¶бһ”бҹӢбһҖбһ¶бһӣбһ”бһҡбһ·бһ…бҹ’бһҶбҹҒбһ‘бһҸбһ№бһ„бһҡбһ№бһ„ (Chronological Order) бһҠбһјбһ…бһҮбһ¶ бһҹбҹҖбһңбһ—бҹ…бһҡбһјбһ”бһҗбһҸбһўбһ“бһ»бһҹбҹ’бһҹбһ¶бһңбһҡбһёбһҷбҹҚ бһҠбҹӮбһӣбһҡбһјбһ”бһҗбһҸбһ“бһёбһҳбһҪбһҷбҹ—бһҸбҹ’бһҡбһјбһңбһ”бһ·бһ‘бһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶бһҸбһ¶бһҳбһӣбҹҶбһҠбһ¶бһ”бҹӢбһҗбҹ’бһ„бҹғбһҒбҹӮбһҶбҹ’бһ“бһ¶бҹҶбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҗбҹ’бһ„бҹғбһҺбһ¶бһҳбһҪбһҷбһ–бһҪбһҖбһӮбҹҒбһ…бһ„бҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳ "бһҡбһјбһ”бһҗбһҸбһ…бһ¶бһҹбҹӢбһҠбҹҸбһҳбһ¶бһ“бһ“бҹҗбһҷбһҳбһҪбһҷбһҹбһ“бҹ’бһӣбһ№бһҖ" бһҠбҹӮбһӣбһҡбһҖбһғбһҫбһүбһЎбһҫбһ„бһңбһ·бһүбһ…бҹҶбһҖбһҺбҹ’бһҸбһ¶бһӣбһҹбҹҖбһңбһ—бҹ… бһ–бһҪбһҖбһӮбҹҒбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҒбҹҶбһ…бһ·бһҸбҹ’бһҸбһ”бһҖ бһ“бһ·бһ„бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҡбһјбһ”бһҗбһҸбһҡбһ¶бһ”бҹӢбһҹбһ·бһ”бһҹбһ“бҹ’бһӣбһ№бһҖбһ–бһёбһҖбҹ’бһҡбҹ„бһҷбһҗбһҷбһҖбҹ’бһҡбҹ„бһҷбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ бһҠбҹӮбһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҒбһјбһ…бһҒбһ¶бһҸбһҖбҹ’бһҡбһҠбһ¶бһҹбһҹбҹҖбһңбһ—бҹ… бһ¬бһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбһҹбһҺбҹ’бһҸбһ¶бһ”бҹӢбһ’бҹ’бһ“бһ¶бһ”бҹӢбһҠбһҫбһҳбҹ”

## бҹӨ. бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бһјбһ‘бҹ…бҹ– бһҸбһ»бһӣбҹ’бһҷбһ—бһ¶бһ–бһҡбһңбһ¶бһ„ Arrays бһ“бһ·бһ„ Dynamic Resizing Mechanics (The General Solution: Optimizing Array Allocations and Dynamic Resizing)

бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбҹ’бһҹбһ¶бһ”бһ¶бһ“бһ“бһјбһңбһӣбҹ’бһ”бһҝбһ“ Direct Access O(1) бһҡбһ”бһҹбҹӢ Array бһ“бһ·бһ„бһ…бҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһҒбһ¶бһҸбһ”бһ„бҹӢбһҗбһ¶бһҳбһ–бһӣбһӮбһҺбһ“бһ¶бһӣбһҫ Shifting/Reallocation бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһўбһ“бһ»бһңбһҸбҹ’бһҸбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҠбҹҸбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ–

1. **бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Dynamic Arrays (ArrayList/Vector)бҹ–** бһ“бҹ…бһ–бҹҒбһӣ Array бһ–бҹҒбһү бһңбһ¶бһ“бһ№бһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖ Memory бһҗбҹ’бһҳбһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶбһ’бҹҶбһҮбһ¶бһ„бһҳбһ»бһ“ бҹў бһҠбһ„ (Capacity * 2) бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбҹ” бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ“бҹҒбҹҮбһҮбһҪбһҷбһұбҹ’бһҷбһҖбһ¶бһҡ Reallocation бһҖбһҫбһҸбһЎбһҫбһ„бһҸбһ·бһ…бһҸбһҪбһ…бһ”бҹҶбһ•бһ»бһҸ (Amortized O(1) Insertion)бҹ”
2. **бһҖбҹҶбһҺбһҸбҹӢбһ‘бҹҶбһ бҹҶбһ‘бһ»бһҖбһҮбһ¶бһҳбһ»бһ“ (Pre-allocation)бҹ–** бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҷбһҫбһ„бһҠбһ№бһ„бһҮбһ¶бһҳбһ»бһ“бһҗбһ¶бһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бҹҶбһ“бһҪбһ“ бҹЎ бһҳбҹүбһәбһ“бһ’бһ¶бһҸбһ» бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҖбһҫбһҸ Array бһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶ бҹЎ бһҳбҹүбһәбһ“бһҮбһ¶бһҳбһ»бһ“ бһҠбһҫбһҳбҹ’бһ”бһёбһ…бҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡ Resizing бһҘбһҸбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбһ„бҹ”
3. **бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ Hybrid Structure бһұбҹ’бһҷбһҸбҹ’бһҡбһјбһңбһ“бһ№бһ„бһҖбһ¶бһҡбһ„бһ¶бһҡбҹ–** 
   * бһ”бһҫбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһ…бһҳбҹ’бһ”бһ„бһӮбһәбһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Access/Read) вһ” бһ”бҹ’бһҡбһҫ **Array**бҹ”
   * бһ”бһҫбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһ…бһҳбҹ’бһ”бһ„бһӮбһәбһҖбһ¶бһҡбһ”бһ“бҹ’бһҗбҹӮбһҳ бһ¬бһӣбһ»бһ”бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹ…бһҖбһҺбҹ’бһҸбһ¶бһӣ (Insert/Delete) вһ” бһ”бҹ’бһҡбһҫ **Linked List** бһҮбҹҶбһ“бһҪбһҹбһңбһ·бһүбҹ”
4. **бһ’бҹ’бһңбһҫбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҮбһ¶бһҖбһүбҹ’бһ…бһ”бҹӢ (Batch Operations)бҹ–** бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҸбҹ’бһҡбһјбһңбһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бҹ’бһҡбһҫбһ“ бһҸбҹ’бһҡбһјбһңбһ”бһ“бҹ’бһҗбҹӮбһҳбһңбһ¶бһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷбһҮбһ¶бһҮбһ¶бһ„бһҹбһҡбһҹбҹҒбһҡ Loop бһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбҹ’бһҸбһ„бһҳбһҪбһҷбҹ— бһҠбһҫбһҳбҹ’бһ”бһёбһ…бҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡ Shifting Memory бһҠбһҠбҹӮбһӣбҹ—бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбһ„бҹ”

## рҹҗҮ бһ’бҹ’бһӣбһ¶бһҖбҹӢбһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһҡбһ“бҹ’бһ’бһ‘бһ“бҹ’бһҹбһ¶бһҷ (Enter the Rabbit Hole)

бһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ…бһ¶бһҡбһҖбһҳбҹ’бһҳбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҡбһ”бһҹбҹӢбһ–бҹ’бһҡбҹҮбһҡбһ¶бһҮбһ¶ бһ¬бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ•бҹ’бһ“бҹӮбһҖбһ‘бһ“бҹӢ бһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ•бҹӮбһ“бһ‘бһёбһ”бһ„бҹ’бһ бһ¶бһүбһ•бҹ’бһӣбһјбһңбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ "бһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҠбҹӮбһӣбһ…бһ„бҹ’бһўбһ»бһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ”бһ“бҹ’бһҸбһӮбҹ’бһ“бһ¶" (Linked Lists) бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһ–бһҪбһҖбһӮбҹҒбһўбһ¶бһ…бһ”бһ“бҹ’бһҗбҹӮбһҳ бһ¬бһӣбһ»бһ”бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһҖбһ¶бһҡбһ„бһ¶бһҡбһ”бһ¶бһ“бһ—бҹ’бһӣбһ¶бһҳбҹ— O(1) бһҠбҹ„бһҷбһҳбһ·бһ“бһ”бһ¶бһ…бҹӢбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһё бһ¬бһҡбҹҖбһ”бһҮбһҪбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбһ…бһ¶бһҹбҹӢбһ‘бһ¶бһӣбҹӢбһҸбҹӮбһҹбҹ„бҹҮ бһҹбһјбһҳбһ”бһ“бҹ’бһҸбһҠбҹҶбһҺбһҫбһҡбһ‘бҹ…бһҖбһ¶бһ“бҹӢбҹ–

* рҹҡҖ **[бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹҶбһҺбһҫбһҡбһҡбһ»бһҖбһҡбһҖ (Start the Journey) вһ” Linked Lists and Dynamic Pointer Chains](./99-the-spys-treasure-hunt.md)**

## бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“ (Conclusion)

> **В«бһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„бһҡбһ”бһҹбҹӢбһўбһ¶бһҡбҹҒ бһӮбһәбһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһӣбһҫбһ—бһ¶бһ–бһҡбһ№бһ„бһҳбһ¶бҹҶ бһ“бһ·бһ„бһӣбҹҶбһҠбһ¶бһ”бҹӢбһӣбҹҶбһҠбҹ„бһҷбһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ—бһ¶бһ–бһ‘бһ“бҹӢбһ—бҹ’бһӣбһ“бҹӢбһ“бҹғбһҖбһ¶бһҡбһӣбһјбһҸбһӣбһ¶бһҹбҹӢ бһ‘бһ¶бһҳбһ‘бһ¶бһҡбһұбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶ capacity бһҠбҹҸбһңбҹғбһҶбҹ’бһӣбһ¶бһҸВ»**

бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҷбһ“бҹ’бһҸбһҖбһ¶бһҡбһҖбһ¶бһҡбһ„бһ¶бһҡ RAM бһҡбһ”бһҹбҹӢ Array бһҮбһҪбһҷбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢ бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ‘бҹҶбһ бҹҶбһ‘бһ»бһҖбһҮбһ¶бһҳбһ»бһ“бһ”бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„бһ‘бһ¶бһүбһҷбһҖбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ–бһёбһӣбҹ’бһ”бһҝбһ“ direct access O(1) бһҠбҹҸбһўбһҹбҹ’бһ…бһ¶бһҡбҹ’бһҷбһ”бҹҶбһ•бһ»бһҸ бһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһҒбһ¶бһҸбһ”бһ„бҹӢбһ’бһ“бһ’бһ¶бһ“бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһЎбһҫбһҷбҹ”
