  - [бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹӨ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ’бҹ’бһҷбһҳ (бһҹбһ„бҹ’бһӮбһҳ/бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„)бҹ– бһҖбһ¶бһҡбһ”бҹ’бһҡбһҮбһ»бҹҶбһ‘бһ¶бһҳбһ‘бһ¶бһҡбһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһҖбһ¶бһҡбһ„бһ¶бһҡбһҡбһ¶бһӣбҹӢбһҗбҹ’бһ„бҹғ (Micromanaged Status Meetings vs Aggregated Weekly Dashboard)](#3-4)

## бһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢбһ•бҹ’бһӣбһјбһңбһ…бһ·бһҸбҹ’бһҸ (The Trap)

бһҸбһҫбһўбҹ’бһ“бһҖбһ’бҹ’бһӣбһ¶бһ”бҹӢбһҮбһҪбһ”бһ”бһүбҹ’бһ бһ¶бһҠбҹӮбһӣбһ‘бҹҶбһ–бҹҗбһҡбһҠбһҫбһҳ (Homepage) бһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһҡбһ бһјбһҸбһҠбһӣбҹӢбһ‘бҹ… бҹҘ бһңбһ·бһ“бһ¶бһ‘бһёбһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһүбҹӢбһҹбҹҶбһҺбһҫ (Requests) бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҠбһ‘бҹғбһ‘бҹҖбһҸбһ…бҹҶбһ“бһҪбһ“ бҹЎбҹ  бһ•бҹ’бһҹбҹҒбһ„бһӮбҹ’бһ“бһ¶бһҮбһ¶бһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶бһҠбҹӮбһҡбһ¬бһ‘бҹҒ?

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ’бһ“бһ’бһ¶бһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбҹ–
* **бһҷбһҫбһ„бһ„бһ¶бһҷбһ“бһ№бһ„бһ’бҹ’бһӣбһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„бһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢ** бһ“бҹғбһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбҹ„бһҷбһұбҹ’бһҷ Client бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһҪбһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбһ¶бһ…бҹӢбһҠбҹ„бһҷбһЎбҹӮбһҖбһ–бһёбһӮбҹ’бһ“бһ¶ (Chatty APIs) бһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһұбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһҹбҹ’бһ‘бҹҮбһ”бһҺбҹ’бһҸбһ¶бһү (Network Overhead) бһ“бһ·бһ„бһҖбһ¶бһҡбһҒбҹ’бһҮбҹҮбһҒбҹ’бһҮбһ¶бһҷбһҗбһ¶бһҳбһ–бһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бҹ”
* **бһҷбһҫбһ„бһҳбһҫбһӣбһҡбҹҶбһӣбһ„** бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣ (Aggregator) бһҠбҹӮбһӣбһ…бҹҒбҹҮбһ”бһ„бҹ’бһҡбһҪбһ”бһ”бһ„бҹ’бһҡбһҪбһҳбһҹбҹҶбһҺбһҫбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҳбһҖбһ’бҹ’бһңбһҫбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷ бһ“бһ·бһ„бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Cache (Strategic Caching) бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ—бҹ’бһӣбһ¶бһҳбҹ—бһ“бҹ…бһ–бҹҒбһӣбһҳбһ¶бһ“бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бһ“бҹ’бһ‘бһ¶бһ“бҹӢбҹ”

бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ‘бҹҶбһ бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһҺбҹ’бһҸбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“ бһ бҹ…бһҗбһ¶ **бһҷбһ“бҹ’бһҸбһҖбһ¶бһҡбһ”бһ„бҹ’бһҡбһҪбһ”бһ”бһ„бҹ’бһҡбһҪбһҳ API бһ“бһ·бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһҺбҹ’бһҸбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“ (API Aggregation & Strategic Caching)**бҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҷбһ“бҹ’бһҸбһҖбһ¶бһҡбһ“бҹҒбҹҮ бһ“бҹҒбҹҮбһҮбһ¶бһ•бҹӮбһ“бһ‘бһёбһ”бһ„бҹ’бһ бһ¶бһүбһ•бҹ’бһӣбһјбһңбҹ–
1. **бһҡбһҝбһ„бһ–бҹ’бһҡбҹҒбһ„бһ“бһ·бһ‘бһ¶бһ“ (The Legend)** вҖ” бһҡбһҝбһ„бһҡбҹүбһ¶бһңбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҠбҹӮбһӣбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһҺбҹ’бһҸбҹҒбһүбһ…бҹҒбһү бһ бһҫбһҷбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбҹҶбһҺбһ¶бһ…бһҠбҹӮбһӣбһ“бҹ…бһҹбҹҒбһҹбһҹбһӣбҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ”бҹҶбһҺбһ»бһӣбһҡбһ”бһҹбҹӢбһҖбһјбһ“бһ”бҹҶбһҺбһ»бһӣбһҗбҹ…бһҖбҹӮ бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһ¶бһ„бһ‘бҹҶбһ“бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһўбһ“бһ¶бһӮбһҸбҹ”
2. **бһ”бһүбҹ’бһ бһ¶ (The Issue)** вҖ” бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһӣбһҫбһҖбһ¶бһҡбһҒбһ¶бһҸбһ”бһ„бҹӢ Latency бһ–бһёбһҖбһ¶бһҡбһ бҹ… API бһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶ бһ“бһ·бһ„бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ Gateway Aggregationбҹ”
3. **бһ§бһ‘бһ¶бһ бһҳбһҺбҹҚбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„ (Real World Examples)** вҖ” бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһ”бһүбҹ’бһ бһ¶бһ“бҹҒбҹҮбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҡбһ·бһҸбһӮбҹ’бһҡбһҪбһҹбһ¶бһҡ бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶ бһ’бһ»бһҡбһҖбһ·бһ…бҹ’бһ… бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бҹ”
4. **бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бһјбһ‘бҹ… (The General Solution)** вҖ” бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ GraphQL/BFF (Backend-For-Frontend) бһ“бһ·бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ Cache-Aside Patternбҹ”

## бҹЎ. бһҡбһҝбһ„бһ–бҹ’бһҡбҹҒбһ„бһ“бһ·бһ‘бһ¶бһ“бҹ– бһўбҹ’бһ“бһҖбһ…бһ¶бһҸбҹӢбһ…бҹӮбһ„бһ‘бҹ’бһҡбһ–бҹ’бһҷбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ·бһҠбҹӮбһӣбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһҺбҹ’бһҸбҹҒбһүбһ…бҹҒбһү (The Legend of the Shrewd Manager)

бһҹбҹҒбһҠбҹ’бһӢбһёбһҳбҹ’бһ“бһ¶бһҖбҹӢбһҳбһ¶бһ“бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ‘бҹ’бһҡбһ–бҹ’бһҷбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ·бһҳбҹ’бһ“бһ¶бһҖбҹӢ (Manager)бҹ” бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһӣбһәбһ–бһ¶бһҖбҹ’бһҷбһ…бһ…бһ¶бһҳбһўбһ¶бһҡбҹүбһ¶бһҳбһҗбһ¶ бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ“бҹ„бҹҮбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҖбҹҒбһ„бһ…бҹҶбһҺбҹҒбһү бһ“бһ·бһ„бһҒбҹ’бһҮбҹҮбһҒбҹ’бһҮбһ¶бһҷбһ‘бҹ’бһҡбһ–бҹ’бһҷбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ· бһҹбҹҒбһҠбҹ’бһӢбһёбһҖбҹҸбһ”бһ¶бһ“бһ бҹ…бһӮбһ¶бһҸбҹӢбһҳбһҖ бһ бһҫбһҷбһ”бҹ’бһҡбһ¶бһ”бҹӢбһҗбһ¶бҹ– *"бһҒбҹ’бһүбһ»бҹҶбһ“бһ№бһ„бһ”бһүбҹ’бһҲбһ”бҹӢбһҜбһ„бһ–бһёбһҖбһ¶бһҡбһ„бһ¶бһҡбһ бһҫбһҷ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһ…бһјбһҡбһҜбһ„бһ‘бҹ…бһҡбҹҖбһ”бһ…бҹҶбһ”бһүбҹ’бһҮбһёбһҹбҹ’бһ“бһ¶бһҳбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҳбһҖбһ”бҹ’бһҡбһӮбһӣбҹӢбһұбҹ’бһҷбһҒбҹ’бһүбһ»бҹҶбһңбһ·бһүбһұбҹ’бһҷбһ”бһ¶бһ“бһҶбһ¶бһ”бҹӢ!"*

### бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ”бҹҶбһҺбһ»бһӣ бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ•бһӣбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһўбһ“бһ¶бһӮбһҸ (Debt Reduction Strategy and Long-term Relationship Capital)

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһӮбһ¶бһҸбҹӢбһ“бһ№бһҖбһғбһҫбһүбһӣбҹ’бһ”бһ·бһ…бһҳбһҪбһҷбҹ” бһӮбһ¶бһҸбҹӢбһ”бһ¶бһ“бһҖбҹ„бҹҮбһ бҹ…бһҖбһјбһ“бһ”бҹҶбһҺбһ»бһӣбһҡбһ”бһҹбҹӢбһҗбҹ…бһҖбҹӮбһӮбһ¶бһҸбҹӢбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҳбһҖбһҮбһҪбһ”бһҮбһ¶бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбҹ–
* бһӮбһ¶бһҸбҹӢбһҹбһҪбһҡбһўбҹ’бһ“бһҖбһ‘бһё бҹЎ бһҗбһ¶бҹ– *"бһҸбһҫбһўбҹ’бһ“бһҖбһҮбҹҶбһ–бһ¶бһҖбҹӢбһ”бҹ’бһҡбҹҒбһ„бһҗбҹ…бһҖбҹӮбһҒбҹ’бһүбһ»бҹҶбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“?"* бһҖбһјбһ“бһ”бҹҶбһҺбһ»бһӣбһҶбҹ’бһӣбһҫбһҷбһҗбһ¶бҹ– *"бҹЎбҹ бҹ  бһ’бһ»бһ„"бҹ”* бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бҹ’бһҡбһ¶бһ”бҹӢбһҗбһ¶бҹ– *"бһҷбһҖбһҖбҹ’бһҡбһҠбһ¶бһҹбһ”бҹҶбһҺбһ»бһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһҖ бһҖбҹӮбһҹбһҡбһҹбҹҒбһҡбһҸбһҳбҹ’бһҡбһ„бһ…бһҡбһ¶бһ…бһҡбһҺбҹҚбһҸбҹӮ **бҹҘбҹ  бһ’бһ»бһ„** бһ”бһ¶бһ“бһ бһҫбһҷ (бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ”бһ“бҹ’бһ‘бһ»бһҖ бҹҘбҹ %)бҹ”"*
* бһӮбһ¶бһҸбҹӢбһҹбһҪбһҡбһўбҹ’бһ“бһҖбһ‘бһё бҹў бһҗбһ¶бҹ– *"бһ…бһ»бҹҮбһўбҹ’бһ“бһҖбһҮбҹҶбһ–бһ¶бһҖбҹӢбһҹбҹ’бһҡбһјбһңбһҹбһ¶бһӣбһёбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“?"* бһҖбһјбһ“бһ”бҹҶбһҺбһ»бһӣбһҶбҹ’бһӣбһҫбһҷбһҗбһ¶бҹ– *"бҹЎбҹ бҹ  бһ”бһ¶бһң"бҹ”* бһӮбһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһ”бҹӢбһҗбһ¶бҹ– *"бһҹбһҡбһҹбҹҒбһҡбһҸбҹӮ **бҹЁбҹ  бһ”бһ¶бһң** бһ”бһ¶бһ“бһ бһҫбһҷ (бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ”бһ“бҹ’бһ‘бһ»бһҖ бҹўбҹ %)бҹ”"*

бһӮбһ¶бһҸбҹӢбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһ”бҹӮбһ”бһ“бҹҒбҹҮбһ…бҹҶбһ–бҹ„бҹҮбһҖбһјбһ“бһ”бҹҶбһҺбһ»бһӣбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷбһ–бһҪбһҖбһӮбҹҒбһҳбһ¶бһ“бһҖбһ¶бһҡбһҠбһ№бһ„бһӮбһ»бһҺбһ…бҹҶбһ–бҹ„бҹҮбһӮбһ¶бһҸбҹӢбҹ” бһ–бҹҒбһӣбһҠбҹӮбһӣбһӮбһ¶бһҸбҹӢбһҸбҹ’бһҡбһјбһңбһ”бһүбҹ’бһҲбһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһ„бһ¶бһҡ бһҖбһјбһ“бһ”бҹҶбһҺбһ»бһӣбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹ„бҹҮбһ“бһ№бһ„бһҹбҹ’бһңбһ¶бһӮбһҳбһ“бҹҚбһӮбһ¶бһҸбҹӢбһұбҹ’бһҷбһ‘бҹ…бһҡбһҹбҹӢбһ“бҹ…бһҮбһ¶бһҳбһҪбһҷбҹ”

бһ“бҹ…бһ–бҹҒбһӣбһҗбҹ…бһҖбҹӮбһҠбһ№бһ„бһҡбһҝбһ„бһ“бҹҒбҹҮ бһҗбҹ…бһҖбҹӮбһҳбһ·бһ“бһ”бһ¶бһ“бһҒбһ№бһ„бһ‘бҹҒ бһ”бҹӮбһҡбһҮбһ¶ **бһҹбһҡбһҹбһҫбһҡбһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ‘бһ»бһ…бҹ’бһ…бһҡбһ·бһҸбһ“бҹ„бҹҮ бһҠбҹӮбһӣбһ…бҹҒбҹҮбһ…бһ¶бһҸбҹӢбһ…бҹӮбһ„бһҖбһ¶бһҡбһ„бһ¶бһҡбһҷбҹүбһ¶бһ„бһҲбҹ’бһӣбһ¶бһҹбһңбҹғ (Shrewdly)** бһ‘бҹ…бһңбһ·бһүбҹ” бһ‘бҹ’бһҡбһ„бҹӢбһ…бһ„бҹӢбһ”бһ„бҹ’бһҡбҹҖбһ“бһұбҹ’бһҷбһҷбһҫбһ„бһ…бҹҒбҹҮбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ’бһ“бһ’бһ¶бһ“бһ”бһҺбҹ’бһҸбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҸбһҳбҹ’бһӣбҹғбһҹбҹ’бһҗбһ·бһҡбһ—бһ¶бһ–бһҷбһјбһҡбһўбһ„бҹ’бһңбҹӮбһ„бҹ”

## бҹў. бһ”бһүбҹ’бһ бһ¶бҹ– бһҖбһ¶бһҡбһ бҹ…бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбһ»бҹҮбһҹбҹ’бһӣбҹӮ бһ“бһ·бһ„бһ”бһ“бҹ’бһ‘бһ»бһҖбһ”бһҺбҹ’бһҸбһ¶бһүбһ…бҹ’бһҡбһҫбһ“бһ бһҪбһҹбһ”бҹ’бһҡбһҳбһ¶бһҺ (The Issue: Network Overhead and Redundant Downstream Requests)

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹ’бһңбһҖбһҳбҹ’бһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„ API Designбҹ–
1. **бһ”бһүбҹ’бһ бһ¶бһ бҹ… Request бһ…бҹ’бһҡбһҫбһ“бһҸбһ„бҹӢ (Underfetching/Overfetching)бҹ–** бһ“бҹ…бһ–бҹҒбһӣ UI бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“ (User Profile, Orders, Notifications, and Recommendations)бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Client бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһүбҹӢ Request бһ‘бҹ…бһҖбһ¶бһ“бҹӢ API бһ“бһёбһҳбһҪбһҷбҹ—бһҠбһ¶бһ…бҹӢбһҠбҹ„бһҷбһЎбҹӮбһҖбһ–бһёбһӮбҹ’бһ“бһ¶ бһңбһ¶бһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢ TCP Handshake бһ…бҹ’бһҡбһҫбһ“бһҠбһ„ бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҷбһәбһҸбһҒбҹ’бһӣбһ¶бҹҶбһ„бһӣбһҫбһ”бһҺбҹ’бһҸбһ¶бһү 3G/4Gбҹ”
2. **бһ”бһ“бҹ’бһ‘бһ»бһҖбһӣбһҫ Database бһ…бһҳбҹ’бһ”бһ„ (Database Exhaustion)бҹ–** бһҡбһ¶бһӣбҹӢбһҖбһ¶бһҡбһҹбһҪбһҡбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҠбҹӮбһӣбһҳбһ·бһ“бһҹбһјбһңбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ (Static Data - бһҠбһјбһ…бһҮбһ¶ бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“ Profile) бһ‘бҹ…бһҖбһ¶бһ“бҹӢ Database бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһҳбһ¶бһ“бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһҒбҹ’бһҮбҹҮбһҒбҹ’бһҮбһ¶бһҷбһ’бһ“бһ’бһ¶бһ“бҹ”

бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮбһҮбһ¶бһҖбһ¶бһҡбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһҡбһңбһ¶бһ„бһҖбһ¶бһҡбһ бҹ…бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҶбҹ… (Fragile) бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ API Aggregation бһҡбһҪбһҳбһҮбһ¶бһҳбһҪбһҷ Redis Cache (Resilient)бҹ–

### Fragile Implementation (Uncached Sequential REST Calls)
бһҖбһјбһҠбһ“бҹҒбҹҮбһ бҹ… API бһ“бһёбһҳбһҪбһҷбҹ—бһҠбһ¶бһ…бҹӢбһҠбҹ„бһҷбһЎбҹӮбһҖбһ–бһёбһӮбҹ’бһ“бһ¶бһҮбһ¶бһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ (Sequential) бһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбҹ’бһ“бһ»бһ„ Cache бһЎбһҫбһҷ бһ“бһ¶бҹҶбһұбҹ’бһҷбһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһҷбһјбһҡбҹ–

```typescript
// fragile_aggregator.ts
import { fetchUserProfile, fetchUserOrders, fetchUserNotifications } from './services';

export async function getUserDashboardSequential(userId: string): Promise<any> {
    console.log("[INFO] Running fragile sequential API fetches...");
    
    // бҹЎ. бһ бҹ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ‘бһё бҹЎ (бһҡбһ„бҹӢбһ…бһ¶бҹҶ бҹЎ бһңбһ·бһ“бһ¶бһ‘бһё)
    const profile = await fetchUserProfile(userId);
    
    // бҹў. бһ бҹ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ‘бһё бҹў (бһҡбһ„бҹӢбһ…бһ¶бҹҶ бҹЎ.бҹҘ бһңбһ·бһ“бһ¶бһ‘бһё)
    const orders = await fetchUserOrders(userId);
    
    // бҹЈ. бһ бҹ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ‘бһё бҹЈ (бһҡбһ„бҹӢбһ…бһ¶бҹҶ бҹ .бҹҘ бһңбһ·бһ“бһ¶бһ‘бһё)
    const notifications = await fetchUserNotifications(userId);

    // бһҹбһҡбһ»бһ”бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҡбһ„бҹӢбһ…бһ¶бҹҶбһӮбһәбҹ– бҹЈ бһңбһ·бһ“бһ¶бһ‘бһё (бһӮбһ¶бҹҶбһ„бһҷбһјбһҡбһҒбҹ’бһӣбһ¶бҹҶбһ„)
    return {
        profile,
        orders,
        notifications
    };
}
```

### Resilient Implementation (Parallel Gateway Aggregator with Redis Cache)
бһҖбһјбһҠбһ“бҹҒбҹҮбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ `Promise.all` бһҠбһҫбһҳбҹ’бһ”бһёбһ бҹ…бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷ (Parallel) бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸ Cache-Aside Pattern бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Redis бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ”бҹҶбһҺбһ»бһӣбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҡбһ„бҹӢбһ…бһ¶бҹҶбһҳбһҖбһҸбҹ’бһҡбһ№бһҳ O(1)бҹ–

```typescript
// resilient_aggregator.ts
import { fetchUserProfile, fetchUserOrders, fetchUserNotifications } from './services';
import { redisClient } from './redis';

export async function getUserDashboardResilient(userId: string): Promise<any> {
    const cacheKey = `dashboard:user:${userId}`;

    // бҹЎ. бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһ‘бһ¶бһүбһҷбһҖбһ–бһё Cache бһҮбһ¶бһҳбһ»бһ“бһҹбһ·бһ“ (Strategic Caching)
    try {
        const cachedData = await redisClient.get(cacheKey);
        if (cachedData) {
            console.log("[CACHE HIT] Returning aggregated data from cache.");
            return JSON.parse(cachedData);
        }
    } catch (err) {
        console.error("[WARN] Redis error, fallback to parallel direct fetches", err);
    }

    console.log("[CACHE MISS] Executing parallel dynamic aggregation...");

    // бҹў. бһ бҹ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷ (Parallel Aggregation)
    const [profile, orders, notifications] = await Promise.all([
        fetchUserProfile(userId),
        fetchUserOrders(userId),
        fetchUserNotifications(userId)
    ]);

    const aggregatedResult = {
        profile,
        orders,
        notifications,
        timestamp: Date.now()
    };

    // бҹЈ. бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһӣбһ‘бҹ’бһ’бһ•бһӣбһҖбҹ’бһ“бһ»бһ„ Cache бһҡбһҷбҹҲбһ–бҹҒбһӣ бҹЎбҹ  бһ“бһ¶бһ‘бһё (TTL: 600 seconds)
    try {
        await redisClient.set(cacheKey, JSON.stringify(aggregatedResult), {
            EX: 600
        });
    } catch (err) {
        console.error("[ERROR] Failed to save to cache", err);
    }

    return aggregatedResult;
}
```

бһҹбһҳбһ¶бһҮбһ·бһҖбһӮбҹ’бһҡбһҪбһҹбһ¶бһҡбһҳбҹ’бһ“бһ¶бһҖбҹӢ бһ”бһҫбһҖбһЎбһ¶бһ“бһ‘бҹ…бһ•бҹ’бһҹбһ¶бһҡбһ‘бһ·бһүбһҒбҹ’бһ‘бһ№бһҳбһҹбһ“бҹ…бһҳбҹүбҹ„бһ„ бҹЁ бһ–бҹ’бһҡбһ№бһҖ бһҡбһҪбһ…бһ”бһҫбһҖбһ‘бҹ…бһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸбһ‘бһ·бһүбһҹбһ¶бһ…бҹӢбһҳбһ¶бһ“бҹӢбһ“бҹ…бһҳбҹүбҹ„бһ„ бҹЎбҹ  бһ–бҹ’бһҡбһ№бһҖ бһ“бһ·бһ„бһ‘бҹ…бһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸбһ‘бһ·бһүбһ‘бһ№бһҖбһҠбҹ„бҹҮбһӮбҹ„бһ“бҹ…бһҳбҹүбҹ„бһ„ бҹў бһҡбһҹбҹҖбһӣбҹ” бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһ“бҹҒбҹҮбһҒбһ¶бһҸбһ”бһ„бҹӢбһ”бҹ’бһҡбҹҒбһ„бһЎбһ¶бһ“ бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶ бһ“бһ·бһ„бһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бҹ” бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ– бһ–бһҪбһҖбһӮбҹҒбһӮбһҪбһҡбһҸбҹӮбһҹбһҡбһҹбҹҒбһҡбһ”бһүбҹ’бһҮбһёбһҡбһҪбһҳбһӮбҹ’бһ“бһ¶ (Aggregation) бһ“бһ·бһ„бһ‘бҹ…бһ‘бһ·бһүбһҸбҹӮбһҳбҹ’бһҸбһ„бһӮбһҸбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ–бҹҒбһүбһҳбһҪбһҷбһҹбһ”бҹ’бһҸбһ¶бһ бҹҚ (Caching)бҹ”

### бһ§бһ‘бһ¶бһ бһҳбһҺбҹҚбһ‘бһё бҹЈ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ’бҹ’бһҷбһҳ (бһ’бһ»бһҡбһҖбһ·бһ…бҹ’бһ…)бҹ– бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһ‘бһ·бһүбһ‘бҹҶбһ“бһ·бһүбһҡбһ¶бһҷбһ–бһёбһўбҹ’бһ“бһҖбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһ…бҹ’бһҡбһҫбһ“бһҖбһ“бҹ’бһӣбҹӮбһ„ (Fragmented Bulk Ordering vs Aggregated Volume Discount)

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһӣбһҖбҹӢбһӮбҹ’бһҡбһҝбһ„бһҹбҹҶбһҺбһ„бҹӢбһҳбҹ’бһ“бһ¶бһҖбҹӢ бһ”бһ¶бһ“бһ‘бһ·бһүбһҠбҹӮбһҖбһҗбҹӮбһ”бһ–бһёбһҡбһ”бһёбһҸбҹ„бһ“бһҡбһ¶бһӣбҹӢбһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһ–бһё ЩҒбһҡбҹ„бһ„бһ…бһҖбҹ’бһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—бһӮбҹ’бһ“бһ¶ бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһӮбһ¶бһҸбҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹӢбһҗбҹ’бһӣбҹғбһҠбһ№бһҖбһҮбһүбҹ’бһҮбһјбһ“бһҗбҹ’бһӣбҹғ бһ“бһ·бһ„бһҳбһ·бһ“бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһ”бҹ„бҹҮбһҠбһ»бҹҶбҹ” бһҖбҹ’бһҡбҹ„бһҷбһҳбһҖ бһӮбһ¶бһҸбҹӢбһ”бһ¶бһ“бһ”бҹ’бһҸбһјбһҡбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹ„бһҷбһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһҡбһ¶бһӣбҹӢбһ”бһүбҹ’бһҮбһ¶бһ‘бһ·бһүбһҡбһ”бһҹбҹӢбһўбһҸбһ·бһҗбһ·бһҮбһ“бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣ бҹЈ бһҒбҹӮбһ”бһүбҹ’бһ…бһјбһӣбһӮбҹ’бһ“бһ¶ (Aggregation) бһҡбһҪбһ…бһ”бһүбҹ’бһҮбһ¶бһ‘бһ·бһүбһҳбҹ’бһҸбһ„бһӮбһҸбҹӢбһ…бҹҶбһ“бһҪбһ“ бҹҘбҹ  бһҸбҹ„бһ“ бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһӮбһ¶бһҸбҹӢбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ»бҹҮбһҸбһҳбҹ’бһӣбҹғбһҡбһ бһјбһҸбһҠбһӣбҹӢ бҹЈбҹ %бҹ”

### бһ§бһ‘бһ¶бһ бһҳбһҺбҹҚбһ‘бһё бҹӨ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ’бҹ’бһҷбһҳ (бһҹбһ„бҹ’бһӮбһҳ/бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„)бҹ– бһҖбһ¶бһҡбһ”бҹ’бһҡбһҮбһ»бҹҶбһ‘бһ¶бһҳбһ‘бһ¶бһҡбһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһҖбһ¶бһҡбһ„бһ¶бһҡбһҡбһ¶бһӣбҹӢбһҗбҹ’бһ„бҹғ (Micromanaged Status Meetings vs Aggregated Weekly Dashboard)

бһ“бһ¶бһҷбһҖбһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһҳбҹ’бһ“бһ¶бһҖбҹӢбһ”бһ¶бһ“бһ бҹ…бһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖбһҡбҹҖбһ”бһҡбһ¶бһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһ„бһ¶бһҡбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒбһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһ–бҹ’бһҡбһ№бһҖ (Daily Meeting) бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖбһҒбһ¶бһҸбһ–бҹҒбһӣ бҹЎ бһҳбҹүбҹ„бһ„бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһҗбҹ’бһ„бҹғбһҠбһҫбһҳбҹ’бһ”бһёбһҡбҹҖбһ”бһ…бҹҶбһҒбҹ’бһӣбһҪбһ“бҹ” бһҖбҹ’бһҡбҹ„бһҷбһҳбһҖ бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҳбҹ’бһ“бһ¶бһҖбҹӢбһ”бһ¶бһ“бһ”бһ„бҹ’бһҡбһҪбһ”бһ”бһ„бҹ’бһҡбһҪбһҳбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҖбһ¶бһҡбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ‘бҹ…бһӣбһҫ Dashboard (бһҠбһјбһ…бһҮбһ¶ Jira/Trello) бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһҹбһ„бҹ’бһҒбҹҒбһ”бһҸбҹӮбһҳбҹ’бһҸбһ„бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҹбһ”бҹ’бһҸбһ¶бһ бҹҚ бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖбһҳбһ¶бһ“бһ–бҹҒбһӣбһҹбҹ’бһ„бһ”бҹӢбһҹбҹ’бһ„бһ¶бһҸбҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһ“бһ•бһӣбһ·бһҸбһ—бһ¶бһ–бһҖбһ¶бһҡбһ„бһ¶бһҡбҹ”

### бһ§бһ‘бһ¶бһ бһҳбһҺбҹҚбһ‘бһё бҹҘ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһ’бҹ’бһ„бһ“бҹӢ (бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„)бҹ– бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһҖбҹ’бһ“бһ»бһ„бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһҳбҹүбҹ„бһ„ (Constant Arguing vs Scheduled Weekly Connection Sync)

бһӮбһјбһҹбҹ’бһ“бҹҒбһ бҹҚбһҒбҹ’бһӣбҹҮбһҸбҹӮбһ„бһҸбҹӮбһҮбһҮбҹӮбһҖбһҠбҹҒбһүбһҠбҹ„бһӣбһӮбҹ’бһ“бһ¶бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһғбһҫбһүбһҡбһҝбһ„бһўбһ¶бһҖбҹӢбһўбһ“бҹӢбһ…бһ·бһҸбҹ’бһҸбһҸбһјбһ…бһҸбһ¶бһ…бһ–бҹҒбһүбһҳбһҪбһҷбһҗбҹ’бһ„бҹғ (Chatty network calls) бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһ–бһҪбһҖбһӮбҹҒбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ“бһ¶бһҖбҹӢбһ бһҸбҹӢбһ“бһҝбһҷбһ•бҹ’бһӣбһјбһңбһ…бһ·бһҸбҹ’бһҸ бһ“бһ·бһ„бһӮбҹ’бһҳбһ¶бһ“бһ–бҹҒбһӣбһҹбһ”бҹ’бһ”бһ¶бһҷбһ…бһ·бһҸбҹ’бһҸбҹ” бһҖбҹ’бһҡбҹ„бһҷбһҳбһҖ бһ–бһҪбһҖбһӮбҹҒбһ”бһ¶бһ“бһ–бҹ’бһҡбһҳбһ–бҹ’бһҡбҹҖбһ„бһӮбҹ’бһ“бһ¶бҹ– бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹҶбһ бһ»бһҹбһҶбҹ’бһӮбһ„бһҸбһјбһ…бһҸбһ¶бһ…бһҖбҹ’бһ“бһ»бһ„бһ…бһ·бһҸбҹ’бһҸбһҹбһ·бһ“ (Cache) бһ“бһ·бһ„бһҷбһҖбһҳбһҖбһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӮбҹ’бһ“бһ¶бһҷбҹүбһ¶бһ„бһ‘бһ“бҹӢбһ—бҹ’бһӣбһ“бҹӢбһҸбҹӮбһҳбҹ’бһҸбһ„бһӮбһҸбҹӢбһ“бҹ…бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһӣбҹ’бһ„бһ¶бһ…бһҗбҹ’бһ„бҹғбһўбһ¶бһ‘бһ·бһҸбҹ’бһҷ (Scheduled Aggregated Sync)бҹ”

## бҹӨ. бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бһјбһ‘бҹ…бҹ– бһҖбһ¶бһҡбһҖбһҹбһ¶бһ„ API Gateway Aggregator бһ“бһ·бһ„бһҷбһ“бҹ’бһҸбһҖбһ¶бһҡ Cache-Aside (The General Solution: Designing Parallel Fetching Gateways and Dynamic TTL Caches)

бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ”бҹҶбһҺбһ»бһӣбһ“бҹғбһҖбһ¶бһҡбһҡбһ„бҹӢбһ…бһ¶бҹҶ бһ“бһ·бһ„бһҹбһҳбҹ’бһҡбҹҒбһ…бһ”бһ¶бһ“бһ“бһјбһңбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҖбһ¶бһҡбһ„бһ¶бһҡбһҒбҹ’бһ–бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸ бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ API Aggregation бһ“бһ·бһ„ Cachingбҹ–

бһҮбҹҶбһ бһ¶бһ“бһ“бҹғбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ–
1. **бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ API Gateway Aggregatorбҹ–** бһҮбҹҶбһ“бһҪбһҹбһұбҹ’бһҷбһҖбһ¶бһҡбһұбҹ’бһҷ Client бһ бҹ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҠбһ¶бһ…бҹӢбһҠбҹ„бһҷбһЎбҹӮбһҖбһ–бһёбһӮбҹ’бһ“бһ¶ бһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҖбһҫбһҸ Endpoint бһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢ (бһҠбһјбһ…бһҮбһ¶ GraphQL аё«аёЈаё·аёӯ JSON Aggregator) бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҡбһҪбһ”бһ”бһ„бҹ’бһҡбһҪбһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹ…бһӣбһҫ Cloud бһҳбһ»бһ“бһ•бҹ’бһүбһҫбһҳбһҖ Clientбҹ”
2. **бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ Cache-Aside Patternбҹ–** бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣ Cache бһҮбһ¶бһҳбһ»бһ“бһҹбһ·бһ“бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Cache Hit) бһҸбҹ’бһҡбһјбһңбһ•бҹ’бһүбһҫбһҸбҹ’бһҡбһЎбһ”бҹӢбһ—бҹ’бһӣбһ¶бһҳбҹ—бҹ” бһ”бһҫбһӮбҹ’бһҳбһ¶бһ“ (Cache Miss) бһҹбһ№бһҳбһ‘бһ¶бһүбһҷбһҖбһ–бһё Database бһ“бһ·бһ„бһҹбһҡбһҹбҹҒбһҡбһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„ Cache бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ”
3. **бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһўбһ¶бһҷбһ»бһҖбһ¶бһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (TTL - Time to Live)бҹ–** бһҖбһ»бҹҶбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ Cache бһҡбһ бһјбһҸбһҮбһ¶бһҡбҹҖбһ„бһҡбһ бһјбһҸбҹ” бһҸбҹ’бһҡбһјбһңбһҖбҹҶбһҺбһҸбҹӢбһўбһ¶бһҷбһ»бһҖбһ¶бһӣбһҹбһҳбһҹбҹ’бһҡбһ” (бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бҹҘ бһ“бһ¶бһ‘бһё бһ¬ бҹЈбҹ  бһ“бһ¶бһ‘бһё) бһ’бһ¶бһ“бһ¶бһҗбһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҗбҹ’бһҳбһёбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһү бһ“бһ·бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бһ¶бһҹбҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ„бһҹбһҹбһҳбҹ’бһўбһ¶бһҸбҹ”
4. **бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ’бһ“бһ’бһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһҮбһёбһңбһ·бһҸбҹ–** бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ’бһ“бһ’бһ¶бһ“бһ”бһҺбҹ’бһҸбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“ бһ“бһ·бһ„бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҠбҹӮбһӣбһ“бҹ…бһҹбһӣбҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһҖбһҹбһ¶бһ„ "бһ”бһҺбҹ’бһҸбһ¶бһүбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„ бһ“бһ·бһ„бһҳбһ·бһҸбҹ’бһҸбһ—бһ¶бһ– (Social Capital)" бһҠбҹӮбһӣбһҮбһ¶ Cache бһҠбҹҸбһҡбһ№бһ„бһҳбһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бҹ’бһҡбһ‘бҹ’бһҡбһ„бҹӢбһўбҹ’бһ“бһҖбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҳбһ¶бһ“бһўбһ¶бһҹбһ“бҹ’бһ“бҹ”

## рҹҗҮ бһ’бҹ’бһӣбһ¶бһҖбҹӢбһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһҡбһ“бҹ’бһ’бһ‘бһ“бҹ’бһҹбһ¶бһҷ (Enter the Rabbit Hole)

бһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҖбһ·бһ…бҹ’бһ…бһҹбһ“бҹ’бһҷбһ¶бһҖбһ¶бһҡбһ„бһ¶бһҡ бһҳбһ·бһ“бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҖбһ¶бһҡбһҹбһ“бҹ’бһҷбһ¶ бһ¬бһҖбһ¶бһҡбһ”бҹ’бһҡбһҖбһ¶бһҹбһ–бһёбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһЎбһҫбһҷ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ бһӣбһҖбҹ’бһҒбһҺбҹҲбһҖбһ·бһ…бҹ’бһ…бһҹбһ“бҹ’бһҷбһ¶ (Interfaces) бһҹбһјбһҳбһ”бһ“бҹ’бһҸбһҠбҹҶбһҺбһҫбһҡбһ‘бҹ…бһҖбһ¶бһ“бҹӢбҹ–

* рҹҡҖ **[бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹҶбһҺбһҫбһҡбһҡбһ»бһҖбһҡбһҖ (Start the Journey) вһ” Interface Implementations vs Declarative Signatures (бһҖбһјбһ“бһ”бҹ’бһҡбһ»бһҹбһ–бһёбһҡбһ“бһ¶бһҖбҹӢ)бҹ– бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ„бһ¶бһҡбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„ бһ’бҹҖбһ”бһ“бһ№бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҖбһ¶бһҹбһҖбһ·бһ…бҹ’бһ…бһҹбһ“бҹ’бһҷбһ¶](./197-jesus-and-the-two-sons.md)**

## бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“ (Conclusion)

> **В«бһ—бһ¶бһ–бһ”бҹүбһ·бһ“бһ”бҹ’бһҡбһҹбһ”бҹӢбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ бһӮбһәбһҖбһ¶бһҡбһҷбһҖбһҡбһ”бһҹбҹӢбһ”бһҺбҹ’бһҸбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“бһ‘бҹ…бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бһјбһңбһ‘бҹ’бһҡбһ–бҹ’бһҷбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ· бһ“бһ·бһ„бһ”бһҺбҹ’бһҸбһ¶бһүбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҠбҹӮбһӣбһҳбһ·бһ“бһ…бҹҒбҹҮбһҡбһӣбһ¶бһҷбһҹбһ¶бһ”бһҹбһјбһ“бҹ’бһҷВ»**

бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ API Aggregation бһ“бһ·бһ„ Strategic Caching бһҮбһҪбһҷбһұбҹ’бһҷбһҷбһҫбһ„бһ”бһ„бҹ’бһҖбһҫбһ“бһӣбҹ’бһ”бһҝбһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶ бһ“бһ·бһ„бһ…бҹҒбҹҮбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҮбһёбһңбһ·бһҸбһҡбһҹбҹӢбһ“бҹ…бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҗбҹ’бһ„бҹғбһҠбҹ„бһҷбһҶбҹ’бһӣбһ¶бһҸбһңбҹғ бһҸбҹ’бһҡбҹҖбһҳбһҒбҹ’бһӣбһҪбһ“бһҡбһҪбһ…бһҮбһ¶бһҹбҹ’бһҡбҹҒбһ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһ¶бһӣбҹӢбһңбһ·бһ”бһҸбҹ’бһҸбһ·бһҠбҹӮбһӣбһўбһ¶бһ…бһҖбһҫбһҸбһЎбһҫбһ„бһ“бһ¶бһ–бҹҒбһӣбһўбһ“бһ¶бһӮбһҸбҹ”
