## рҹ“Ң бһҳбһ¶бһҸбһ·бһҖбһ¶ (Table of Contents)
- [бһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢбһ•бҹ’бһӣбһјбһңбһ…бһ·бһҸбҹ’бһҸ (The Trap)](#0)
- [бҹЎ. бһҡбһҝбһ„бһ–бҹ’бһҡбҹҒбһ„бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбҹ– бһӮбҹ’бһҡбһёбһҹбҹ’бһҸбһ¶бһӣбҹӢбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶ бһ“бһ·бһ„бһҖбһ¶бһҡбһ•бҹ’бһҹбһ„бһ–бҹ’бһҡбҹҒбһ„бһ”бҹ’бһҡбһҗбһ»бһҷбһ”бҹ’бһҡбһҗбһ¶бһ“ (The Legend of the Checkpoint Crystal)](#1)
  - [бһ‘бҹҒбһңбһҸбһ¶бһҗбҹӮбһҡбһҖбҹ’бһҹбһ¶ бһ“бһ·бһ„бһҖбһ¶бһҡбһҸбҹ’бһҡбһӣбһ”бҹӢбһҳбһҖбһҹбһ—бһ¶бһ–бһҠбһҫбһҳбһ—бҹ’бһӣбһ¶бһҳбҹ— (The Caretaker and Memento Solution)](#1-1)
- [бҹў. бһ”бһүбҹ’бһ бһ¶бҹ– бһҖбһ¶бһҡбһӣбҹҒбһ…бһ’бҹ’бһӣбһ¶бһҷбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҜбһҖбһҮбһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹҶбһ•бҹ’бһӣбһ¶бһүбһҖбһ¶бһҡбһӣбһ¶бһҖбҹӢбһ”бһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (The Issue: Encapsulation Violation and Internal State Exposure)](#2)
- [бҹЈ. бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҖбҹ’бһ“бһ»бһ„бһ–бһ·бһ—бһ–бһ–бһ·бһҸ (Real World Examples)](#3)
  - [бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹЎ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбҹ’бһҡбһ¶бһӣ (бһӮбҹ’бһҡбһҪбһҹбһ¶бһҡ)бҹ– бһҖбһ¶бһҡбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҖбһҳбҹ’бһ–бһҹбҹӢбһҖбһјбһ“бҹ—бһ“бҹ…бһӣбһҫбһҮбһүбҹ’бһҮбһ¶бҹҶбһ„ (Recording Children's Height Milestones on a Wall)](#3-1)
  - [бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹў вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ’бҹ’бһҷбһҳ (бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ)бҹ– бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ‘бһ¶бһүбһҗбһҷбһҖбҹ’бһҡбҹ„бһҷ Undo/Redo бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбһ¶бһҷбһўбһҸбҹ’бһҗбһ”бһ‘ (Text Editor Undo/Redo Mechanism)](#3-2)
  - [бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹЈ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ’бҹ’бһҷбһҳ (бһ’бһ»бһҡбһҖбһ·бһ…бҹ’бһ…)бҹ– бһҖбһ¶бһҡбһҗбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһҹбҹ’бһҸбһ»бһҖбһ‘бҹҶбһ“бһ·бһүбһҳбһ»бһ“бһ–бҹҒбһӣбһҖбҹӮбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ (Automatic Inventory Snapshot Before Batch Updates)](#3-3)
  - [бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹӨ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ’бҹ’бһҷбһҳ (бһҹбһ„бҹ’бһӮбһҳ/бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„)бҹ– бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһјбһҠбһ”бһҳбҹ’бһҡбһ»бһ„ Git Commit бһҳбһ»бһ“бһ–бҹҒбһӣбһ”бҹ’бһҸбһјбһҡбһҖбһјбһҠбһ–бһ·бһҹбҹ„бһ’бһ“бҹҚ (Creating a Git Commit Before Experimental Branches)](#3-4)
  - [бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹҘ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһ’бҹ’бһ„бһ“бҹӢ (бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„)бҹ– бһҖбһ¶бһҡбһҷбһӣбҹӢбһ–бҹ’бһҡбһҳ "бһ•бҹ’бһўбһ¶бһҖ" бһ“бһ·бһ„бһҖбһ¶бһҡбһҸбҹ’бһҡбһӣбһ”бҹӢбһ‘бҹ…бһҡбһҖбһҹбһ“бҹ’бһҷбһ¶бһҠбһҫбһҳ (Agreeing to a "Reset" or "Time-out" in Relationships)](#3-5)
- [бҹӨ. бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бһјбһ‘бҹ…бҹ– бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ Memento Pattern бһҸбһ¶бһҳбһҡбһҷбҹҲ Opaque Tokens (The General Solution: Memento Pattern with Encapsulated State Tokens)](#4)

## бһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢбһ•бҹ’бһӣбһјбһңбһ…бһ·бһҸбҹ’бһҸ (The Trap)

бһҸбһҫбһўбҹ’бһ“бһҖбһ’бҹ’бһӣбһ¶бһ”бҹӢбһҮбһҪбһ”бһ”бһүбҹ’бһ бһ¶бһҠбҹӮбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҗбһҸбһ…бһҳбҹ’бһӣбһ„ бһ“бһ·бһ„бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“ бһ¬бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– (State) бһҡбһ”бһҹбҹӢ Object бһ’бҹҶбҹ—бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ¶бһүбһҗбһҷбһҖбҹ’бһҡбҹ„бһҷ (Undo) бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҒбҹҶбһ…бһ·бһҸбҹ’бһҸбһ”бһҫбһҖбһ”бһ„бҹ’бһ бһ¶бһүбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸбһҜбһҖбһҮбһ“ (Encapsulated fields) бһҡбһ”бһҹбҹӢбһңбһ¶бһ‘бҹ…бһұбҹ’бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бһҠбһ№бһ„бһҠбҹӮбһҡбһ¬бһ‘бҹҒ?

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбҹ–
* **бһҷбһҫбһ„бһ„бһ¶бһҷбһ“бһ№бһ„бһ’бҹ’бһӣбһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„бһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢ** бһ“бҹғбһҖбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ… (Caretaker) бһ…бһјбһӣбһ‘бҹ…бһўбһ¶бһ“ бһ“бһ·бһ„бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһңбһ¶бһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҡбһ”бһҹбҹӢ Object бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢ бһҠбҹӮбһӣбһ”бҹҶбһ•бҹ’бһӣбһ¶бһүбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһӣбһ¶бһҖбҹӢбһ”бһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Encapsulation) бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҖбһјбһҠбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһ„бһ¶бһҷбһҡбһ„бһӮбҹ’бһҡбҹ„бҹҮ бһ“бһ·бһ„бһҳбһ¶бһ“бһ—бһ¶бһ–бһҮбҹҶбһ–бһ¶бһҖбҹӢбһҮбҹҶбһ–бһ·бһ“бһӮбҹ’бһ“бһ¶бһҒбҹ’бһ–бһҹбҹӢбҹ”
* **бһҷбһҫбһ„бһҳбһҫбһӣбһҡбҹҶбһӣбһ„** бһҷбһ“бҹ’бһҸбһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷ Object бһҒбҹ’бһӣбһҪбһ“бһҜбһ„бһҮбһ¶бһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸ бһ“бһ·бһ„бһҹбҹ’бһҸбһ¶бһҡбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһңбһ¶бһЎбһҫбһ„бһңбһ·бһү бһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһ”бҹ’бһҡбһӮбһӣбҹӢ "бһҹбһүбҹ’бһүбһ¶бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһ”бһ·бһ‘бһҮбһ·бһҸ (Memento Token)" бһ‘бҹ…бһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҖбҹ’бһҡбҹ…бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҹбһ·бһ‘бҹ’бһ’бһ·бһҳбһҫбһӣ бһ¬бһҖбҹӮбһ”бҹ’бһҡбҹӮбһЎбһҫбһҷбҹ”

бһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҜбһҖбһҮбһ“бһҡбһ”бһҹбҹӢ Object бһҳбһҖбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ“бҹ…бһҖбҹ’бһҡбҹ… Class бһ бҹ…бһҗбһ¶ **бһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢбһ”бҹҶбһ•бҹ’бһӣбһ¶бһүбһҖбһ¶бһҡбһӣбһ¶бһҖбҹӢбһ”бһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Encapsulation Violation Trap)**бҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҡбһ”бҹҖбһ”бһҗбһҸбһ…бһҳбҹ’бһӣбһ„ бһ“бһ·бһ„бһҹбҹ’бһҸбһ¶бһҡбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– Object бһ”бҹ’бһҡбһҖбһ”бһҠбҹ„бһҷбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢ бһ“бҹҒбҹҮбһҮбһ¶бһ•бҹӮбһ“бһ‘бһёбһ”бһ„бҹ’бһ бһ¶бһүбһ•бҹ’бһӣбһјбһңбҹ–
1. **бһҡбһҝбһ„бһ–бҹ’бһҡбҹҒбһ„бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ (The Historic Legend)** вҖ” бһҡбһҝбһ„бһҡбҹүбһ¶бһңбһҡбһ”бһҹбҹӢбһңбһёбһҡбһ”бһ»бһҡбһҹбһҠбҹӮбһӣбһ”бҹ’бһҡбһҷбһ»бһ‘бҹ’бһ’бһҮбһ¶бһҳбһҪбһҷбһ”бһ·бһҹбһ¶бһ… бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҸбһ¶бһҳбһҡбһҷбҹҲбһӮбҹ’бһҡбһёбһҹбҹ’бһҸбһ¶бһӣбҹӢбһңбҹҒбһ‘бһҳбһ“бҹ’бһҸбҹ”
2. **бһ”бһүбҹ’бһ бһ¶ (The Issue)** вҖ” бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһ—бһ¶бһ–бһӣбҹҒбһ…бһ’бҹ’бһӣбһ¶бһҷбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҜбһҖбһҮбһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹҶбһ•бҹ’бһӣбһ¶бһү Encapsulation бһҖбҹ’бһ“бһ»бһ„ OOPбҹ”
3. **бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҖбҹ’бһ“бһ»бһ„бһ–бһ·бһ—бһ–бһ–бһ·бһҸ (Real World Examples)** вҖ” бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһ”бһүбҹ’бһ бһ¶бһ“бҹҒбҹҮбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҡбһ·бһҸбһӮбҹ’бһҡбһҪбһҹбһ¶бһҡ бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶ бһ’бһ»бһҡбһҖбһ·бһ…бҹ’бһ… бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бҹ”
4. **бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бһјбһ‘бҹ… (The General Solution)** вҖ” бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ Memento Pattern бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһүбҹ’бһүбһ¶бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһ”бһ·бһ‘бһҮбһ·бһҸбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бҹ”

## бҹЎ. бһҡбһҝбһ„бһ–бҹ’бһҡбҹҒбһ„бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбҹ– бһӮбҹ’бһҡбһёбһҹбҹ’бһҸбһ¶бһӣбҹӢбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶ бһ“бһ·бһ„бһҖбһ¶бһҡбһ•бҹ’бһҹбһ„бһ–бҹ’бһҡбҹҒбһ„бһ”бҹ’бһҡбһҗбһ»бһҷбһ”бҹ’бһҡбһҗбһ¶бһ“ (The Legend of the Checkpoint Crystal)

бһҖбһ¶бһӣбһ–бһёбһ–бҹ’бһҡбҹҒбһ„бһ“бһ¶бһҷ бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ–бһ·бһ—бһ–бһңбҹҒбһ‘бһҳбһ“бҹ’бһҸбһҠбҹҸбһўбһ¶бһҗбҹҢбһҖбҹҶбһ”бһ¶бҹҶбһ„ бһҳбһ¶бһ“бһңбһёбһҡбһ”бһ»бһҡбһҹбһҳбҹ’бһ“бһ¶бһҖбҹӢ (Originator) бһҸбҹ’бһҡбһјбһңбһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһјбһ„бһ—бҹ’бһ“бҹҶбһ”бһ·бһҹбһ¶бһ…бһҠбҹҸбһҒбҹ’бһҳбҹ…бһ„бһ„бһ№бһҸ бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһҳбҹ’бһ…бһ¶бһҸбҹӢбһҹбҹ’бһҸбҹҒбһ…бһ”бһ·бһҹбһ¶бһ…бҹ”

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһӮбһ¶бһҸбҹӢбһҠбһҫбһҡбһ…бһјбһӣбһ‘бҹ…бһ”бҹ’бһҡбһҷбһ»бһ‘бҹ’бһ’бһ—бҹ’бһӣбһ¶бһҳбҹ— бһ бһҫбһҷбһ—бҹ’бһӣбһ¶бһҸбҹӢбһҹбҹ’бһ“бҹҖбһҸбһҸбҹ’бһҡбһјбһңбһ”бһ·бһҹбһ¶бһ…бһңбһ¶бһҷбһҹбһҳбҹ’бһҡбһ»бһҖбһ‘бһ¶бһӣбҹӢбһҸбҹӮбһҹбҹ’бһӣбһ¶бһ”бҹӢ (бһҸбҹҶбһҺбһ¶бһ„бһұбҹ’бһҷбһҖбһ¶бһҡбһӮбһ¶бҹҶбһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһ¬ Error) бһ“бҹ„бҹҮбһҖбһ¶бһҡбһҒбһ·бһҸбһҒбҹҶбһ”бҹ’бһҡбһ№бһ„бһ”бҹ’бһҡбҹӮбһ„ бһ“бһ·бһ„бһҮбһёбһңбһ·бһҸбһ•бҹ’бһҹбһ„бһ–бҹ’бһҡбҹҒбһ„бһҡбһ”бһҹбҹӢбһӮбһ¶бһҸбҹӢбһҖбһ“бҹ’бһӣбһ„бһҳбһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„бҹ” бһӮбһ¶бһҸбҹӢбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһҹбҹ’бһӣбһ¶бһ”бҹӢбһҮбһ¶бһҹбҹ’бһҗбһ¶бһ–бһҡ бһ бһҫбһҷбһӮбҹ’бһҳбһ¶бһ“бһҮбһҳбҹ’бһҡбһҫбһҹ "Undo" бһЎбһҫбһҷбҹ”

бһӮбһ¶бһҸбҹӢбһҳбһ·бһ“бһўбһ¶бһ…бһҠбһҫбһҡбһҡбһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһӮбҹ’бһҡбһ”бҹӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„ бһҲбһ¶бһҳ бһ“бһ·бһ„бһҖбҹ’бһ”бһ¶бһ…бҹӢбһӮбһ»бһ“бһҡбһ”бһҹбҹӢбһӮбһ¶бһҸбҹӢбһ‘бҹ…бһұбҹ’бһҷбһҳбһ“бһ»бһҹбҹ’бһҹбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһ“бһ¶бһҖбҹ’бһ“бһ»бһ„бһ—бһјбһҳбһ·бһҠбһ№бһ„бһ”бһ¶бһ“бһЎбһҫбһҷ бһ–бҹ’бһҡбҹ„бҹҮбһҖбһ¶бһӣбһҺбһ¶бһ…бҹ„бһҡ бһ¬бһҹбһҸбҹ’бһҡбһјбһңбһҠбһ№бһ„бһ–бһёбһ…бҹҶбһҺбһ»бһ…бһҒбҹ’бһҹбҹ„бһҷбһҡбһ”бһҹбҹӢбһӮбһ¶бһҸбҹӢ бһ–бһҪбһҖбһӮбҹҒбһ“бһ№бһ„бһҡбһҖбһңбһ·бһ’бһёбһҖбһҳбҹ’бһ…бһ¶бһҸбҹӢбһӮбһ¶бһҸбҹӢбһҮбһ¶бһҳбһ·бһ“бһҒбһ¶бһ“ (бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ–бһ¶бһҡ Encapsulation бһҡбһ”бһҹбҹӢбһҡбһ¶бһ„бһҖбһ¶бһҷ)бҹ”

### бһ‘бҹҒбһңбһҸбһ¶бһҗбҹӮбһҡбһҖбҹ’бһҹбһ¶ бһ“бһ·бһ„бһҖбһ¶бһҡбһҸбҹ’бһҡбһӣбһ”бҹӢбһҳбһҖбһҹбһ—бһ¶бһ–бһҠбһҫбһҳбһ—бҹ’бһӣбһ¶бһҳбҹ— (The Caretaker and Memento Solution)

бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбһ¶бһҡбһ”бҹ’бһҡбһҲбһҳбһҳбһ»бһҒбһ“бҹҒбҹҮ бһңбһёбһҡбһ”бһ»бһҡбһҹбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ **бһӮбҹ’бһҡбһёбһҹбҹ’бһҸбһ¶бһӣбҹӢбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶ (The Checkpoint Crystal / Memento)** бһҳбһҪбһҷбһӮбҹ’бһҡбһ¶бһ”бҹӢбҹ”

бһҳбһ»бһ“бһ–бҹҒбһӣбһ”бҹ„бҹҮбһҮбҹҶбһ бһ¶бһ“бһ…бһјбһӣбһ‘бҹ…бһҮбһҪбһ”бһ”бһ·бһҹбһ¶бһ… бһңбһёбһҡбһ”бһ»бһҡбһҹбһ”бһ¶бһ“бһҹбһјбһҸбҹ’бһҡбһҳбһ“бҹ’бһҸбһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһ”бҹӢбһҷбһҖ (Capture) бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҡбһ¶бһ„бһҖбһ¶бһҷбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһҡбһ”бһҹбҹӢбһӮбһ¶бһҸбҹӢ (бһҲбһ¶бһҳ бҹЎбҹ бҹ %, бһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„бһҳбҹүбһ¶бһҺбһ¶ бҹҘбҹ , бһўбһ¶бһңбһ»бһ’бһ–бҹҒбһүбһӣбҹҒбһү) бһҡбһҪбһ…бһ…бһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣ бһ“бһ·бһ„бһ”бһ·бһ‘бһҮбһ·бһҸбһҷбҹүбһ¶бһ„бһҺбҹӮбһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһӮбҹ’бһҡбһёбһҹбҹ’бһҸбһ¶бһӣбҹӢбһ“бҹ„бҹҮбҹ”

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһӮбһ¶бһҸбҹӢбһ”бһ¶бһ“бһ”бҹ’бһҡбһӮбһӣбҹӢбһӮбҹ’бһҡбһёбһҹбҹ’бһҸбһ¶бһӣбҹӢбһ”бһ·бһ‘бһҮбһ·бһҸбһ“бҹ„бҹҮбһ‘бҹ…бһұбҹ’бһҷ **бһ‘бҹҒбһңбһҸбһ¶бһҗбҹӮбһҡбһҖбҹ’бһҹбһ¶ (The Caretaker)** бһҳбҹ’бһ“бһ¶бһҖбҹӢбһҠбҹӮбһӣбһ бҹ„бҹҮбһ бһҫбһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһӮбһ¶бһҸбҹӢбһ–бһёбһҒбһ¶бһ„бһӣбһҫбҹ” 
* бһ‘бҹҒбһңбһҸбһ¶бһҗбҹӮбһҡбһҖбҹ’бһҹбһ¶бһӮбҹ’бһҳбһ¶бһ“бһҹбһ·бһ‘бҹ’бһ’бһ·бһҳбһҫбһӣ бһ¬бһҖбҹӮбһ”бҹ’бһҡбҹӮбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһӮбҹ’бһҡбһёбһҹбҹ’бһҸбһ¶бһӣбҹӢбһЎбһҫбһҷ (Opaque Object)бҹ”
* бһ‘бҹҒбһңбһҸбһ¶бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһҡбһҖбҹ’бһҹбһ¶бһңбһ¶бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһ”бһјбһ”бһ‘бһ·бһ–бҹ’бһңбҹ”

бһңбһёбһҡбһ”бһ»бһҡбһҹбһ”бҹ„бҹҮбһҮбҹҶбһ бһ¶бһ“бһ‘бҹ…бһ”бҹ’бһҡбһҷбһ»бһ‘бҹ’бһ’! бһӮбһ¶бһҸбҹӢбһҡбһ„бһҖбһ¶бһҡбһңбһ¶бһҷбһ”бҹ’бһҡбһ бһ¶бһҡбһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҡбһ бһјбһҸбһҠбһӣбҹӢбһҲбһ¶бһҳбһҹбһӣбҹӢбһҸбҹ’бһҡбһ№бһҳ бҹЎбҹ % бһ“бһ·бһ„бһ бҹҖбһ”бһ“бһ№бһ„бһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбһҮбһёбһңбһ·бһҸбҹ” бһ—бҹ’бһӣбһ¶бһҳбҹ—бһ“бҹ„бҹҮ бһӮбһ¶бһҸбҹӢбһ”бһ¶бһ“бһҹбҹ’бһҡбҹӮбһҖбһ бҹ…бһ‘бҹҒбһңбһҸбһ¶бһҗбһ¶бҹ– *"бһҹбһјбһҳбһ бһ»бһ…бһӮбҹ’бһҡбһёбһҹбҹ’бһҸбһ¶бһӣбҹӢбһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶбһҳбһҖбһңбһ·бһү!"*

бһ–бҹҒбһӣбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһӮбҹ’бһҡбһёбһҹбҹ’бһҸбһ¶бһӣбҹӢбһ—бҹ’бһӣбһ¶бһҳ бһңбһёбһҡбһ”бһ»бһҡбһҹбһ”бһ¶бһ“бһ”бһҫбһҖбһңбһ¶бһ…бҹҒбһүбҹ” бһҳбһҪбһҷбһҡбҹҶбһ–бҹҒбһ…бһ“бҹ„бҹҮ бһҡбһ¶бһ„бһҖбһ¶бһҷбһҡбһ”бһҹбҹӢбһӮбһ¶бһҸбҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбҹ’бһҸбһ¶бһҡ (Restore) бһҸбҹ’бһҡбһӣбһ”бҹӢбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҲбһ¶бһҳ бҹЎбҹ бҹ % бһ“бһ·бһ„бһ‘бһёбһҸбһ¶бҹҶбһ„бһҲбһҡбһҳбһ»бһ“бһ–бҹҒбһӣбһ”бҹ’бһҡбһҷбһ»бһ‘бҹ’бһ’бһңбһ·бһүбһ—бҹ’бһӣбһ¶бһҳбҹ—бһҠбҹ„бһҷбһҮбҹ„бһӮбһҮбҹҗбһҷ (State Restoration)бҹ” бһӮбһ¶бһҸбҹӢбһўбһ¶бһ…бһ”бһ“бҹ’бһҸбһ”бҹ’бһҡбһҷбһ»бһ‘бҹ’бһ’ бһ“бһ·бһ„бһҖбһҳбҹ’бһ…бһ¶бһҸбҹӢбһ”бһ·бһҹбһ¶бһ…бһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбҹ”

## бҹў. бһ”бһүбҹ’бһ бһ¶бҹ– бһҖбһ¶бһҡбһӣбҹҒбһ…бһ’бҹ’бһӣбһ¶бһҷбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҜбһҖбһҮбһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹҶбһ•бҹ’бһӣбһ¶бһүбһҖбһ¶бһҡбһӣбһ¶бһҖбҹӢбһ”бһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (The Issue: Encapsulation Violation and Internal State Exposure)

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹ’бһңбһҖбһҳбҹ’бһҳбһҹбһјбһ бҹ’бһңбһңбҹӮбһҡ бһ”бһүбҹ’бһ бһ¶бһ“бҹҒбҹҮбһҖбһҫбһҸбһЎбһҫбһ„бһ“бҹ…бһ–бҹҒбһӣбһҷбһҫбһ„бһҹбһҡбһҹбҹҒбһҡбһҳбһ»бһҒбһ„бһ¶бһҡ Undo/Redo бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбһ¶бһҷбһўбһҸбҹ’бһҗбһ”бһ‘ бһ¬бһ бҹ’бһӮбҹҒбһҳбҹ–

```java
// бһҖбһјбһҠбһҠбҹӮбһӣбһӮбҹ’бһҳбһ¶бһ“ Memento бһӮбһә Client бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бһҸбҹ’бһҡбһјбһңбһҠбһ№бһ„бһ–бһё variables бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ
EditorState state = new EditorState();
state.text = editor.getText();
state.cursorPosition = editor.getCursor();
// бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫ Editor бһҖбҹӮбһ”бҹ’бһҡбҹӮ variable бһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„ бһ“бҹ„бҹҮбһҖбһјбһҠбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ“бһ№бһ„ Error
```

* **бһҖбһ¶бһҡбһӣбҹҒбһ…бһ’бҹ’бһӣбһ¶бһҷбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸ (Loss of Encapsulation)бҹ–** бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бһҸбҹ’бһҡбһјбһңбһҠбһ№бһ„бһ–бһёбһӮбҹ’бһҡбһ”бҹӢ Variable бһҜбһҖбһҮбһ“бһҡбһ”бһҹбҹӢ Object бһҹбҹ’бһ“бһјбһӣ бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ¶бһ…бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ”бһ¶бһ“ бһҠбҹӮбһӣбһ•бҹ’бһ‘бһ»бһҷбһ–бһёбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚ SOLIDбҹ”
* **бһ—бһ¶бһ–бһҮбҹҶбһ–бһ¶бһҖбҹӢбһҮбҹҶбһ–бһ·бһ“бһӮбҹ’бһ“бһ¶бһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„ (Tight Coupling)бҹ–** бһҡбһ¶бһӣбҹӢбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ Variables бһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһҡбһ”бһҹбҹӢ Object бһ’бҹҶ бһ“бһ№бһ„бһҸбһҳбҹ’бһҡбһјбһңбһұбҹ’бһҷбһҖбҹӮбһҖбһјбһҠбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ (Caretaker) бһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ”

**Memento Design Pattern** бһҮбһҪбһҷбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ“бҹҒбҹҮбһҠбҹ„бһҷбһұбҹ’бһҷ Object бһҹбҹ’бһ“бһјбһӣ (Originator) бһҮбһ¶бһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸ бһ“бһ·бһ„бһўбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ Memento бһҡбһ”бһҹбҹӢбһҒбҹ’бһӣбһҪбһ“бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбҹ” Memento Token бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҮбһ¶ Interface бһ”бһ·бһ‘бһҮбһ·бһҸ бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҠбһ‘бҹғбһўбһ¶бһ…бһҖбһ¶бһ“бҹӢбһ”бһ¶бһ“ бһҸбҹӮбһҳбһ·бһ“бһўбһ¶бһ…бһҳбһҫбһӣ бһ¬бһҖбҹӮбһ”бҹ’бһҡбҹӮбһңбһ¶бһӣбһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһ”бһ¶бһ“бһЎбһҫбһҷбҹ”

### бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹЎ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбҹ’бһҡбһ¶бһӣ (бһӮбҹ’бһҡбһҪбһҹбһ¶бһҡ)бҹ– бһҖбһ¶бһҡбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҖбһҳбҹ’бһ–бһҹбҹӢбһҖбһјбһ“бҹ—бһ“бҹ…бһӣбһҫбһҮбһүбҹ’бһҮбһ¶бҹҶбһ„ (Recording Children's Height Milestones on a Wall)

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһӮбҹ’бһҡбһҪбһҹбһ¶бһҡбһҳбһҪбһҷ бһӘбһ–бһ»бһҖбһҳбҹ’бһҸбһ¶бһҷбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҖбһ¶бһҡбһӣбһјбһҸбһӣбһ¶бһҹбҹӢбһҖбһҳбҹ’бһ–бһҹбҹӢбһҡбһ”бһҹбҹӢбһҖбһјбһ“бҹ—бһҠбҹ„бһҷбһӮбһјбһҹбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһ“бҹ…бһӣбһҫбһӮбҹӮбһҳбһҮбһүбҹ’бһҮбһ¶бҹҶбһ„бһ•бҹ’бһ‘бҹҮбһ”бһ¶бһҷбһҡбһ¶бһӣбҹӢбһҶбҹ’бһ“бһ¶бҹҶбҹ” бһҮбҹҶбһ“бһҪбһҹбһұбҹ’бһҷбһҖбһ¶бһҡбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҡбһ¶бһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү (бһҡбһ”бһ”бһўбһ¶бһ бһ¶бһҡбһ”бҹ’бһҡбһ…бһ¶бҹҶбһҗбҹ’бһ„бҹғ бһҳбҹүбҹ„бһ„бһӮбҹҒбһ„ бһ‘бһҳбҹ’бһ„бһ“бҹӢбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„) бһӘбһ–бһ»бһҖбһҳбҹ’бһҸбһ¶бһҷбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҸбҹӮ "бһҹбһүбҹ’бһүбһ¶бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҖбһҳбҹ’бһ–бһҹбҹӢбһҹбһ¶бһҳбһүбҹ’бһү" бһҠбһҫбһҳбҹ’бһ”бһёбһҠбһ№бһ„бһ–бһёбһҖбһ¶бһҡбһӣбһјбһҸбһӣбһ¶бһҹбҹӢ бһ“бһ·бһ„бһҹбҹ’бһҸбһ¶бһҡбһҖбһ¶бһҡбһ…бһ„бһ…бһ¶бҹҶбһЎбһҫбһ„бһңбһ·бһүбһҷбҹүбһ¶бһ„бһҡбһ бҹҗбһҹбҹ”

## бҹӨ. бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бһјбһ‘бҹ…бҹ– бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ Memento Pattern бһҸбһ¶бһҳбһҡбһҷбҹҲ Opaque Tokens (The General Solution: Memento Pattern with Encapsulated State Tokens)

бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷ Object бһ…бһ¶бһ”бҹӢбһҷбһҖ бһ“бһ·бһ„бһҹбҹ’бһҸбһ¶бһҡбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһңбһ¶бһЎбһҫбһ„бһңбһ·бһүбһҠбҹ„бһҷбһҳбһ·бһ“бһ”бҹҶбһ–бһ¶бһ“ Encapsulation бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹҶбһ“бһ¶бҹҶбһҡбһ…бһ“бһ¶ **Memento Pattern**бҹ–

бһҮбҹҶбһ бһ¶бһ“бһ“бҹғбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ–
1. **бһ”бһ„бҹ’бһҖбһҫбһҸ Memento Interface/Tokenбҹ–** бһ”бҹ’бһҡбһҖбһ¶бһҹ Interface бһ”бһ·бһ‘бһҮбһ·бһҸбһҳбһҪбһҷ (Opaque interface) бһҠбҹӮбһӣбһӮбҹ’бһҳбһ¶бһ“ Method бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹӮбһ”бҹ’бһҡбҹӮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһЎбһҫбһҷ бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ»бҹҶбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҠбһ‘бҹғбһӣбһјбһҖбһҠбҹғбһўбһ¶бһ“бһ”бһ¶бһ“бҹ”
2. **бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Originator (Object бһҹбҹ’бһ“бһјбһӣ)бҹ–** бһҹбһҡбһҹбҹҒбһҡ Method `save()` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸ Memento Object бһҗбҹ’бһҳбһёбһҳбһҪбһҷбһҠбҹӮбһӣбһ•бҹ’бһ‘бһ»бһҖбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһңбһ¶ бһ“бһ·бһ„ Method `restore(memento)` бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһё Memento бһ“бҹ„бҹҮ бһҡбһҪбһ…бһҖбҹҶбһҺбһҸбҹӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҒбҹ’бһӣбһҪбһ“бһҜбһ„бһЎбһҫбһ„бһңбһ·бһүбҹ”
3. **бһҡбҹҖбһ”бһ…бҹҶ Caretaker Classбҹ–** бһ”бһ„бҹ’бһҖбһҫбһҸбһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ… (Caretaker) бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ”бһүбҹ’бһҮбһё Mementos (бһҠбһјбһ…бһҮбһ¶ Stack бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Undo) бһҠбҹ„бһҷбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ‘бһ‘бһҪбһӣбһҷбһҖ бһ“бһ·бһ„бһ бһ»бһ… Memento Tokens бһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮ бһҸбҹ’бһҡбһӣбһ”бҹӢбһ‘бҹ…бһұбҹ’бһҷ Originator бһңбһ·бһүбһ“бҹ…бһ–бҹҒбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ Undoбҹ”

## рҹҗҮ бһ’бҹ’бһӣбһ¶бһҖбҹӢбһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһҡбһ“бҹ’бһ’бһ‘бһ“бҹ’бһҹбһ¶бһҷ (Enter the Rabbit Hole)

бһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ…бҹӮбһҖбһ…бһ¶бһҷбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“ бһ”бһ¶бһ“бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһңбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҹбҹҠбһёбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҷбҹүбһ¶бһ„бһ’бҹ’бһ„бһ“бҹӢбһ’бҹ’бһ„бһҡ бһ“бҹ…бһ–бҹҒбһӣбһўбһҸбһ·бһҗбһ·бһҮбһ“бһҡбһ¶бһ”бҹӢбһҳбҹүбһәбһ“бһ“бһ¶бһҖбҹӢбһҸбҹ’бһҡбһјбһңбһўбһ„бҹ’бһӮбһ»бһҷбһҶбҹӮбһҖбһҳбһҫбһӣбһ”бһҳбҹ’бһҡбҹӮбһ”бһҳбҹ’бһҡбһҪбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ¶бһӣбҹӢбһ“бһ¶бһ‘бһё (Polling Bias) бһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮ бһ“бһ·бһ„бһҖбһ¶бһҡбһ•бҹ’бһүбһҫбһҠбҹҶбһҺбһ№бһ„бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ“бҹ…бһ–бҹҒбһӣбһҳбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ (Observer Pattern) бһҹбһјбһҳбһ”бһ“бҹ’бһҸбһҠбҹҶбһҺбһҫбһҡбһ‘бҹ…бһҖбһ¶бһ“бҹӢбҹ–

* рҹҡҖ **[бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹҶбһҺбһҫбһҡбһҡбһ»бһҖбһҡбһҖ (Start the Journey) вһ” Observer Pattern and Event Broadcasting](./92-the-newspaper-subscription.md)**

## бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“ (Conclusion)

> **В«бһҖбһ»бҹҶбһ”бһҺбҹ’бһҸбҹ„бһҷбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҠбһ‘бҹғбһӣбһјбһҖбһҠбҹғбһўбһ¶бһ“бһңбһ¶бһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһҮбһҪбһҷбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһңбһ¶бҹ” бһ…бһјбһҡбһ…бҹҒбҹҮбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҹбһүбҹ’бһүбһ¶бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһ”бһ·бһ‘бһҮбһ·бһҸ бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбҹ’бһҹбһ¶бһ—бһ¶бһ–бһҹбҹ’бһўбһ¶бһҸбһҹбҹ’бһўбҹҶ бһ“бһ·бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ“бҹғбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”В»**

бһ…бһјбһҡбһ’бҹ’бһңбһҫбһҒбҹ’бһӣбһҪбһ“бһҮбһ¶бһңбһ·бһҹбҹ’бһңбһҖбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҹбһ·бһӣбҹ’бһ”бҹҲбһ“бҹғбһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһҹбҹ’бһҗбһ·бһҡбһ—бһ¶бһ–бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҡбһ”бһҹбҹӢ Object (Encapsulated State Management)бҹ” бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ Memento Design Pattern бһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһҮбһҪбһҷбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡ Undo/Redo бһ”бҹ’бһҡбһҖбһ”бһҠбҹ„бһҷбһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһ‘бҹҒ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһңбһ¶бһҖбҹҸбһҮбһҪбһҷбһҖбһ¶бһҡбһ–бһ¶бһҡбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ“бһ·бһ„бһ—бһ¶бһ–бһҹбҹ’бһўбһ¶бһҸбһҹбҹ’бһўбҹҶбһ“бҹғбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһұбҹ’бһҷбһ“бҹ…бһҠбһҠбҹӮбһӣбһ•бһ„бһҠбҹӮбһҡбҹ”
