## рҹ“Ң бһҳбһ¶бһҸбһ·бһҖбһ¶ (Table of Contents)
- [бһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢбһ•бҹ’бһӣбһјбһңбһ…бһ·бһҸбҹ’бһҸ (The Trap)](#0)
- [бҹЎ. бһҡбһҝбһ„бһ–бҹ’бһҡбҹҒбһ„бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбҹ– бһ–бҹ’бһҡбҹғбһҲбһҫбһҡбһ¶бһ”бҹӢбһӣбһ¶бһ“бһҠбһҫбһҳ бһ“бһ·бһ„бһңбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ„бҹ’бһңбҹҮбһҒбһ¶бһҸ Memory (The Legend of the Forest of a Million Trees)](#1)
  - [бһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһңбһ·бһүбҹ’бһүбһ¶бһҺбһҡбһҪбһҳ бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһ“бҹ’бһҹбҹҶбһҹбҹҶбһ…бҹғбһ’бһ“бһ’бһ¶бһ“ (The Shared Model Solution)](#1-1)
- [бҹў. бһ”бһүбҹ’бһ бһ¶бҹ– бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Memory бһ бһҪбһҹбһ”бҹ’бһҡбһҳбһ¶бһҺ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ Object бһҹбҹ’бһ‘бһҪбһ“бҹ— (The Issue: Massive Object Memory Overhead and Duplicate States)](#2)
- [бҹЈ. бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҖбҹ’бһ“бһ»бһ„бһ–бһ·бһ—бһ–бһ–бһ·бһҸ (Real World Examples)](#3)
  - [бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹЎ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбҹ’бһҡбһ¶бһӣ (бһӮбҹ’бһҡбһҪбһҹбһ¶бһҡ)бҹ– бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ’бҹ’бһ„бһ“бҹӢбҹ—бһҡбһҪбһҳбһӮбҹ’бһ“бһ¶бһҖбҹ’бһ“бһ»бһ„бһҹбһ бһӮбһҳбһ“бҹҚ (Sharing Heavy Lawn Mowers in the Neighborhood)](#3-1)
  - [бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹў вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ’бҹ’бһҷбһҳ (бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ)бҹ– бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ—бһ¶бһӮбһӣбҹ’бһўбһ·бһҸ бһ¬бһӮбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бҹҶбһ—бҹ’бһӣбһҫбһ„бһҖбҹ’бһ“бһ»бһ„бһ бҹ’бһӮбҹҒбһҳ 3D (Rendering Millions of Particles in Game Engines)](#3-2)
  - [бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹЈ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ’бҹ’бһҷбһҳ (бһ’бһ»бһҡбһҖбһ·бһ…бҹ’бһ…)бҹ– бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бҹ’бһӣбһ„бҹӢбһӮбҹҶбһҡбһјбһ бһ¶бһ„бһҠбһҠбҹӮбһӣбҹ—бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ бһ¶бһ„бһҳбҹүбһ¶бһҖбһӣбҹ’бһ”бһё (Reusing Store Blueprints for Franchise Outlets)](#3-3)
  - [бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹӨ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ’бҹ’бһҷбһҳ (бһҹбһ„бҹ’бһӮбһҳ/бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„)бҹ– бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбҹҶбһҡбһјбһҖбһ¶бһҡбһ„бһ¶бһҡбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһ“бҹӮбһҖбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһўбһҸбһ·бһҗбһ·бһҮбһ“ (Standardized Task Templates for Customer Support Tickets)](#3-4)
  - [бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹҘ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһ’бҹ’бһ„бһ“бҹӢ (бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„)бҹ– бһҖбһ¶бһҡбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ”бҹ’бһҡбһ–бҹғбһҺбһёбһҹбһ бһӮбһҳбһ“бҹҚбһҡбһҪбһҳбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ—бһ¶бһ–бһҹбҹ’бһ“бһ·бһ‘бҹ’бһ’бһҹбҹ’бһ“бһ¶бһӣ (Relying on Shared Community Traditions to Bond)](#3-5)
- [бҹӨ. бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бһјбһ‘бҹ…бҹ– бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ Flyweight Pattern бһҸбһ¶бһҳбһҡбһҷбҹҲ Intrinsic-Extrinsic Separation (The General Solution: Flyweight Pattern with Shared Intrinsic State Pools)](#4)
- [бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“ (Conclusion)](#5)

## бһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢбһ•бҹ’бһӣбһјбһңбһ…бһ·бһҸбҹ’бһҸ (The Trap)

бһҸбһҫбһўбҹ’бһ“бһҖбһ’бҹ’бһӣбһ¶бһ”бҹӢбһҮбһҪбһ”бһ”бһүбҹ’бһ бһ¶бһҠбҹӮбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһӮбһ¶бҹҶбһ„ бһ¬бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҷбһәбһҸбһҷбҹүбһ¶бһңбһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„ бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһҸбҹ’бһҡбһјбһңбһ•бҹ’бһ‘бһ»бһҖ бһ¬бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһ”бһҹбҹӢбһҠбҹӮбһӣбһҹбҹ’бһҡбһҠбҹҖбһ„бҹ—бһӮбҹ’бһ“бһ¶бһҡбһ¶бһ”бҹӢбһҹбһ·бһ”бһҳбҹүбһәбһ“бһҠбһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Memory бһҠбҹӮбһҡбһ¬бһ‘бҹҒ?

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бҹ–
* **бһҷбһҫбһ„бһ„бһ¶бһҷбһ“бһ№бһ„бһ’бҹ’бһӣбһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„бһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢ** бһ“бҹғбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ Object бһҗбҹ’бһҳбһёбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ (Instantiation from scratch) бһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶ Object бһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮбһҳбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһҪбһҳбһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бҹ©бҹ©% бһҖбҹҸбһҠбҹ„бһҷ бһҠбҹӮбһӣбһ“бһ¶бҹҶбһұбҹ’бһҷбһҒбһ¶бһҸбһ”бһ„бҹӢбһ’бһ“бһ’бһ¶бһ“бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһҹбҹҠбһё RAM бһ бһҪбһҹбһҖбһҳбҹ’бһҡбһ·бһҸбҹ”
* **бһҷбһҫбһ„бһҳбһҫбһӣбһҡбҹҶбһӣбһ„** бһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһ•бҹ’бһ“бҹӮбһҖбһҠбҹӮбһӣбһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһ“бһ·бһ„бһҳбһ·бһ“бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ (Intrinsic State) бһ‘бҹ…бһҠбһ¶бһҖбҹӢбһҡбһҪбһҳбһӮбҹ’бһ“бһ¶бһҖбҹ’бһ“бһ»бһ„ Object бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһҸбҹӮбһҳбһҪбһҷ бһҡбһҪбһ…бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҠбһ¶бһ…бҹӢбһҠбҹ„бһҷбһЎбҹӮбһҖбһҸбҹӮбһ•бҹ’бһ“бҹӮбһҖбһҠбҹӮбһӣбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣ (Extrinsic State) бһҡбһ”бһҹбҹӢ Object бһ“бһёбһҳбһҪбһҷбҹ—бҹ”

бһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһ”бһҹбҹӢбһҡбһ”бһҡбһҡбһ¶бһ”бҹӢбһӣбһ¶бһ“бһҠбҹӮбһӣбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһЎбһҫбһ„бһңбһ·бһүбһҠбһҠбҹӮбһӣбҹ—бһҖбҹ’бһ“бһ»бһ„ Memory бһ бҹ…бһҗбһ¶ **бһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбҹ’бһ‘бһҪбһ“бһ”бҹҶбһ•бҹ’бһӣбһ¶бһүбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ (Redundant Object Memory Exhaustion Trap)**бҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҡбһ”бҹҖбһ”бһҹбһ“бҹ’бһҹбҹҶбһҹбҹҶбһ…бҹғ Memory бһ“бһ·бһ„бһҖбһ¶бһҡбһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹҸбһӣбҹ’бһўбһҘбһҸбһҒбҹ’бһ…бҹ„бҹҮ бһ“бҹҒбҹҮбһҮбһ¶бһ•бҹӮбһ“бһ‘бһёбһ”бһ„бҹ’бһ бһ¶бһүбһ•бҹ’бһӣбһјбһңбҹ–
1. **бһҡбһҝбһ„бһ–бҹ’бһҡбҹҒбһ„бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ (The Historic Legend)** вҖ” бһҡбһҝбһ„бһҡбҹүбһ¶бһңбһҡбһ”бһҹбҹӢбһңбһ·бһҹбҹ’бһңбһҖбһҡбһ бҹ’бһӮбҹҒбһҳбһҠбҹӮбһӣбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһҠбһ¶бҹҶбһҠбһҫбһҳбһҲбһҫбһҡбһ¶бһ”бҹӢбһӣбһ¶бһ“бһҠбһҫбһҳбһҡбһ бһјбһҸбһҠбһӣбҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһӮбһ¶бҹҶбһ„бҹ”
2. **бһ”бһүбҹ’бһ бһ¶ (The Issue)** вҖ” бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһҖбһ¶бһҡбһҹбҹҠбһё Memory бһҒбҹ’бһ–бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„ OOP бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ Object бһҹбҹ’бһ‘бһҪбһ“бһӮбҹ’бһ“бһ¶бҹ”
3. **бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҖбҹ’бһ“бһ»бһ„бһ–бһ·бһ—бһ–бһ–бһ·бһҸ (Real World Examples)** вҖ” бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһ”бһүбҹ’бһ бһ¶бһ“бҹҒбҹҮбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҡбһ·бһҸбһӮбҹ’бһҡбһҪбһҹбһ¶бһҡ бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶ бһ’бһ»бһҡбһҖбһ·бһ…бҹ’бһ… бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бҹ”
4. **бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бһјбһ‘бҹ… (The General Solution)** вҖ” бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ Flyweight Pattern бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҷбһ“бҹ’бһҸбһҖбһ¶бһҡбһҹбһ“бҹ’бһҹбҹҶбһҹбҹҶбһ…бҹғ Memory бһҠбҹҸбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ”

## бҹЎ. бһҡбһҝбһ„бһ–бҹ’бһҡбҹҒбһ„бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбҹ– бһ–бҹ’бһҡбҹғбһҲбһҫбһҡбһ¶бһ”бҹӢбһӣбһ¶бһ“бһҠбһҫбһҳ бһ“бһ·бһ„бһңбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ„бҹ’бһңбҹҮбһҒбһ¶бһҸ Memory (The Legend of the Forest of a Million Trees)

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһ‘бһјбһҢбһёбһҷбҹ„бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһ бҹ’бһӮбҹҒбһҳ 3D бһҠбҹҸбһ’бҹҶбһҳбһҪбһҷ бһҖбҹ’бһҡбһ»бһҳбһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ…бһ„бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһ–бһ·бһ—бһ–бһӣбҹ„бһҖбһ бҹ’бһӮбҹҒбһҳбһҠбҹҸбһўбһҹбҹ’бһ…бһ¶бһҡбҹ’бһҷ бһҠбҹӮбһӣбһҳбһ¶бһ“ **бһ–бҹ’бһҡбҹғбһҲбһҫбһҠбҹҸбһҖбҹ’бһҡбһ¶бһҹбҹӢбһғбҹ’бһҳбһ№бһҖбһҳбһ¶бһ“бһҠбһҫбһҳбһҲбһҫбһ…бҹҶбһ“бһҪбһ“ бҹЎ,бҹ бҹ бҹ ,бҹ бҹ бҹ  бһҠбһҫбһҳ**бҹ”

бһ…бҹҶбһ–бҹ„бҹҮбһҠбһҫбһҳбһҲбһҫбһ“бһёбһҳбһҪбһҷбҹ— бһ–бһҪбһҖбһӮбҹҒбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸ Object бһҗбҹ’бһҳбһёбһҳбһҪбһҷбһ–бһёбһ”бһ¶бһҸбһҠбҹғбһ‘бһ‘бҹҒ (бһ бҹ…бһҗбһ¶ `new Tree()`) бһҠбҹӮбһӣбһҳбһ¶бһ“бһ•бҹ’бһ‘бһ»бһҖбҹ–
* бһҡбһјбһ”бһҡбһ¶бһ„бһӣбһҳбҹ’бһўбһ·бһҸ 3D Mesh (бһ‘бҹҶбһ бҹҶ 10MB)
* бһ–бһҺбҹҢ бһ“бһ·бһ„бһҡбһјбһ”бһ—бһ¶бһ–бһҹбһҳбҹ’бһ”бһҖбһҲбһҫ (Texture бһ‘бҹҶбһ бҹҶ 10MB)
* бһ‘бһёбһҸбһ¶бҹҶбһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ бҹ’бһӮбҹҒбһҳ (Coordinates X, Y coordinates: 1KB)

бһҠбһҫбһҳбһҲбһҫбһҳбһҪбһҷбһҠбһҫбһҳбһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Memory бһ‘бҹҶбһ бҹҶ 20MBбҹ” бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһ бҹ’бһӮбҹҒбһҳбһ”бһ„бҹ’бһҖбһҫбһҸбһҠбһҫбһҳбһҲбһҫбһ…бҹҶбһ“бһҪбһ“ бҹЎ бһӣбһ¶бһ“бһҠбһҫбһҳбһҠбһҫбһҳбҹ’бһ”бһёбһҠбһ¶бҹҶбһҮбһ¶бһ–бҹ’бһҡбҹғбһҲбһҫ бһңбһ¶бһҖбҹҸбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбһ¶бһҡбһ…бһ„бһ…бһ¶бҹҶ RAM бһҡбһ бһјбһҸбһҠбһӣбҹӢбһ‘бҹ… **бҹўбҹ ,бҹ бҹ бҹ  бһҮбһёбһ бҹ’бһӮбһ¶бһ”бҹғ (20 TB)**! бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһўбҹ’бһ“бһҖбһӣбҹҒбһ„бһ”бһҫбһҖбһ бҹ’бһӮбҹҒбһҳбһ—бҹ’бһӣбһ¶бһҳ бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ”бһ¶бһ“бһӮбһ¶бҹҶбһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ“бһ·бһ„бһӣбҹ„бһҸбһ…бҹҒбһүбһ–бһёбһ бҹ’бһӮбҹҒбһҳбһ—бҹ’бһӣбһ¶бһҳбҹ—бһҠбҹ„бһҷбһҹбһ¶бһҡбһҒбҹ’бһңбҹҮбһҒбһ¶бһҸ Memory (Out of Memory)бҹ”

### бһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһңбһ·бһүбҹ’бһүбһ¶бһҺбһҡбһҪбһҳ бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһ“бҹ’бһҹбҹҶбһҹбҹҶбһ…бҹғбһ’бһ“бһ’бһ¶бһ“ (The Shared Model Solution)

бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһ„бҹ’бһӮбҹ’бһҡбҹ„бҹҮбһӮбһҳбҹ’бһҡбҹ„бһ„бһ бҹ’бһӮбҹҒбһҳбһ“бҹҒбҹҮ бһңбһ·бһҹбҹ’бһңбһҖбһҡбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбҹҸбһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҳбҹ’бһ“бһ¶бһҖбҹӢбһ”бһ¶бһ“бһ…бһјбһӣбһҳбһҖбһҖбһҸбҹӢбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһғбһҫбһүбһ–бһёбһ—бһ¶бһ–бһӮбҹ’бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ“бҹҒбҹҮбҹ” бһӮбһ¶бһҸбҹӢбһ”бһ¶бһ“бһңбһ·бһ—бһ¶бһӮбһғбһҫбһүбһҗбһ¶ бһҠбһҫбһҳбһҲбһҫбһ‘бһ¶бҹҶбһ„ бҹЎ бһӣбһ¶бһ“бһҠбһҫбһҳбһ“бҹ„бҹҮбҹ–
1. **бһҡбһјбһ”бһҡбһ¶бһ„ 3D бһ“бһ·бһ„бһ–бһҺбҹҢ (Intrinsic State - бһңбһ·бһүбҹ’бһүбһ¶бһҺбһҡбһҪбһҳ)бҹ–** бһӮбһәбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ”бҹҒбҹҮбһ”бһ·бһ‘бһҡбһңбһ¶бһ„бһҠбһҫбһҳбһҲбһҫбһ“бһёбһҳбһҪбһҷбҹ— бһ“бһ·бһ„бһҳбһ·бһ“бһҠбҹӮбһӣбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһЎбһҫбһҷбҹ”
2. **бһ‘бһёбһҸбһ¶бҹҶбһ„ X, Y (Extrinsic State - бһӣбһҖбҹ’бһҒбһҺбҹҲбһҡбҹҖбһ„бһҒбҹ’бһӣбһҪбһ“)бҹ–** бһӮбһәбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢбһҠбҹӮбһӣбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ–бһёбһҠбһҫбһҳбһҲбһҫбһҳбһҪбһҷбһ‘бҹ…бһҠбһҫбһҳбһҲбһҫбһҳбһҪбһҷбһ‘бҹҖбһҸбҹ”

бһӮбһ¶бһҸбҹӢбһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ **бһӣбҹҶбһ“бһ¶бҹҶбһҡбһ…бһ“бһ¶ Flyweight Pattern** бһҠбҹ„бһҷбһҠбһҖбһҷбһҖбһҡбһјбһ”бһҡбһ¶бһ„ 3D Mesh бһ“бһ·бһ„ Texture бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ бһ…бҹҒбһүбһ–бһёбһҠбһҫбһҳбһҲбһҫбһ“бһёбһҳбһҪбһҷбҹ— бһ бһҫбһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһ¶ Object бһҖбһҺбҹ’бһҸбһ¶бһӣбһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢбһ бҹ…бһҗбһ¶ `TreeModel` (бһ‘бҹҶбһ бҹҶ 20MB)бҹ”

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһҠбһҫбһҳбһҲбһҫбһ‘бһ¶бҹҶбһ„ бҹЎ бһӣбһ¶бһ“бһҠбһҫбһҳбһӣбҹӮбһ„бһ•бҹ’бһ‘бһ»бһҖбһҡбһјбһ”бһ—бһ¶бһ– 3D бһ’бҹ’бһ„бһ“бҹӢбҹ—бһ‘бҹҖбһҸбһ бһҫбһҷбҹ” бһ–бһҪбһҖбһңбһ¶бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһёбһҸбһ¶бҹҶбһ„ (X, Y) бһҡбһ”бһҹбҹӢбһңбһ¶ бһ“бһ·бһ„ **бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҒбҹ’бһҹбҹӮбһҹбҹ’бһ–бһ¶бһ“ Reference** бһ‘бҹ…бһҖбһ¶бһ“бҹӢ `TreeModel` бһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢбһ“бҹ„бҹҮбҹ” 

бһ–бҹҒбһӣбһ“бҹҒбҹҮ бһҠбһҫбһҳбһҲбһҫбһ‘бһ¶бҹҶбһ„ бҹЎ бһӣбһ¶бһ“бһ”бҹ’бһҡбһҫбһўбһҹбҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹӮ 1MB (бһ–бҹ’бһҡбҹ„бҹҮбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҸбҹӮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ X, Y бһҸбһјбһ…) бһ”бһјбһҖбһҡбһҪбһҳбһ“бһ№бһ„бһ‘бҹҶбһ бҹҶ 20MB бһҡбһ”бһҹбҹӢ `TreeModel` бһ“бһ¶бҹҶбһұбҹ’бһҷбһ бҹ’бһӮбҹҒбһҳбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Memory бһҹбһҡбһ»бһ”бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮ **бҹўбҹЎ MB** бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ! бһ–бҹ’бһҡбҹғбһҲбһҫбһҡбһ¶бһ”бҹӢбһӣбһ¶бһ“бһҠбһҫбһҳбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһЎбһҫбһ„бһҷбҹүбһ¶бһ„бһҡбһҹбҹӢбһҡбһңбһҫбһҖ бһ“бһ·бһ„бһҡбһӣбһјбһ“бһ”бҹҶбһ•бһ»бһҸбһӣбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ’бһҳбҹ’бһҳбһҸбһ¶бҹ”

## бҹў. бһ”бһүбҹ’бһ бһ¶бҹ– бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Memory бһ бһҪбһҹбһ”бҹ’бһҡбһҳбһ¶бһҺ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ Object бһҹбҹ’бһ‘бһҪбһ“бҹ— (The Issue: Massive Object Memory Overhead and Duplicate States)

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһ”бһүбҹ’бһ бһ¶бһ“бҹҒбҹҮбһҖбһҫбһҸбһҳбһ¶бһ“бһЎбһҫбһ„бһҮбһ¶бһүбһ№бһҖбһүбһ¶бһ”бҹӢбһ“бҹ…бһ–бҹҒбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҡбһ”бһҹбҹӢбһҡбһ¶бһ”бҹӢбһҳбҹүбһәбһ“бһҠбһ„ бһҠбһјбһ…бһҮбһ¶ бһўбһҖбҹ’бһҹбһҡбһ“бһёбһҳбһҪбһҷбҹ—бһҖбҹ’бһ“бһ»бһ„ Word, бһ¬ Cell бһ“бһёбһҳбһҪбһҷбҹ—бһҖбҹ’бһ“бһ»бһ„ Excelбҹ–

* **бһ—бһ¶бһ–бһӣбҹҶбһ”бһ¶бһҖбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Redundancy)бҹ–** бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ…бһ„бҹӢбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҡбһјбһ”бһ—бһ¶бһ– Texture бһҡбһ”бһҹбҹӢбһҠбһҫбһҳбһҲбһҫ бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһҠбһҫбһҡбһҖбҹӮбһ”бҹ’бһҡбҹӮ Object бһ‘бһ¶бҹҶбһ„ бҹЎ бһӣбһ¶бһ“бһҠбһҫбһҳбһ“бҹ„бҹҮбҹ”

### бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹЎ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбҹ’бһҡбһ¶бһӣ (бһӮбҹ’бһҡбһҪбһҹбһ¶бһҡ)бҹ– бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ’бҹ’бһ„бһ“бҹӢбҹ—бһҡбһҪбһҳбһӮбҹ’бһ“бһ¶бһҖбҹ’бһ“бһ»бһ„бһҹбһ бһӮбһҳбһ“бҹҚ (Sharing Heavy Lawn Mowers in the Neighborhood)

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбһ бһӮбһҳбһ“бҹҚбһ•бҹ’бһ‘бҹҮбһҮбһ·бһҸбһҒбһ¶бһ„бһӮбҹ’бһ“бһ¶ бһҮбҹҶбһ“бһҪбһҹбһұбҹ’бһҷбһҖбһ¶бһҡбһұбҹ’бһҷбһӮбҹ’бһҡбһ”бҹӢбһҖбҹ’бһҡбһ»бһҳбһӮбҹ’бһҡбһҪбһҹбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ‘бҹ…бһ‘бһ·бһүбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҖбһ¶бһҸбҹӢбһҹбҹ’бһҳбҹ…бһҠбҹҸбһ’бҹ’бһ„бһ“бҹӢ бһ“бһ·бһ„бһҗбҹ’бһӣбҹғбһҳбҹ’бһ“бһ¶бһҖбҹӢбһҳбһҪбһҷбһӮбҹ’бһҡбһҝбһ„бһ‘бһ»бһҖбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ‘бҹҮбһҡбҹҖбһ„бҹ—бһҒбҹ’бһӣбһҪбһ“ (бһҠбҹӮбһӣбһ“бһ¶бҹҶбһұбҹ’бһҷбһҒбҹ’бһҮбҹҮбһҒбҹ’бһҮбһ¶бһҷбһҗбһңбһ·бһҖбһ¶бһҡ бһ“бһ·бһ„бһҒбһјбһ…бһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһ“бһҡбһҺбһ¶бһ”бҹ’бһҡбһҫ) бһҹбһ бһӮбһҳбһ“бҹҚбһ”бһ¶бһ“бһ‘бһ·бһүбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҖбһ¶бһҸбҹӢбһҹбҹ’бһҳбҹ…бһӮбҹҶбһҡбһјбһҸбҹӮбһҳбһҪбһҷ (Flyweight) бһ‘бһ»бһҖбһ“бҹ…бһҹбһ¶бһӣбһ¶бһҹбһ бһӮбһҳбһ“бҹҚ бһҡбһҪбһ…бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһҹбһҳбһ¶бһҮбһ·бһҖбһҳбҹ’бһ“бһ¶бһҖбҹӢбҹ—бһҳбһҖбһҒбҹ’бһ…бһёбһҷбһҖбһ‘бҹ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҸбһ¶бһҳбһ•бҹ’бһ‘бҹҮбһҡбҹҖбһ„бҹ—бһҒбҹ’бһӣбһҪбһ“бҹ”

## бҹӨ. бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бһјбһ‘бҹ…бҹ– бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ Flyweight Pattern бһҸбһ¶бһҳбһҡбһҷбҹҲ Intrinsic-Extrinsic Separation (The General Solution: Flyweight Pattern with Shared Intrinsic State Pools)

бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһ“бҹ’бһҹбҹҶбһҹбҹҶбһ…бҹғ Memory бһ“бһ·бһ„бһҮбҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ Object бһҹбҹ’бһ‘бһҪбһ“бҹ— бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹҶбһ“бһ¶бҹҶбһҡбһ…бһ“бһ¶ **Flyweight Pattern**бҹ–

бһҮбҹҶбһ бһ¶бһ“бһ“бҹғбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ–
1. **бһ”бҹҶбһ”бҹӮбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһ–бһёбһҡбһ•бҹ’бһ“бҹӮбһҖбҹ–** бһҖбҹҶбһҺбһҸбҹӢ Intrinsic State (бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһҪбһҳ бһҳбһ·бһ“бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ) бһ“бһ·бһ„ Extrinsic State (бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһҡбҹҖбһ„бһҒбҹ’бһӣбһҪбһ“)бҹ”
2. **бһ”бһ„бҹ’бһҖбһҫбһҸ Flyweight Classбҹ–** бһ”бһ„бҹ’бһҖбһҫбһҸ Class бһҠбҹӮбһӣбһ•бҹ’бһ‘бһ»бһҖбһҸбҹӮ Intrinsic State бһ“бһ·бһ„бһҳбһ¶бһ“ Method бһ‘бһ‘бһҪбһӣбһҷбһҖ Extrinsic State бһҮбһ¶ Parameter бһ–бҹҒбһӣбһ бҹ…бһҖбһ¶бһҡбһ„бһ¶бһҡбҹ”
3. **бһ”бһ„бҹ’бһҖбһҫбһҸ Flyweight Factoryбҹ–** бһ”бһ„бҹ’бһҖбһҫбһҸ Class бһҖбһҺбҹ’бһҸбһ¶бһӣбһҳбһҪбһҷбһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ Pool бһҡбһ”бһҹбҹӢ Flyweights (бһҠбһјбһ…бһҮбһ¶ Map Cache)бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫ Object бһҠбҹӮбһӣбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹ’бһ“бһ»бһ„ Pool бһңбһ¶бһ“бһ№бһ„бһ”бҹ’бһҡбһӮбһӣбҹӢбһҡбһ”бһҹбҹӢбһ“бҹ„бҹҮбһ‘бҹ…бһңбһ·бһү бһҸбҹӮбһ”бһҫбһӮбҹ’бһҳбһ¶бһ“бһ‘бҹҒ бһңбһ¶бһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҗбҹ’бһҳбһё бһ“бһ·бһ„бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„ Pool бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһӣбһҫбһҖбһҖбҹ’бһҡбҹ„бһҷбҹ”

## рҹҗҮ бһ’бҹ’бһӣбһ¶бһҖбҹӢбһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһҡбһ“бҹ’бһ’бһ‘бһ“бҹ’бһҹбһ¶бһҷ (Enter the Rabbit Hole)

бһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ–бҹ’бһҡбҹҮбһҡбһ¶бһҮбһ¶бһҺбһ¶бһ…бһҖбҹ’бһҡбһҠбҹҸбһўбһҹбҹ’бһ…бһ¶бһҡбҹ’бһҷбһҳбһҪбһҷ бһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһ–бҹ’бһҡбҹҶбһҠбҹӮбһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ…бҹ’бһҡбһҖбһ‘бҹ’бһңбһ¶бһҡбһ”бҹ’бһҡбһ¶бһҹбһ¶бһ‘ бһҠбҹ„бһҷбһҠбһ¶бһҖбҹӢ "бһ‘бҹ’бһңбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһҺбҹ’бһҸбһ¶бһӣ" бһҠбһҫбһҳбҹ’бһ”бһёбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ“бһ·бһ„бһҖбһ¶бһҡбһ–бһ¶бһҡбһ•бҹ’бһ‘бҹҮбһҹбҹ’бһҸбҹҒбһ…бһ–бһёбһҮбһ“бһҒбһ·бһӣбһҒбһјбһ… (Proxy Pattern) бһҹбһјбһҳбһ”бһ“бҹ’бһҸбһҠбҹҶбһҺбһҫбһҡбһ‘бҹ…бһҖбһ¶бһ“бҹӢбҹ–

## бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“ (Conclusion)

> **В«бһҖбһ»бҹҶбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһ”бһҹбҹӢбһҡбһ”бһҡбһҡбһ¶бһ”бҹӢбһӣбһ¶бһ“бһЎбһҫбһ„бһңбһ·бһүбһ–бһёбһ…бҹҶбһҺбһ»бһ…бһҹбһјбһ“бҹ’бһҷ бһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһңбһ·бһүбҹ’бһүбһ¶бһҺбһҡбһҪбһҳбһҠбҹҸбһӣбҹ’бһўбһҘбһҸбһҒбҹ’бһ…бҹ„бҹҮбһ”бһ¶бһ“бҹ” бһ…бһјбһҡбһ…бҹҒбҹҮбһҹбһ“бҹ’бһҹбҹҶбһҹбҹҶбһ…бҹғбһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбҹ’бһҹбһ¶бһҹбҹ’бһҗбһ·бһҡбһ—бһ¶бһ–бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бҹ”В»**

бһ…бһјбһҡбһ’бҹ’бһңбһҫбһҒбҹ’бһӣбһҪбһ“бһҮбһ¶бһңбһ·бһҹбҹ’бһңбһҖбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҹбһ·бһӣбҹ’бһ”бҹҲбһ“бҹғбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ’бһ“бһ’бһ¶бһ“бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ (Memory Optimization)бҹ” бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ Flyweight Design Pattern бһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһҮбһҪбһҷбһҹбһ„бҹ’бһӮбҹ’бһҡбҹ„бҹҮбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ–бһёбһңбһ·бһ”бһҸбҹ’бһҸбһ· OOM (Out of Memory) бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһ‘бҹҒ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһңбһ¶бһҖбҹҸбһҮбһҪбһҷбһұбҹ’бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһӣбҹ’бһ”бһҝбһ“бһӣбһҝбһ“ бһ“бһ·бһ„бһўбһ¶бһ…бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҒбҹ’бһ“бһ¶бһҸбһҷбһҖбҹ’бһҹбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбҹ”
