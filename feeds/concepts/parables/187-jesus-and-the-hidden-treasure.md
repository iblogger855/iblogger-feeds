## бһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢбһ•бҹ’бһӣбһјбһңбһ…бһ·бһҸбҹ’бһҸ (The Trap)

бһҸбһҫбһўбҹ’бһ“бһҖбһ’бҹ’бһӣбһ¶бһ”бҹӢбһғбһҫбһүбһҹбҹ’бһҗбһ¶бһ”бҹҗбһ“ бһ¬бһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһҳбһҪбһҷбһҠбҹӮбһӣбһҒбҹҶбһ”бҹ’бһҡбһ№бһ„бһ’бҹ’бһңбһҫбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһҡбһ¶бһ”бҹӢбһҡбһҷбһҷбҹүбһ¶бһ„бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҹбҹӢ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһӣбһ‘бҹ’бһ’бһ•бһӣбһҳбһ·бһ“бһҹбһјбһңбһӣбҹ’бһўбһҹбҹ„бҹҮ бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһ–бһҪбһҖбһӮбҹҒбһҳбһ·бһ“бһ”бһ¶бһ“бһҠбһ№бһ„бһ–бһё "бһ‘бҹҒбһ–бһҖбҹ„бһҹбһӣбҹ’бһҷ бһ¬бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбҹ’бһ“бһјбһӣ (Core Competence)" бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбһҡбһ”бһҹбҹӢбһҒбҹ’бһӣбһҪбһ“бһҠбҹӮбһҡбһ¬бһ‘бҹҒ? бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ бһҳбһ“бһ»бһҹбҹ’бһҹбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһ…бҹӮбһҖбһ…бһ¶бһҷбһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„ бһ“бһ·бһ„бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ‘бҹ…бһӣбһҫбһҡбһҝбһ„бһҡбҹүбһ¶бһңбһҸбһјбһ…бһҸбһ¶бһ…бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҠбҹ„бһҷбһҳбһ·бһ“бһ бҹҠбһ¶бһ“бһ”бҹ„бҹҮбһ”бһ„бҹӢбһҡбһ”бһҹбҹӢбһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮ бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһұбһҖбһ¶бһҹбһҳбһ¶бһҹбһҸбҹӮбһҳбһҪбһҷбҹ”

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ бһ“бһ·бһ„бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бҹ–
* **бһҷбһҫбһ„бһ„бһ¶бһҷбһ“бһ№бһ„бһ’бҹ’бһӣбһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„бһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢ** бһ“бҹғбһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ бһ“бһ·бһ„бһўбһ—бһ·бһңбһҢбҹ’бһҚбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– бһ¬бһ•бһӣбһ·бһҸбһ•бһӣбһ’бһҳбҹ’бһҳбһҸбһ¶бҹ— (Generalist Trap) бһ–бҹ’бһҡбҹ„бҹҮбһҳбһ·бһ“бһ…бһ„бҹӢбһ”бҹ„бҹҮбһ”бһ„бҹӢ бһ¬бһҒбһ¶бһҸбһ”бһ„бҹӢбһўбҹ’бһңбһёбһҳбһҪбһҷ бһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶бһҠбһ№бһ„бһҗбһ¶бһңбһ¶бһҳбһ·бһ“бһўбһ¶бһ…бһ“бһ¶бҹҶбһҳбһҖбһ“бһјбһңбһ§бһҸбҹ’бһҸбһҳбһ—бһ¶бһ–бһ”бҹ’бһҡбһҖбһҪбһҸбһ”бҹ’бһҡбһҮбҹӮбһ„бһҖбҹҸбһҠбҹ„бһҷбҹ”
* **бһҷбһҫбһ„бһҳбһҫбһӣбһҡбҹҶбһӣбһ„** "бһҖбҹҶбһҺбһ”бҹӢбһӣбһ¶бһҖбҹӢбһҖбҹҶбһ”бһ¶бҹҶбһ„" (Dormant Value) бһҠбҹӮбһӣбһҮбһ¶бһ…бҹҶбһҺбһ»бһ…бһҒбҹ’бһӣбһ¶бҹҶбһ„бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ бһҠбҹӮбһӣбһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҷбһҫбһ„бһ”бһҺбҹ’бһҸбһ¶бһҖбҹӢбһ‘бһ»бһ“ бҹЎбҹ бҹ % бһӣбһҫбһңбһ¶ бһңбһ¶бһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҸбһҳбҹ’бһӣбҹғбһӣбһҫбһҹбһӣбһ”бҹӢбһҮбһ¶бһ„бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбһ‘бҹғбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ”бһјбһҖбһ”бһүбҹ’бһ…бһјбһӣбһӮбҹ’бһ“бһ¶бҹ”

бһҖбһ¶бһҡбһҳбһ·бһ“бһ бҹҠбһ¶бһ“бһӣбһ»бһ”бһ…бҹ„бһӣбһҳбһ»бһҒбһ„бһ¶бһҡбһҸбһјбһ…бһҸбһ¶бһ…бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҸбһҳбҹ’бһӣбҹғбһҹбҹ’бһ“бһјбһӣ бһ бҹ…бһҗбһ¶ **бһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢбһ”бҹҶбһ”бҹӮбһҖбһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„ (Diluted Energy Trap)**бҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҡбһ”бҹҖбһ”бһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбҹ’бһ“бһјбһӣ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһ“бһҸбһҳбҹ’бһӣбҹғбһӣбһ¶бһҖбҹӢбһҖбҹҶбһ”бһ¶бҹҶбһ„ бһ“бҹҒбҹҮбһҮбһ¶бһ•бҹӮбһ“бһ‘бһёбһ”бһ„бҹ’бһ бһ¶бһүбһ•бҹ’бһӣбһјбһңбҹ–
1. **бһҡбһҝбһ„бһ–бҹ’бһҡбҹҒбһ„бһ“бһ·бһ‘бһ¶бһ“ (The Legend)** вҖ” бһҡбһҝбһ„бһҡбҹүбһ¶бһңбһҡбһ”бһҹбҹӢбһ”бһ»бһҡбһҹбһҳбҹ’бһ“бһ¶бһҖбҹӢбһҠбҹӮбһӣбһӣбһҖбҹӢбһ‘бҹ’бһҡбһ–бҹ’бһҷбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ·бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ·бһүбһҠбһёбһ…бһҳбҹ’бһҖбһ¶бһҡбһҳбһҪбһҷбһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбҹҶбһҺбһ”бҹӢбһӣбһ¶бһҖбҹӢбһ‘бһ»бһҖбҹ”

```python
        for feature in self.features:
            allocation[feature] = 100.0 / len(self.features)
        return allocation

# Good/Resilient: Going all-in on Core Competence (Unlocking dormant value by deprecating side assets)
class CoreOptimizedApplication:
    def __init__(self):
        # Deprecated and removed low-value side features to focus completely on the hidden treasure
        self.core_competence = "core_analytics_engine"
        
    def allocate_resources(self):
        # 100% focus and optimization on core engine
        return {
            self.core_competence: 100.0
        }
```

* **бһҖбһ¶бһҡбһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбһҖбһ¶бһҡбһ•бҹ’бһҸбҹ„бһҸбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚ (Diluted Development):** бһҖбһ¶бһҡбһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһӮбһјбһ”бҹ’бһҡбһҮбҹӮбһ„бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһӣбҹ’бһўбһҮбһ¶бһ„ (бһҠбһјбһ…бһҮбһ¶ бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ Chat) бһҮбҹҶбһ“бһҪбһҹбһұбҹ’бһҷбһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбҹ’бһ“бһјбһӣбһҡбһ”бһҹбҹӢбһҒбҹ’бһӣбһҪбһ“ (Core Algorithm)бҹ”
* **бһҸбһҳбҹ’бһӣбҹғбһӣбһ¶бһҖбҹӢбһҖбҹҶбһ”бһ¶бҹҶбһ„бһҠбҹӮбһӣбһӣбҹӮбһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ (Dormant Value):** бһ’бһ“бһ’бһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Data assets) бһ¬бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢ бһҸбҹӮбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҷбһҖбһҳбһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһ¶бһ”бҹ’бһҡбһ—бһ–бһ…бҹҶбһҺбһјбһӣбһҗбҹ’бһҳбһёбҹ”

4. **бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһӮбһ¶бҹҶбһ‘бҹ’бһҡ (Align the Organization):** бһҡбҹҖбһ”бһ…бҹҶбһұбҹ’бһҷбһҡбһ¶бһӣбҹӢбһӮбҹ„бһӣбһ“бһҷбҹ„бһ”бһ¶бһҷ бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһңбһ”бҹ’бһ”бһ’бһҳбҹҢбһҖбһ¶бһҡбһ„бһ¶бһҡ бһҠбһҫбһҡбһҸбһҪбһҮбһ¶бһўбҹ’бһ“бһҖбһӮбһ¶бҹҶбһ‘бҹ’бһҡ бһ“бһ·бһ„бһҮбҹҶбһҡбһ»бһүбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбҹ’бһ“бһјбһӣбһ“бҹ„бҹҮбһұбҹ’бһҷбһҠбһӣбҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһҖбҹҶбһ–бһјбһӣбҹ”

## рҹҗҮ бһ’бҹ’бһӣбһ¶бһҖбҹӢбһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһҡбһ“бҹ’бһ’бһ‘бһ“бҹ’бһҹбһ¶бһҷ (Enter the Rabbit Hole)

бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ–бһёбһҗбҹ’бһӣбҹғбһҠбһҫбһҳбһұбһҖбһ¶бһҹ бһ“бһ·бһ„бһҖбһ¶бһҡбһ…бһ¶бһҸбҹӢбһўбһ¶бһ‘бһ·бһ—бһ¶бһ–бһҳбһ»бһҒбһ„бһ¶бһҡбһҖбһ¶бһҡбһ„бһ¶бһҡ (Opportunity Cost & Feature Prioritization) бһҮбһҪбһҷбһұбҹ’бһҷбһҷбһҫбһ„бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҷбһҖбһҸбҹӮбһҡбһ”бһҹбҹӢбһҠбҹӮбһӣбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҒбҹ’бһ–бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸбһҳбһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚ бһҹбһјбһҳбһ”бһ“бҹ’бһҸбһҠбҹҶбһҺбһҫбһҡбһ‘бҹ…бһҖбһ¶бһ“бҹӢбҹ–

* рҹҡҖ **[бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹҶбһҺбһҫбһҡбһҡбһ»бһҖбһҡбһҖ (Start the Journey) вһ” The Parable of the Pearl of Great Price](./188-jesus-and-the-pearl-of-great-price.md)**

<a id="5"></a>
## бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“ (Conclusion)

> **В«бһҖбһ¶бһҡбһӣбҹҮбһ”бһ„бҹӢбһ‘бҹ’бһҡбһ–бҹ’бһҷбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ·бһ’бһҳбҹ’бһҳбһҸбһ¶бҹ—бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҠбһјбһҡбһҷбһҖбһҠбһёбһ…бһҳбҹ’бһҖбһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбҹҶбһҺбһ”бҹӢбһӣбһ¶бһҖбҹӢбһ‘бһ»бһҖ бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһҖбһ¶бһҡбһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбһ“бҹ„бҹҮбһ‘бҹҒ бһҸбҹӮбһҮбһ¶бһҖбһ¶бһҡбһңбһ·бһ“бһ·бһҷбҹ„бһӮбһҠбҹҸбһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһ”бҹҶбһ•бһ»бһҸбһҠбҹӮбһӣбһӮбҹ’бһҳбһ¶бһ“бһ“бһҡбһҺбһ¶бһҳбҹ’бһ“бһ¶бһҖбҹӢбһӮбһҪбһҡбһҡбһ»бһүбһҡбһ¶бһЎбһҫбһҷВ»**

бһҖбһ¶бһҡбһҹбҹ’бһӮбһ¶бһӣбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбҹ’бһ“бһјбһӣ (Core Competence) бһ“бһ·бһ„бһҖбһ¶бһҡбһ бҹҠбһ¶бһ“бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҸбһҳбҹ’бһӣбҹғбһӣбһ¶бһҖбҹӢбһҖбҹҶбһ”бһ¶бҹҶбһ„ бһӮбһәбһҮбһ¶бһӮбһ“бҹ’бһӣбһ№бҹҮбһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһҡбҹҶбһҠбҹ„бҹҮбһҒбҹ’бһӣбһҪбһ“бһ…бҹҒбһүбһ–бһёбһ—бһ¶бһ–бһҳбһ’бҹ’бһҷбһҳ бһ“бһ·бһ„бһҲбһ¶бһ“бһ‘бҹ…бһҹбһҳбҹ’бһҡбҹҒбһ…бһ”бһ¶бһ“бһ“бһјбһңбһ—бһ¶бһ–бһҮбҹ„бһӮбһҮбҹҗбһҷбһҠбҹҸбһўбһҹбҹ’бһ…бһ¶бһҡбҹ’бһҷбһ”бҹҶбһ•бһ»бһҸбҹ”
