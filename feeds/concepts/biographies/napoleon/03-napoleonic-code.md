## ១. តើអ្វីទៅជាក្រមណាប៉ូឡេអុង? (What is the Napoleonic Code?)

**ក្រមណាប៉ូឡេអុង (The Napoleonic Code ឬ French Civil Code of 1804)** គឺជាប្រព័ន្ធច្បាប់រដ្ឋប្បវេណី ដែលត្រូវបានអនុម័តឱ្យប្រើប្រាស់ដោយអធិរាជ ណាប៉ូឡេអុង បូណាប៉ាត (Napoleon Bonaparte)។ 

វាគឺជាក្រមច្បាប់ដ៏សំខាន់បំផុតមួយនៅក្នុងប្រវត្តិសាស្ត្រទំនើប ដែលបានលុបបំបាត់ច្បាប់ចាស់ៗដ៏ស្មុគស្មាញ និងពោរពេញដោយអយុត្តិធម៌ពីសម័យសក្តិភូមិ (Feudalism) ដោយជំនួសមកវិញនូវច្បាប់តែមួយដែលអនុវត្តស្មើៗគ្នាសម្រាប់ប្រជាជនបារាំងទាំងអស់ ទោះជាអ្នកមាន ឬអ្នកក្រក៏ដោយ។

## ២. ហេតុអ្វីបានជាគេបង្កើតវាឡើង? (Why was it created?)

មុនបដិវត្តន៍បារាំង (French Revolution) ប្រទេសបារាំងមិនមានច្បាប់ជាតិរួមទេ។ ភាគខាងត្បូងប្រើច្បាប់រ៉ូម៉ាំង ឯភាគខាងជើងប្រើច្បាប់ទំនៀមទម្លាប់។ មានច្បាប់ខុសៗគ្នារហូតដល់ ៤០០ ប្រភេទនៅតាមតំបន់នីមួយៗ! បន្ថែមពីនេះ វណ្ណៈអភិជន និងសង្ឃ មានសិទ្ធិពិសេស (Privileges) មិនបាច់បង់ពន្ធ និងរួចផុតពីការកាត់ទោស ខណៈដែលប្រជាជនសាមញ្ញរងទុក្ខវេទនា។

ក្រោយពេលណាប៉ូឡេអុងឡើងកាន់អំណាច គាត់ចង់រៀបចំសណ្តាប់ធ្នាប់សង្គមឡើងវិញ។ គាត់ចង់បានប្រទេសមួយដែលមានរដ្ឋបាលរឹងមាំ និងមានច្បាប់ដែលងាយយល់ និងមានយុត្តិធម៌។ គាត់បានបង្កើតគណៈកម្មការច្បាប់មួយ ហើយដោយសារភាពឆ្លាតវៃរបស់គាត់ គាត់តែងតែចូលរួមប្រជុំកែសម្រួលមាត្រាច្បាប់ដោយផ្ទាល់ ដោយផ្តោតលើភាពសមហេតុផល (Logic) និងសិទ្ធិរបស់ពលរដ្ឋ។

<a id="3"></a>
## ៣. តើវាផ្លាស់ប្តូរសង្គមយ៉ាងដូចម្តេច? (How did it change society?)

ក្រមណាប៉ូឡេអុងបាននាំមកនូវគោលការណ៍បដិវត្តន៍សំខាន់ៗ ៣ ដែលបានវាយកម្ទេចរបបសក្តិភូមិទាំងស្រុង៖

1. **សមភាពចំពោះមុខច្បាប់ (Equality Before the Law):** លុបបំបាត់ឋានន្តរស័ក្តិ និងសិទ្ធិពិសេសពីកំណើតរបស់ពួកអភិជន។ មនុស្សគ្រប់រូប (បុរស) គឺស្មើគ្នាចំពោះមុខតុលាការ។ ការតែងតាំងតួនាទីរដ្ឋ គឺផ្អែកលើសមត្ថភាព (Meritocracy) មិនមែនផ្អែកលើខ្សែស្រឡាយសាច់ញាតិឡើយ។
2. **សេរីភាពខាងសាសនា (Freedom of Religion):** ពលរដ្ឋមានសិទ្ធិជឿលើសាសនាណាក៏បាន ហើយសាសនាចក្រ (Church) លែងមានអំណាចក្នុងការកាត់ក្តីលើជីវិតរស់នៅរបស់ប្រជាជនទៀតហើយ។
3. **ការការពារកម្មសិទ្ធិឯកជន (Protection of Property):** សិទ្ធិក្នុងការកាន់កាប់ដីធ្លី និងទ្រព្យសម្បត្តិ ត្រូវបានធានាដោយច្បាប់។ ពួកទាសករដី (Serfs) ត្រូវបានរំដោះ ហើយអាចទិញដីផ្ទាល់ខ្លួនបាន។

*(ចំណាំ៖ ទោះបីជាវាជាច្បាប់ជឿនលឿននៅសម័យនោះ ក៏ក្រមនេះនៅមានចន្លោះប្រហោង គឺវាមិនបានផ្តល់សិទ្ធិស្មើគ្នាដល់ស្ត្រីទេ ដោយចាត់ទុកស្ត្រីថាស្ថិតក្រោមការគ្រប់គ្រងរបស់ឪពុក ឬប្តី។)*

<a id="4"></a>
## ៤. កេរដំណែលដល់ពិភពលោក (Global Legacy)

នៅពេលដែលកងទ័ពណាប៉ូឡេអុងវាយកាន់កាប់ទ្វីបអឺរ៉ុប គាត់បានយកក្រមច្បាប់នេះទៅអនុវត្តនៅតាមប្រទេសនានាដូចជា អ៊ីតាលី អាល្លឺម៉ង់ អេស្ប៉ាញ និងហូឡង់ ដោយបានលុបបំបាត់របបសក្តិភូមិនៅទីនោះចោលអស់។

ណាប៉ូឡេអុងធ្លាប់បានពោលនៅពេលគាត់ត្រូវគេនិរទេសថា៖ 
*"សិរីរុងរឿងពិតប្រាកដរបស់ខ្ញុំ មិនមែនជាការឈ្នះសង្គ្រាម ៤០ ដងនោះទេ។ សមរភូមិ Waterloo នឹងលុបបំបាត់ការចងចាំនៃជ័យជម្នះទាំងនោះ។ ប៉ុន្តែអ្វីដែលនឹងមិនអាចលុបបំបាត់បាន ហើយនឹងរស់រានមានជីវិតជារៀងរហូត គឺក្រមរដ្ឋប្បវេណីរបស់ខ្ញុំ (My Civil Code)។"*

សព្វថ្ងៃនេះ ប្រទេសជាង ៧០ នៅជុំវិញពិភពលោក រួមទាំងប្រទេសកម្ពុជាផងដែរ (ដែលទទួលឥទ្ធិពលតាមរយៈអាណានិគមបារាំង) មានប្រព័ន្ធច្បាប់រដ្ឋប្បវេណី (Civil Law System) ដែលមានឫសគល់ចេញពីក្រមណាប៉ូឡេអុងនេះឯង។
