## бһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢбһ•бҹ’бһӣбһјбһңбһ…бһ·бһҸбҹ’бһҸ (The Trap)

бһҹбҹ’бһҡбһҳбҹғбһҗбһ¶бһўбҹ’бһ“бһҖбһҮбһ¶бһўбҹ’бһ“бһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶бһҳбҹ’бһ“бһ¶бһҖбҹӢ бһҖбҹ’бһ“бһ»бһ„бһҶбҹ’бһ“бһ¶бҹҶ бҹЎбҹ©бҹӨбҹҘ бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбһ„бһ‘бҹҗбһ–бһҹбһ бһҡбһҠбҹ’бһӢбһўбһ¶бһҳбҹҒбһҡбһ·бһҖбһ”бҹ’бһҡбһӮбһӣбҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһұбҹ’бһҷбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһӮбһҺбһ“бһ¶бһӮбһ“бҹ’бһӣбһ„бһӮбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бҹҶбһ—бҹ’бһӣбһҫбһ„бһ’бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“бһӣбһҝбһ“бһ”бҹҶбһ•бһ»бһҸ бһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһҷбһҹбһ„бҹ’бһӮбҹ’бһҡбҹ„бҹҮбһҮбһёбһңбһ·бһҸбһ‘бһ¶бһ бһ¶бһ“бһ“бҹ…бһҹбһҳбһҡбһ—бһјбһҳбһ·бҹ”

бһўбҹ’бһ“бһҖбһҠбһҫбһҡбһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһ“бҹ’бһ‘бһ”бҹӢбһҠбҹҸбһ’бҹҶбһҳбһ бһ·бһҳбһ¶бһҳбһҪбһҷ бһҠбҹӮбһӣбһ–бҹ„бһҡбһ–бҹҒбһүбһ‘бҹ…бһҠбҹ„бһҷ В«бһ”бһ·бһҹбһ¶бһ…бһҳбһ бһ¶бһҷбһҖбҹ’бһҹ ENIACВ» бһ‘бһҳбҹ’бһ„бһ“бҹӢ бҹЈбҹ  бһҸбҹ„бһ“бҹ” бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһҳбһ¶бһ“бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҷбһҪбһҡбһҠбҹғбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһ‘бҹҒ, бһӮбҹ’бһҳбһ¶бһ“ Keyboard бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһ¶бһҷбһўбһҖбҹ’бһҹбһҡ, бһ бһҫбһҷбһҖбҹҸбһӮбҹ’бһҳбһ¶бһ“бһўбҹҒбһҖбҹ’бһҡбһ„бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһӣбһ‘бҹ’бһ’бһ•бһӣбһЎбһҫбһҷбҹ” бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһӮбһә бһҒбҹ’бһҹбҹӮбһҖбһ¶бһ”бһҡбһ¶бһ”бҹӢбһҳбҹүбһәбһ“бһҹбһҡбһҹбҹғ бһҖбһ»бһ„бһҸбһ¶бһҖбҹӢбһҡбһ¶бһ”бҹӢбһ–бһ¶бһ“бҹӢбһӮбҹ’бһҡбһ¶бһ”бҹӢ бһ“бһ·бһ„бһ”бһ“бҹ’бһ‘бҹҮбһҡбһ“бҹ’бһ’бһҠбҹ„бһҸ (Panels) бһҠбҹӮбһӣбһҖбҹ’бһҸбҹ…бһ бһҪбһҸбһ бҹӮбһ„бһҠбһјбһ…бһ…бһ„бҹ’бһҡбҹ’бһҖбһ¶бһ“бһ”бһ¶бһҷбҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһҮбһ¶бһұбҹ’бһҷбһңбһ¶бһ”бһјбһҖбһӣбҹҒбһҒбһҹбһ¶бһҳбһүбҹ’бһүбһ–бһёбһҡ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҠбһҫбһҡбһҠбҹ„бһҸбһҒбҹ’бһҹбҹӮбһ—бҹ’бһӣбһҫбһ„бһҡбһјбһ”бһңбһ“бҹ’бһҸбһҡбһ¶бһ”бҹӢбһҳбҹүбҹ„бһ„ бһҠбһҖбһҠбҹ„бһҸбһҒбҹ’бһҹбҹӮбһҖбһ¶бһ”бһ…бҹҒбһүбһ–бһёбһҡбһ“бҹ’бһ’бһҳбһҪбһҷбһ‘бҹ…бһҡбһ“бҹ’бһ’бһҳбһҪбһҷбһ‘бҹҖбһҸ бһҡбһҪбһ…бһҡбһҸбҹӢбһ‘бҹ…бһ”бһ„бҹ’бһңбһ·бһӣбһҖбһ»бһ„бһҸбһ¶бһҖбҹӢ Dials бһӮбҹ’бһҡбһ”бҹӢбһ‘бһјбҹ” бһ“бҹ…бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһ…бһ»бһ…бһ”бҹҠбһјбһҸбһ»бһ„ Run бһҹбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹӮбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһҳбһ·бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ–бҹ’бһҡбҹ„бҹҮбһҸбҹӮбһҳбһ¶бһ“бһҒбҹ’бһҹбҹӮбһҖбһ¶бһ”бһҳбһҪбһҷбһҹбһҡбһҹбҹғбһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбҹ„бһҳбһҡбһ¶бһ”бҹӢбһҳбҹүбһәбһ“ бһ”бһ¶бһ“бһ’бҹ’бһӣбһ¶бһҖбҹӢбһҡбһ”бһјбһҸ бһ¬бһҡбһӣбһ»бһ„бһ”бһ“бҹ’бһҸбһ·бһ…бҹ” бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣ бҹЈ бһҗбҹ’бһ„бҹғбһ–бҹҒбһү бһҠбһҫбһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҒбҹ’бһҹбҹӮбһ—бҹ’бһӣбһҫбһ„бһҳбҹ’бһҸбһ„бһҳбһҪбһҷбһҮбһҪбһҡбҹ—бһ‘бһ¶бҹҶбһ„бһ”бҹӮбһҖбһүбһҫбһҹ бһҠбҹ„бһҷбһҳбһ·бһ“бһҠбһ№бһ„бһҗбһ¶бһҖбҹҶбһ бһ»бһҹбһ“бҹ…бһҸбҹ’бһҡбһ„бҹӢбһҺбһ¶бһЎбһҫбһҷбҹ”

бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢбһ“бҹғбһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһ…бһ„бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ“бһ№бһ„ **бһӮбҹ’бһҡбһҝбһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбһјбһ”бһңбһ“бҹ’бһҸ (Hardware-Bound Programming)**бҹ”

---

## бҹЎ. бһ”бһүбҹ’бһ бһ¶бҹ– бһҷбһ»бһӮбһҹбһҳбҹҗбһҷбһҠбҹӮбһӣбһҖбһјбһҠ бһ“бһ·бһ„бһӮбҹ’бһҡбһҝбһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ“бҹ…бһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһҠбһ¶бһ…бҹӢбһ…бҹҒбһүбһ–бһёбһӮбҹ’бһ“бһ¶ (The Issue: Hardware-Bound Programming)

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҷбһ»бһӮбһҹбһҳбҹҗбһҷбһҠбҹҶбһ”бһјбһ„бһ”бһ„бҹ’бһўбһҹбҹӢбһ“бҹғбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ бһ–бһ¶бһҖбҹ’бһҷбһҗбһ¶ В«SoftwareВ» бһ“бһ·бһ„ В«HardwareВ» бһӮбһәбһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹҶбһ”бҹӮбһҖбһ…бҹҒбһүбһ–бһёбһӮбҹ’бһ“бһ¶бһЎбһҫбһҷбҹ” Logic бһ“бҹғбһҖбһ¶бһҡбһӮбһҺбһ“бһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҠбҹ„бһҷбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ В«бһҒбҹ’бһҹбҹӮбһҖбһ¶бһ”бһҡбһјбһ”бһңбһ“бҹ’бһҸВ» бһҠбһҫбһҳбҹ’бһ”бһёбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ…бһҡбһ“бҹ’бһҸбһўбһӮбҹ’бһӮбһ·бһҹбһ“бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһ¶бһҹбҹҖбһӮбҹ’бһңбһёбҹ”

бһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бҹӮбһ”бһ“бҹҒбҹҮ бһ”бһ„бҹ’бһҖбһұбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһӣбҹҶбһ”бһ¶бһҖбһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бҹ–
1. **Manual Labor Blockбҹ–** бһҖбһ¶бһҡ Setup бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бһёбһҳбһҪбһҷбҹ—бһҸбҹ’бһҡбһјбһңбһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһҡбһ¶бһ”бҹӢбһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһ“бҹғбһҖбһ¶бһҡбһҠбҹ„бһҸбһҒбҹ’бһҹбҹӮ бһ“бһ·бһ„бһ”бһ„бҹ’бһңбһ·бһӣбһҖбһ»бһ„бһҸбһ¶бһҖбҹӢбһҠбҹ„бһҷбһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„бһ–бһӣбһҖбһҳбҹ’бһҳбҹ” бһ•бһӣбһ·бһҸбһ—бһ¶бһ–бһҖбһ¶бһҡбһ„бһ¶бһҡбһҳбһ¶бһ“бһҖбһҳбҹ’бһҡбһ·бһҸбһ‘бһ¶бһ”бһ”бҹҶбһ•бһ»бһҸбҹ”
2. **Physical Troubleshooting Nightmareбҹ–** бһ–бҹҒбһӣбһҳбһ¶бһ“бһҖбҹҶбһ бһ»бһҹбһҶбҹ’бһӮбһ„ (Bug) бһӮбҹ’бһҳбһ¶бһ“бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ debugger бһҺбһ¶бһҳбһҖбһ”бҹ’бһҡбһ¶бһ”бҹӢбһЎбһҫбһҷбҹ” бһ–бһҪбһҖбһӮбҹҒбһҸбҹ’бһҡбһјбһңбһҠбһҫбһҡбһҶбҹӮбһҖбһҳбһҫбһӣбһҒбҹ’бһҹбҹӮбһ—бҹ’бһӣбһҫбһ„ бһ“бһ·бһ„бһ”бҹҶбһ–бһ„бҹӢбһҖбҹӮбһңбһҹбһ»бһүбҹ’бһүбһҖбһ¶бһҹ (Vacuum Tubes) бһ…бҹҶбһ“бһҪбһ“ бҹЎбҹЁ,бҹ бҹ бҹ  бһ”бҹҶбһ–бһ„бҹӢбһҳбҹ’бһҸбһ„бһҳбһҪбһҷбҹ—бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ—бҹ’бһ“бҹӮбһҖбҹ”
3. **Ego and Knowledge Silosбҹ–** бһҳбһ¶бһ“бһҸбҹӮбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһҡ (Operators) бһ“бһ¶бһҡбһё бҹҰ бһҡбһјбһ” (ENIAC Girls) бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһҠбҹӮбһӣбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһ”бҹ’бһӣбһ„бҹӢбһҹбҹҖбһӮбҹ’бһңбһёбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһ“бҹҒбҹҮбҹ” бһӮбҹ’бһҳбһ¶бһ“бһҜбһҖбһҹбһ¶бһҡ Specs бһҡбһҪбһҳ бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҖбһ¶бһҡбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„бһӣбһҫбһ”бһ»бһӮбҹ’бһӮбһӣбҹ”

---

## бҹў. бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҖбҹ’бһ“бһ»бһ„бһҷбһ»бһӮбһҹбһҳбҹҗбһҷ ENIAC

бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣ **бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһ…бҹҶбһ“бһҪбһ“ бҹҘ** бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһҸбһҹбҹҠбһјбһҡбһ”бһҹбҹӢ Software Engineers бһҮбҹҶбһ“бһ¶бһ“бҹӢбһҠбҹҶбһ”бһјбһ„ бһ“бһ·бһ„бһ”бһҠбһ·бһңбһҸбҹ’бһҸбһ“бҹҚ Stored-Program Conceptбҹ–

### бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹЎ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбҹ’бһҡбһ¶бһӣбҹ– бһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹ„бһҷбһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„бһ–бһӣбһҖбһҳбҹ’бһҳ (Programming Through Physical Patching)

**бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бҹ–** бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ Setup бһӮбһҳбҹ’бһҡбҹ„бһ„бһӮбһҺбһ“бһ¶бһӮбһ“бҹ’бһӣбһ„бһ•бҹ’бһӣбһјбһңбһӮбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бҹҶбһ—бҹ’бһӣбһҫбһ„бһ’бҹҶбһҗбҹ’бһҳбһёбһҳбһҪбһҷбҹ”

* **бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– Low EQ (бһҖбҹҶбһ бһ»бһҹбһҶбҹ’бһӮбһ„)бҹ–** бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹ„бһҷ В«бһҠбһҖбһҠбҹ„бһҸбһҒбҹ’бһҹбҹӮбһҖбһ¶бһ”бһҡбһјбһ”бһңбһ“бҹ’бһҸ (Patch Cables)В» бһҡбһ¶бһ”бҹӢбһ–бһ¶бһ“бҹӢбһҹбһҡбһҹбҹғ бһ“бһ·бһ„бһ”бһ„бҹ’бһңбһ·бһӣбһҖбһ»бһ„бһҸбһ¶бһҖбҹӢбһҖбҹҶбһҺбһҸбҹӢбһўбһҗбҹҒбһҡ (Dials) бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҠбҹғбһӣбһҫ Panelsбҹ” бһҖбһ¶бһҡбһ„бһ¶бһҡбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„бһ”бҹӮбһҖбһүбһҫбһҹбһҡбһ¶бһ”бҹӢбһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһ“бҹҒбҹҮбһ„бһ¶бһҷбһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҖбҹҶбһ бһ»бһҹбһҶбҹ’бһӮбһ„бһҠбҹ„бһҸбһҒбһ»бһҹбһҡбһ“бҹ’бһ’бҹ”
* **бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– High EQ (бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ)бҹ–** бһ”бҹҶбһ”бҹ’бһӣбҹӮбһ„ Instructions бһұбҹ’бһҷбһ‘бҹ…бһҮбһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһҡбһјбһ”бһё (Abstract Data) бһ бһҫбһҷбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Memory бһҡбһ”бһҹбҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ (Stored-Program Concept)бҹ” бһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһңбһ¶бһҷбһҖбһјбһҠбһ”бһүбҹ’бһҮбһ¶бһҸбһ¶бһҳбһҡбһҷбҹҲ Keyboard бһ¬ Punch Cards бһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбҹ”
* **бһӣбһ‘бҹ’бһ’бһ•бһӣбҹ–** бһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹ„бһҷбһҠбҹғбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһҡбһ¶бһ”бҹӢбһҹбһ”бҹ’бһҸбһ¶бһ бҹҚ бһ“бһ·бһ„бһ бһҸбҹӢбһ“бһҝбһҷбһҒбҹ’бһӣбһ¶бҹҶбһ„бҹ” бһҖбһ¶бһҡбһӮбҹ„бһҡбһ–бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚ Stored-Program бһҮбһҪбһҷбһұбҹ’бһҷбһҖбһ¶бһҡ Setup бһҸбҹ’бһҡбһ№бһҳбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһ“бһ¶бһ‘бһё бһ“бһ·бһ„бһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһ”бһ¶бһ“бһӣбһҝбһ“бҹ”

### бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹў вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ’бҹ’бһҷбһҳбҹ– бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҖбҹҶбһ бһ»бһҹбһҠбҹҸбһҹбҹӮбһ“бһ бһҸбҹӢбһ“бһҝбһҷ (The Physical Debugging Nightmare)

**бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бҹ–** бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһӮбһҺбһ“бһ¶ ENIAC бһҹбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹӮбһӮбһ¶бҹҶбһ„бһӣбҹӮбһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ¬бһ•бҹ’бһҸбһӣбҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣбһҒбһ»бһҹбһ–бһ¶бһҖбҹӢбһҖбһҺбҹ’бһҸбһ¶бһӣбһ‘бһёбҹ”

* **бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– Low EQ (бһҖбҹҶбһ бһ»бһҹбһҶбҹ’бһӮбһ„)бҹ–** бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһҡбһӮбҹ’бһҳбһ¶бһ“бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ compiler бһ¬ debugger бһ”бҹ’бһҡбһ¶бһ”бҹӢбһҖбҹҶбһ бһ»бһҹбһЎбһҫбһҷбҹ” бһ–бһҪбһҖбһӮбҹҒбһҸбҹ’бһҡбһјбһңбһҠбһҫбһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһӮбҹҶбһ“бһҡбһҒбҹ’бһҹбҹӮбһ—бҹ’бһӣбһҫбһ„бһҡбһ¶бһ”бҹӢбһҳбҹүбһәбһ“ бһ“бһ·бһ„бһ”бҹҶбһ–бһ„бҹӢ Vacuum бҹЎбҹЁ,бҹ бҹ бҹ  бһӮбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹ’бһҸбһ„бһҳбһҪбһҷбҹ—бһҠбҹ„бһҷбһҠбҹғбһ•бҹ’бһ‘бһ¶бһӣбҹӢ бһ“бһ·бһ„бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ¶бһҹбҹӢбһ…бһҡбһ“бҹ’бһҸбһўбһӮбҹ’бһӮбһ·бһҹбһ“бһё бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҒбһ¶бһҸбһ”бһ„бҹӢбһ–бҹҒбһӣ бҹЈ бһҗбҹ’бһ„бҹғбһ–бҹҒбһүбһҡбһҖбһҳбһ·бһ“бһғбһҫбһүбһҖбҹҶбһ бһ»бһҹбһҒбҹ’бһҹбҹӮбһҡбһӣбһ»бһ„бһҳбһҪбһҷбҹ”
* **бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– High EQ (бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ)бҹ–** бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Software-based Debugging Tools бһ–бҹ’бһҡбһҳбһ‘бһ¶бҹҶбһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ Compiler бһҠбҹӮбһӣбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ scan бһҖбһјбһҠбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҖбҹҶбһ бһ»бһҹ бһ“бһ·бһ„бһ”бһ¶бһүбҹӢ Error Logs бһ”бҹ’бһҡбһ¶бһ”бҹӢбһӣбҹҒбһҒбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһјбһҠбһҒбһ»бһҹбһҶбҹ’бһӮбһ„бһ—бҹ’бһӣбһ¶бһҳбҹ— бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢбһұбҹ’бһҷ Developer бһҠбһ№бһ„бҹ”
* **бһӣбһ‘бҹ’бһ’бһ•бһӣбҹ–** бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҖбҹҶбһ бһ»бһҹбһӣбһҫ Hardware бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҠбҹғбһ”бҹ’бһҡбһҫбһ–бҹҒбһӣбһҡбһ¶бһ”бҹӢбһҗбҹ’бһ„бҹғ бһ“бһ·бһ„бһӮбһ¶бҹҶбһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бҹ” бһҖбһ¶бһҡбһҳбһ¶бһ“ Debugger бһҮбһҪбһҷбһұбҹ’бһҷбһҡбһҖбһғбһҫбһү бһ“бһ·бһ„ Fix Bug бһ”бһ¶бһ“бһӣбһҝбһ“бһҖбҹ’бһ“бһ»бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһңбһ·бһ“бһ¶бһ‘бһёбһҠбҹ„бһҷбһҳбһ·бһ“бһ”бһ¶бһ…бҹӢбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһӮбҹ’бһҡбһҝбһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһЎбһҫбһҷбҹ”

### бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹЈ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһ’бҹ’бһҷбһҳбҹ– бһ—бһ¶бһ–бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһҡбһҒбҹ’бһҹбҹӮбһ—бҹ’бһӣбһҫбһ„ (The Indispensable Wiring Hero Dependency)

**бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бҹ–** бһҳбҹүбһ¶бһҹбҹҠбһёбһ“ ENIAC бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбһ„бһ‘бҹҗбһ–бһўбһ¶бһҳбҹҒбһҡбһ·бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһ”бһ“бҹ’бһ‘бһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбһҳбһёбһҖбһ¶бһҡбһҹбһ„бҹ’бһӮбҹ’бһҡбһ¶бһҳбһҹбҹ’бһ“бһјбһӣбҹ”

* **бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– Low EQ (бһҖбҹҶбһ бһ»бһҹбһҶбҹ’бһӮбһ„)бҹ–** бһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһ…бһ„бһҖбҹ’бһҡбһ„бһҜбһҖбһҹбһ¶бһҡ Specs бһ“бҹғбһӮбҹҶбһ“бһјбһҹбһҹбҹҖбһӮбҹ’бһңбһёбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһЎбһҫбһҷ (Knowledge Silos)бҹ” бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҒбһҪбһҡбһҖбҹ’бһ”бһ¶бһӣбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһҡбһ“бһ¶бһҡбһё бҹҰ бһҡбһјбһ”бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ” бһ“бҹ…бһ–бҹҒбһӣбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһҡбһ…бһҳбҹ’бһ”бһ„бһҳбҹ’бһ“бһ¶бһҖбҹӢбһҳбһ¶бһ“бһҮбҹҶбһ„бһәбһҲбһәбһҹбһҳбҹ’бһҡбһ¶бһҖбһ–бҹ’бһҷбһ¶бһ”бһ¶бһӣ бһҳбҹүбһ¶бһҹбҹҠбһёбһ“ ENIAC бһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһҸбҹ’бһҡбһјбһңбһӮбһ¶бҹҶбһ„бһ…бҹ„бһӣ бһӮбҹ’бһҳбһ¶бһ“бһ“бһҡбһҺбһ¶бһ…бҹҒбҹҮбһҠбҹҶбһЎбһҫбһ„бһҒбҹ’бһҹбҹӮбһ—бҹ’бһӣбһҫбһ„бһҮбҹҶбһ“бһҪбһҹбһЎбһҫбһҷбҹ”
* **бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– High EQ (бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ)бҹ–** бһҹбһҡбһҹбҹҒбһҡ Specification бһ“бҹғбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбһҪбһҳбһӮбҹ’бһ“бһ¶ бһ“бһ·бһ„бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбһјбһҠбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ (Standardized Stored-Program)бҹ” бһҡбһ¶бһӣбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҸбҹ’бһҡбһјбһңбһ…бһ„бһҖбҹ’бһҡбһ„бһҮбһ¶ Text Files бһ“бһ·бһ„бһҳбһ¶бһ“бһҖбһ¶бһҡ Review бһӮбҹ’бһ“бһ¶бһ‘бҹ…бһңбһ·бһүбһ‘бҹ…бһҳбһҖ бһ’бһ¶бһ“бһ¶бһҗбһ¶бһ“бһҡбһҺбһ¶бһҖбҹҸбһўбһ¶бһ…бһўбһ¶бһ“ бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһңбһ¶бһ”бһ¶бһ“бһҠбҹӮбһҡбҹ”
* **бһӣбһ‘бҹ’бһ’бһ•бһӣбҹ–** бһҖбһ¶бһҡбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫ В«бһңбһёбһҡбһ”бһ»бһҡбһҹбһҒбҹ’бһҹбҹӮбһ—бҹ’бһӣбһҫбһ„В» бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһӮбһҳбҹ’бһҡбҹ„бһ„бһҮбһ¶бһҸбһ·бһ„бһ¶бһҷбһ“бһ№бһ„бһҖбһҖбһҹбҹ’бһ‘бҹҮ бһ“бһ·бһ„бһҠбһҪбһӣбһҡбһӣбҹҶбҹ” бһҖбһ¶бһҡбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶ Specs бһ“бһ·бһ„бһ”бҹ’бһҡбһҫ Software бһҮбһҪбһҷбһұбҹ’бһҷбһҖбҹ’бһҡбһ»бһҳбһҖбһ¶бһҡбһ„бһ¶бһҡбһ”бһ“бҹ’бһҸбһңбҹҒбһ“бһӮбҹ’бһ“бһ¶бһ”бһ¶бһ“бһ—бҹ’бһӣбһ¶бһҳбҹ— бһ“бһ·бһ„бһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһЎбһҫбһҷбҹ”

### бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹӨ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһ’бҹ’бһ„бһ“бҹӢбҹ– бһҖбһ¶бһҡбһ”бһ“бҹ’бһҗбҹӮбһҳбһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҖбһ¶бһҡбһ„бһ¶бһҡ бһ“бһ·бһ„бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖ Logic (The Nightmare of Scaling & Logical Modifications)

**бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бҹ–** бһӮбһҳбҹ’бһҡбҹ„бһ„бһӮбһҺбһ“бһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһҸбһ·бһҗбһ·бһҮбһ“бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһ”бһ“бҹ’бһҗбҹӮбһҳбһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһўбһҗбҹҒбһҡбһҗбҹ’бһҳбһёбһҳбһҪбһҷ (e.g. бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„бһҒбҹ’бһҷбһӣбҹӢбһ”бҹ„бһҖбһ”бһҖбҹӢ) бһ‘бҹ…бһӣбһҫбһҹбһҳбһёбһҖбһ¶бһҡбһ…бһ¶бһҹбҹӢбҹ”

* **бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– Low EQ (бһҖбҹҶбһ бһ»бһҹбһҶбҹ’бһӮбһ„)бҹ–** бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮ Logic бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бһ¶бһҖбҹӢбһҹбҹ„бһҡбһӣбһҫбһҒбҹ’бһҹбҹӮбһ—бҹ’бһӣбһҫбһ„ бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҒбҹҶбһ…бһ·бһҸбҹ’бһҸ В«бһҠбһҖбһҒбҹ’бһҹбҹӮбһҖбһ¶бһ”бһҡбһјбһ”бһңбһ“бҹ’бһҸбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢВ» бһ“бҹ…бһӣбһҫбһ‘бһј Panels бһҡбһҪбһ…бһҡбҹҖбһ”бһ…бҹҶбһ•бҹӮбһ“бһҖбһ¶бһҡ бһ“бһ·бһ„бһҠбҹ„бһҸбһҒбҹ’бһҹбҹӮбһ—бҹ’бһӣбһҫбһ„бһҗбҹ’бһҳбһёбһЎбһҫбһ„бһңбһ·бһүбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ–бһёбһҠбҹҶбһ”бһјбһ„ бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҒбһ¶бһҸбһ”бһ„бҹӢбһ–бҹҒбһӣ бҹЎ бһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһ–бҹҒбһү бһ“бһ·бһ„бһ”бҹ’бһҡбһҲбһҳбһ“бһ№бһ„бһҖбҹҶбһ бһ»бһҹбһҹбҹҖбһӮбҹ’бһңбһёбһҗбҹ’бһҳбһёбҹ—бҹ”
* **бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– High EQ (бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ)бҹ–** бһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бҹӮбһ”бһ”бһҸбҹӢбһ”бҹӮбһ“ (Dynamic Programming)бҹ” Developer бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ…бһјбһӣбһ‘бҹ…бһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһўбһҗбҹҒбһҡ (Variable `wind_speed`) бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠ Software бһ“бһ·бһ„бһ”бһүбҹ’бһ…бһјбһӣбһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢ `if/else` бһҗбҹ’бһҳбһё бһҡбһҪбһ…бһ…бһ»бһ… Save бһҖбҹ’бһ“бһ»бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣ бҹЎ бһ“бһ¶бһ‘бһёбҹ”
* **бһӣбһ‘бҹ’бһ’бһ•бһӣбҹ–** бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһҒбҹ’бһҹбҹӮбһ—бҹ’бһӣбһҫбһ„бһЎбһҫбһ„бһңбһ·бһүбһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһҖбҹӮбһ”бҹ’бһҡбҹӮ Logic бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһ•бһӣбһ·бһҸбһ—бһ¶бһ–бһ’бҹ’бһӣбһ¶бһҖбҹӢбһ…бһ»бҹҮ бһ“бһ·бһ„бһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһҷбһјбһҡбҹ” бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Software бһҮбһҪбһҷбһұбҹ’бһҷбһҖбһјбһҠбһ”бһҸбҹӢбһ”бҹӮбһ“бһ”бһ¶бһ“бһҒбҹ’бһ–бһҹбҹӢ бһ“бһ·бһ„бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһӣбһҝбһ“бһ”бҹҶбһ•бһ»бһҸбҹ”

### бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһё бҹҘ вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһ’бҹ’бһ„бһ“бҹӢбҹ– бһҮбһҳбҹ’бһӣбҹ„бҹҮбһ“бҹғбһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһ’бһ“бһ’бһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһҺбһ“бһ¶ (The Resource Conflict of Concurrent Executions)

**бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бҹ–** бһҖбһ„бһ‘бҹҗбһ–бһўбһ¶бһҳбҹҒбһҡбһ·бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҡбһҸбҹӢбһҹбһҳбһёбһҖбһ¶бһҡбһӮбһҺбһ“бһ¶бһ–бһёбһҡбһ•бҹ’бһҹбҹҒбһ„бһӮбҹ’бһ“бһ¶ (бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ„бһ‘бҹҗбһ–бһҮбһҫбһ„бһ‘бһ№бһҖ бһ“бһ·бһ„бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ„бһ‘бҹҗбһ–бһҮбһҫбһ„бһӮбҹ„бһҖ) бһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷбҹ”

* **бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– Low EQ (бһҖбҹҶбһ бһ»бһҹбһҶбҹ’бһӮбһ„)бҹ–** бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮ ENIAC бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбҹӮбһ•бҹ’бһ“бҹӮбһҖ Hardware бһҡбһјбһ”бһңбһ“бҹ’бһҸ бһ–бһҪбһҖбһӮбҹҒбһҳбһ·бһ“бһўбһ¶бһ…бһҡбһҸбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ–бһёбһҡбһ‘бһ“бҹ’бһ‘бһ№бһҳбһӮбҹ’бһ“бһ¶бһ”бһ¶бһ“бһЎбһҫбһҷ бһӣбһ»бҹҮбһҸбҹ’бһҡбһ¶бһҸбҹӮ В«бһҹбһ¶бһ„бһҹбһ„бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“ ENIAC бһҗбҹ’бһҳбһёбһҳбһҪбһҷбһӮбҹ’бһҡбһҝбһ„бһ‘бҹҖбһҸВ» бһҠбҹӮбһӣбһ…бҹҶбһҺбһ¶бһҷбһҗбһңбһ·бһҖбһ¶бһҡбһ¶бһ”бҹӢбһӣбһ¶бһ“бһҠбһ»бһӣбҹ’бһӣбһ¶бһҡ бһ“бһ·бһ„бһҹбҹҠбһёбһ‘бҹҶбһ бҹҶбһ”бһ“бҹ’бһ‘бһ”бҹӢбһ•бҹ’бһ‘бһ»бһҖбһ’бҹҶбһ’бҹҒбһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбҹ”
* **бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– High EQ (бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ)бҹ–** бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһӮбҹҶбһ“бһ·бһҸ **Memory Virtualization & Multi-threading**бҹ” бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ (Operating System) бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹӮбһ„бһ…бҹӮбһҖбһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶ (Virtual Memory) бһ“бһ·бһ„бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢ CPU бһұбҹ’бһҷбһ‘бҹ…бһҮбһ¶ Process бһ…бҹ’бһҡбһҫбһ“ бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӮбһҺбһ“бһ¶бһ”бһ¶бһ“бһҡбһ¶бһ”бҹӢбһҹбһ·бһ”бһҹбһҳбһёбһҖбһ¶бһҡбһ‘бһ“бҹ’бһ‘бһ№бһҳбһӮбҹ’бһ“бһ¶бһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҸбҹӮбһҳбһҪбһҷбҹ”
* **бһӣбһ‘бҹ’бһ’бһ•бһӣбҹ–** бһҖбһ¶бһҡбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ•бҹ’бһ“бҹӮбһҖ Hardware бһҸбҹӮбһҳбҹ’бһҒбһ¶бһ„ бһ“бһ¶бҹҶбһұбҹ’бһҷбһҒбҹ’бһҮбҹҮбһҒбҹ’бһҮбһ¶бһҷбһҗбһңбһ·бһҖбһ¶бһҡбһҠбҹ’бһӢ бһ“бһ·бһ„бһҳбһ·бһ“бһўбһ¶бһ…бһ–бһ„бҹ’бһҡбһёбһҖ (Scale) бһ”бһ¶бһ“бҹ” бһҖбһ¶бһҡбһҳбһ¶бһ“ OS бһ“бһ·бһ„ Software бһҮбһҪбһҷбһ”бҹӮбһ„бһ…бҹӮбһҖбһ’бһ“бһ’бһ¶бһ“бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸ бһҹбһ“бҹ’бһҹбҹҶбһҹбҹҶбһ…бҹғбһ’бһ“бһ’бһ¶бһ“ бһ“бһ·бһ„бһ–бһ„бҹ’бһҡбһёбһҖбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҳбһ·бһ“бһҖбҹҶбһҺбһҸбҹӢбҹ”

## бҹЈ. бһҖбһҸбҹ’бһҸбһ¶бһҮбһҳбҹ’бһҡбһ»бһүбҹ– бһ—бһ¶бһ–бһ‘бһ“бҹӢбһҒбҹ’бһҹбҹ„бһҷбһ“бҹғбһӮбҹ’бһҡбһҝбһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһҳбһҫбһӣбһҹбҹ’бһҡбһ¶бһӣбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһҡбһјбһ”бһё (The Aggravator: Fragile Hardware & Underestimating Abstract Software)

бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶бһҷбһ»бһӮбһҹбһҳбҹҗбһҷ ENIAC бһҸбҹ’бһҡбһјбһңбһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹ„бһҷбһҖбһ¶бһҡбһҠбҹ„бһҸбһҒбҹ’бһҹбҹӮбһҖбһ¶бһ”бһҡбһјбһ”бһңбһ“бҹ’бһҸбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҶбҹ’бһ“бһ¶бҹҶ?

1. **бһ—бһ¶бһ–бһ‘бһ“бҹӢбһҒбҹ’бһҹбҹ„бһҷбһ“бҹғбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһӮбҹ’бһҡбһҝбһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“ (Fragile Hardware Limitations)бҹ–** бһ“бһ¶бһҷбһ»бһӮбһҹбһҳбҹҗбһҷбһ“бҹ„бҹҮ бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶ (RAM) бһҠбҹӮбһӣбһҳбһ¶бһ“бһӣбҹ’бһ”бһҝбһ“бһӣбһҝбһ“ бһ“бһ·бһ„бһ‘бҹҶбһ бҹҶбһ’бҹҶ бһӮбһәбһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһ“бҹ…бһЎбһҫбһҷбҹ” бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бһ„бһҹбҹ„бһҡбһҠбҹ„бһҷбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ”бҹҶбһ–бһ„бҹӢ Vacuum Tubes бһҠбҹӮбһӣбһ„бһ¶бһҷбһ“бһ№бһ„бһҶбҹҒбҹҮ бһ“бһ·бһ„бһҒбһјбһ…бһҒбһ¶бһҸбҹ”
2. **бһҖбһ¶бһҡбһҳбһҫбһӣбһҹбҹ’бһҡбһ¶бһӣбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһҡбһјбһ”бһё (Underestimating Software)бҹ–** бһңбһ·бһҹбҹ’бһңбһҖбһҡбһ”бһ»бһҡбһҹбҹ—бһҮбҹҶбһ“бһ¶бһ“бҹӢбһ“бҹ„бҹҮ бһ•бҹ’бһҸбҹ„бһҸбһҸбҹӮбһӣбһҫбһҖбһ¶бһҡбһҹбһ¶бһ„бһҹбһ„бҹӢбһӮбҹ’бһҡбһҝбһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ’бҹҶбҹ— бһ“бһ·бһ„бһҠбҹӮбһҖбһҗбҹӮбһ” (Hardware) бһҠбҹ„бһҷбһ…бһ¶бһҸбҹӢбһ‘бһ»бһҖбһңбһ¶бһҮбһ¶ В«бһҹбҹ’бһ“бһјбһӣбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠВ»бҹ” бһ–бһҪбһҖбһӮбҹҒбһҷбһӣбҹӢбһғбһҫбһүбһҗбһ¶ бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶ бһ¬бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶ Logic (Software) бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһ„бһ¶бһҡбһҡбһҠбҹ’бһӢбһ”бһ¶бһӣбһҖбҹҶбһ”бҹүбһ·бһҖбһҖбҹҶбһ”бҹүбһ»бһҖбһҠбҹӮбһӣбһӮбҹ’бһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбҹ” бһ“бҹҒбҹҮбһҮбһ¶бһҖбһ¶бһҡбһҷбһӣбҹӢбһ…бҹ’бһҡбһЎбҹҶбһҠбҹҸбһ’бҹҶбһ’бҹҒбһ„бһҠбҹӮбһӣбһ–бһ“бҹ’бһҷбһ¶бһҡбһ–бҹҒбһӣбһҖбҹҶбһҺбһҫбһҸбһҡбһ”бһҹбҹӢ Software Engineering бһўбһҹбҹӢбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҶбҹ’бһ“бһ¶бҹҶбҹ”

---

## бҹӨ. бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бһјбһ‘бҹ… (The General Solution)

бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһӣбҹӮбһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– бһ“бһ·бһ„бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ–бһ·бһ—бһ–бһӣбҹ„бһҖбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶ бһўбҹ’бһ“бһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбҹ’бһ“бһјбһӣбҹ–

### бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ бһңбҹүбһ»бһ“ бһ“бһјбһҳбһ¶бһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбҹ’бһ“бһ»бһ„ Memory (The Von Neumann Architecture & Stored-Program Concept)
бһӮбҹҶбһ“бһ·бһҸбһ”бһҠбһ·бһңбһҸбҹ’бһҸбһ“бҹҚбһҡбһ”бһҹбҹӢбһӣбҹ„бһҖ **John von Neumann** бһӮбһәбһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһ¶бҹҶбһ„ Instructions (бһҖбһјбһҠбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё) бһ“бһ·бһ„ Data (бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ) бһұбҹ’бһҷбһ‘бҹ…бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһўбһҡбһјбһ”бһё (Binary Numbers) бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶ (Memory / RAM) бһҸбҹӮбһҳбһҪбһҷбҹ” бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҳбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҒбҹ’бһҹбҹӮбһ—бҹ’бһӣбһҫбһ„бһҡбһјбһ”бһңбһ“бҹ’бһҸбһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһЎбһҫбһҷбҹ”

### бһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖбһҖбһјбһҠбһ…бҹҒбһүбһ–бһёбһӮбҹ’бһҡбһҝбһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“ (Decoupling Software from Hardware)
бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһ“бһјбһңбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ (OS) бһ“бһ·бһ„бһ—бһ¶бһҹбһ¶бһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ (High-level Programming Languages бһҠбһјбһ…бһҮбһ¶ Fortran, C, Python)бҹ” Developer бһҹбһҡбһҹбҹҒбһҡбһҸбҹӮбһҖбһјбһҠбһўбһҖбҹ’бһҹбһҡбһўбһҡбһјбһ”бһё бһҡбһҪбһ…бһұбҹ’бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ Compiler бһ¬ Interpreter бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮбһ‘бҹ…бһҮбһ¶бһ…бһҡбһ“бҹ’бһҸбһўбһӮбҹ’бһӮбһ·бһҹбһ“бһёбһҮбҹҶбһ“бһҪбһҹбһңбһ·бһүбҹ”

### бһ–бһёбһҖбһ¶бһҡбһ…бһ»бһ… Config бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҠбҹғ бһҳбһҖбһҖбһ¶бһ“бҹӢбһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһ”бһүбҹ’бһҮбһ¶ (From ClickOps to Infrastructure as Code)
бһҳбҹҒбһҡбҹҖбһ“бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҡбһ”бһҹбҹӢ ENIAC бһ”бһ„бҹ’бһҡбҹҖбһ“бһҷбһҫбһ„бһҗбһ¶бһҡбһ¶бһӣбҹӢбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ„бһ¶бһҡбһҠбҹ„бһҷбһҠбҹғ (Manual configuration / ClickOps) бһ„бһ¶бһҷбһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҖбҹҶбһ бһ»бһҹ бһ“бһ·бһ„бһҳбһ·бһ“бһўбһ¶бһ…бһ–бһ„бҹ’бһҡбһёбһҖбһ”бһ¶бһ“бһЎбһҫбһҷбҹ” бһ“бһ¶бһҹбһҳбҹҗбһҷбһ‘бҹҶбһ“бһҫбһ” бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ **Infrastructure as Code (IaC / Terraform)** бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһҠбһҫбһҡбһҸбһ¶бһҳбһӮбһ“бҹ’бһӣбһ„бһҡбһ”бһҹбҹӢ Stored-Program Concept бһҠбҹ„бһҷбһ”бҹ’бһҡбҹӮбһҖбҹ’бһӣбһ¶бһҷбһҡбһ¶бһӣбҹӢбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҖбҹҶбһҺбһҸбҹӢ Cloud Configurations бһұбҹ’бһҷбһ‘бҹ…бһҮбһ¶бһҜбһҖбһҹбһ¶бһҡбһҖбһјбһҠбһўбһҖбҹ’бһҹбһҡбһҠбҹӮбһӣбһўбһ¶бһ…бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹ’бһ“бһ»бһ„ Git бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бҹ”

---

## бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“ (Conclusion)

бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹҸбһўбһҹбҹ’бһ…бһ¶бһҡбҹ’бһҷбһҡбһ”бһҹбҹӢ **ENIAC Girls** бһ“бһ·бһ„бһҳбһ бһ¶бһҷбһҖбҹ’бһҹ **ENIAC** бһ”бһ„бҹ’бһҡбҹҖбһ“бһҷбһҫбһ„бһҗбһ¶ Software бһҳбһ·бһ“бһҳбҹӮбһ“бһҖбһҫбһҸбһ…бҹҒбһүбһ–бһёбһӮбҹ’бһҡбһҝбһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ“бҹ„бҹҮбһЎбһҫбһҷ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһңбһ¶бһӮбһәбһҮбһ¶ **бһӮбҹҶбһ“бһ·бһҸбһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ Logic бһҠбҹҸбһҹбҹҠбһёбһҮбһҳбҹ’бһҡбҹ…бһҡбһ”бһҹбҹӢбһҳбһ“бһ»бһҹбҹ’бһҹбһҮбһ¶бһҸбһ·**бҹ” бһҖбһ¶бһҡбһҠбҹ„бҹҮбһӣбҹӮбһ„бһҖбһјбһҠбһ…бҹҒбһүбһ–бһёбһҒбҹ’бһҹбҹӮбһҖбһ¶бһ”бһҡбһјбһ”бһңбһ“бҹ’бһҸ бһӮбһәбһҮбһ¶бһҮбҹҶбһ бһ¶бһ“бһҠбҹҸбһўбһҹбҹ’бһ…бһ¶бһҡбҹ’бһҷбһ”бҹҶбһ•бһ»бһҸбһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһҖбҹҶбһҺбһҫбһҸбһҠбһӣбҹӢбһҷбһ»бһӮбһҹбһҳбҹҗбһҷбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһңбһ·бһ‘бҹ’бһҷбһ¶ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ–бһ·бһ—бһ–бһӣбҹ„бһҖбһҢбһёбһҮбһёбһҗбһӣбһҠбҹҸбһўбһҹбҹ’бһ…бһ¶бһҡбҹ’бһҷбһҮбһ¶бһҡбҹҖбһ„бһҡбһ бһјбһҸбҹ”
