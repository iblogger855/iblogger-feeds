## бҹЎ. бһҖбһ·бһ…бҹ’бһ…бһҹбһ“бҹ’бһ‘бһ“бһ¶бһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚ (Interview Transcript)

**бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚ (бһӣбҹ„бһҖбһ”бҹ’бһҡбһ’бһ¶бһ“):** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһҷбһҫбһ„бһҳбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ•бҹ’бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҠбһјбһ…бһҮбһ¶ Tree бһ“бһ·бһ„ Graphбҹ” бһҖбһјбһ“бһ…бҹ…бһҒбҹ’бһүбһ»бҹҶбһҳбҹ’бһ“бһ¶бһҖбҹӢбҹ—бһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠ Loop (Traversal) бһҒбһ»бһҹбҹ—бһӮбҹ’бһ“бһ¶бҹ” бһўбҹ’бһ“бһҖбһҒбҹ’бһӣбҹҮбһҹбһҡбһҹбҹҒбһҡ DFS бһҖбҹ’бһ“бһ»бһ„ Business Logic, бһўбҹ’бһ“бһҖбһҒбҹ’бһӣбҹҮбһҹбһҡбһҹбҹҒбһҡ BFS... бһ’бҹ’бһңбһҫбһўбҹ„бһҷбһҖбһјбһҠбһҡбһүбҹүбҹҒбһҡбһүбҹүбҹғ! бһҸбһҫбһ”бҹ’бһўбһјбһ“бһҳбһ¶бһ“бһҖбҹ’бһ”бһҪбһ“бһўбҹ’бһңбһёбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҡбһҪбһ”бһ”бһ„бҹ’бһҡбһҪбһҳбһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹҒбҹҮ бһҠбҹ„бһҷбһҳбһ·бһ“бһ”бһ¶бһ…бҹӢбһўбҹ„бһҷ Client бһҠбһ№бһ„бһ–бһёбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһ‘бҹҒ?

**бһ”бҹҒбһҖбҹ’бһҒбһҮбһ“ (бһҹбһ»бһҒбһ¶):** бһ”бһ¶бһ‘! бһ”бһүбҹ’бһ бһ¶бһ“бҹҒбҹҮбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҠбҹ„бһҷ **Iterator Pattern**бҹ” бһҒбҹ’бһүбһ»бҹҶбһ“бһ№бһ„бһҹбҹ’бһҡбһ„бҹӢбһҷбһ“бҹ’бһҸбһҖбһ¶бһҡбһ“бҹғбһҖбһ¶бһҡбһҡбһ»бһҖбһҡбһҖ (Traversal algorithms бһҠбһјбһ…бһҮбһ¶ DFS, BFS) бһ…бҹҒбһүбһ–бһё Business Logic бһ бһҫбһҷбһҠбһ¶бһҖбҹӢбһңбһ¶бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ Class бһҗбҹ’бһҳбһёбһҳбһҪбһҷбһҠбҹӮбһӣбһ бҹ…бһҗбһ¶ Iteratorбҹ” 
Client бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһҠбһ№бһ„бһ–бһё Interface `Iterator` бһҠбҹӮбһӣбһҳбһ¶бһ“ Method бһҹбһ¶бһҳбһүбҹ’бһүбһ–бһёбһҡбһӮбһә `hasNext()` бһ“бһ·бһ„ `next()`бҹ” Client бһўбһ¶бһ…бһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһ¶бһ“бһҠбҹ„бһҷбһҡбһӣбһјбһ“ бһҳбһ·бһ“бһ”бһ¶бһ…бҹӢбһҒбҹ’бһңбһӣбҹӢбһҗбһ¶бһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһҮбһ¶ Tree бһ¬ Graph бһЎбһҫбһҷ!

**бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚ (бһҹбһҪбһҡбһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢ):** бһӣбҹ’бһў! бһ…бһ»бҹҮбһ”бһҫ Iterator бһ“бҹ„бҹҮбһҠбһҫбһҡбһҠбһӣбҹӢбһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһ“бҹғ Collection бһ бһҫбһҷ... бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮ Client бһ“бҹ…бһҸбҹӮбһҡбһ№бһ„бһҖбҹ’бһ”бһ¶бһӣбһ бҹ… Method `next()` бһ”бһ“бҹ’бһҸбһ‘бҹҖбһҸ бһҸбһҫбһҳбһ¶бһ“бһўбҹ’бһңбһёбһҖбһҫбһҸбһЎбһҫбһ„?

**бһ”бҹҒбһҖбҹ’бһҒбһҮбһ“ (бһҹбһ»бһҒбһ¶):** бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫ Client бһҳбһ·бһ“бһ–бҹ’бһҡбһҳбһҶбҹӮбһҖ `hasNext()` бһҮбһ¶бһҳбһ»бһ“ бһ бһҫбһҷбһ…бһ…бҹҒбһҹбһ бҹ… `next()` бһ‘бһ¶бҹҶбһ„бһҠбҹӮбһӣбһўбһҸбҹӢбһҳбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бҹ„бҹҮ Iterator бһҠбҹҸбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһӮбһ”бҹӢ `NoSuchElementException` бһ—бҹ’бһӣбһ¶бһҳбҹ—! бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Java бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһҲбһ”бҹӢбһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҒбһ»бһҹбһҶбҹ’бһӮбһ„бҹ”

**бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚ:** бһ…бһ»бҹҮбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ Iterator бһҳбһ¶бһ“бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһўбҹ’бһңбһёбһӣбһҫбһҹбһ–бһёбһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡ `for-loop` бһ’бһҳбҹ’бһҳбһҸбһ¶? 

**бһ”бҹҒбһҖбҹ’бһҒбһҮбһ“ (бһҹбһ»бһҒбһ¶):** бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҠбҹҸбһ’бҹҶбһ”бҹҶбһ•бһ»бһҸбһӮбһә "Lazy Traversal" бһӣбҹ„бһҖбһ”бҹ’бһҡбһ’бһ¶бһ“! бһңбһ¶бһҳбһ·бһ“бһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҳбһҖбһҠбһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„ Memory бһ–бҹ’бһҡбһҳбһӮбҹ’бһ“бһ¶бһ‘бҹҒбҹ” бһңбһ¶бһ…бһ„бһ…бһ¶бҹҶбһ‘бһёбһҸбһ¶бҹҶбһ„ (Cursor/State) бһҡбһ”бһҹбҹӢбһңбһ¶бҹ” бһ–бҹҒбһӣбһ бҹ… `next()` бһҳбҹ’бһҸбһ„ бһ‘бһҫбһ”бһңбһ¶бһ‘бҹ…бһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбҹ’бһҸбһ„! бһ“бҹҒбҹҮбһҹбҹҗбһҖбҹ’бһҸбһ·бһҹбһҳбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ’бҹҶбҹ—бһ–бһё Database (Pagination) бһҳбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ” бһҳбҹ’бһҷбҹүбһ¶бһ„бһ‘бҹҖбһҸ бһҷбһҫбһ„бһўбһ¶бһ…бһҳбһ¶бһ“ Iterator бһ…бҹ’бһҡбһҫбһ“бһҡбһҸбҹӢбһӣбһҫ Collection бһҸбҹӮбһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷбһ”бһ¶бһ“бһҠбҹ„бһҷбһҜбһҖбһҡбһ¶бһҮбҹ’бһҷбһ–бһёбһӮбҹ’бһ“бһ¶!

**бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚ (бһ„бһҖбҹӢбһҖбҹ’бһ”бһ¶бһӣбһ–бҹҒбһүбһ…бһ·бһҸбҹ’бһҸ):** бһ–бһ“бҹ’бһҷбһӣбҹӢбһ”бһ¶бһ“бһӣбҹ’бһўбһҺбһ¶бһҹбҹӢ! бһ…бҹҶбһҺбһ»бһ… Lazy Traversal бһӮбһәбһҮбһ¶бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбһҫбһ„бһ…бһ¶бһҹбҹӢбһҸбҹӮбһ„бһҸбҹӮбһ”бҹ’бһҡбһҫ!

## бҹў. бһҖбһјбһҠбһҠбҹӮбһӣбһ”бҹҒбһҖбҹ’бһҒбһҮбһ“бһҹбһҡбһҹбҹҒбһҡ (Candidate's Code)

```java
// бҹЎ. Iterator Interface (бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбҹҶбһ“бһҪбһҷбһҖбһ¶бһҡ)
public interface Iterator<T> {
    boolean hasNext();
    T next();
}

// бҹў. Iterable Interface (Collection бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһўбҹ„бһҷбһӮбҹҒ Loop бһӣбһҫбһ”бһ¶бһ“)
public interface IterableCollection<T> {
    Iterator<T> createIterator();
}

// бҹЈ. Concrete Collection (бһҖбһ“бҹ’бһӣбҹӮбһ„бһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ)
public class BookCollection implements IterableCollection<String> {
    private String[] books = {"Design Patterns", "Clean Code", "Refactoring"};

    @Override
    public Iterator<String> createIterator() {
        return new BookIterator(this); 
    }

    public String[] getBooks() { return books; }
}

// бҹӨ. Concrete Iterator (бһўбҹ’бһ“бһҖбһҠбһ№бһ„бһ–бһёбһҡбһ”бҹҖбһ” Loop)
public class BookIterator implements Iterator<String> {
    private BookCollection collection;
    private int currentIndex = 0; // State бһҡбһ”бһҹбҹӢ Iterator (Cursor)

    public BookIterator(BookCollection collection) {
        this.collection = collection;
    }

    @Override
    public boolean hasNext() {
        return currentIndex < collection.getBooks().length;
    }

    @Override
    public String next() {
        // бһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһ¶бһҡбһ бҹ… next() бһ•бҹ’бһҸбҹҒбһҹбһ•бҹ’бһҸбһ¶бһҹ
        if (!hasNext()) {
            throw new NoSuchElementException("бһўбһҹбҹӢбһҹбҹҖбһңбһ—бҹ…бһ бһҫбһҷбһӣбҹ„бһҖбһ”бҹ’бһҡбһ’бһ¶бһ“!");
        }
        return collection.getBooks()[currentIndex++];
    }
}
```

## бҹЈ. бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚ (Interviewer's Evaluation)

вң… **бһ…бҹҶбһҺбһ»бһ…бһҒбҹ’бһӣбһ¶бҹҶбһ„:** бһҷбһӣбҹӢбһ…бҹ’бһ”бһ¶бһҹбҹӢбһ–бһёбһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖ Traversal Logic бһ…бҹҒбһүбһ–бһё Business Logic бһ“бһ·бһ„ Collectionбҹ” бһ–бһ“бҹ’бһҷбһӣбҹӢбһ”бһ¶бһ“бһҖбҹ’бһ”бҹ„бҹҮбһҖбҹ’бһ”бһ¶бһҷбһ–бһёбһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ“бҹғ Lazy Traversal бһҠбҹӮбһӣбһҹбһ“бҹ’бһҹбҹҶ Memory бһ–бҹҒбһӣбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ’бҹҶбҹ—бҹ”
вқҢ **бһўбһ“бҹ’бһ‘бһ¶бһҖбҹӢ:** бһҹбһҪбһҡбһ–бһёбһҖбһҡбһҺбһёбһ бҹ… `next()` бһӣбһҫбһҹбһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбҹ” бһ”бҹҒбһҖбҹ’бһҒбһҮбһ“бһҶбҹ’бһӣбһҫбһҷбһ”бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҗбһ¶бһҸбҹ’бһҡбһјбһңбһӮбһ”бҹӢ Exceptionбҹ”
