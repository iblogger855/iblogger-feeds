## бҹЎ. бһҖбһ·бһ…бҹ’бһ…бһҹбһ“бҹ’бһ‘бһ“бһ¶бһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚ (Interview Transcript)

**бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚ (бһӣбҹ„бһҖбһ”бҹ’бһҡбһ’бһ¶бһ“ - бһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһҹбһҪбһҡбһҡбһҖбһҡбһҝбһ„):** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Project бһ…бһ¶бһҹбҹӢбһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһҳбһ¶бһ“ File бһҳбһҪбһҷбһ”бҹ’бһҡбһңбҹӮбһ„ бҹўбһ–бһ¶бһ“бҹӢбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢ бһҠбҹӮбһӣбһ–бҹ„бһҡбһ–бҹҒбһүбһ‘бҹ…бһҠбҹ„бһҷ `switch(loginType)`бҹ” бһ”бһҫ `loginType == "facebook"` бһңбһ¶ `new FacebookAuth()`, бһ”бһҫ `"google"` бһңбһ¶ `new GoogleAuth()`бҹ” бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһҷбһҫбһ„бһҗбҹӮбһҳбһңбһ·бһ’бһё Login бһҗбҹ’бһҳбһё (бһҠбһјбһ…бһҮбһ¶ Apple, TikTok) бһҖбҹ’бһҡбһ»бһҳбһҖбһ¶бһҡбһ„бһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ…бһјбһӣбһҳбһҖбһҖбҹӮ File бһҳбҹҒбһ бҹ’бһ“бһ№бһ„бһҡбһ бһјбһҸ бһҡбһ бһјбһҸбһҠбһӣбҹӢбһҗбҹ’бһ„бҹғбһҳбһҪбһҷбһҳбһ¶бһ“бһҖбһјбһ“бһ…бҹ…бһҳбҹ’бһ“бһ¶бһҖбҹӢбһҖбҹӮбһҒбһ»бһҹ бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ Login бһҠбһҪбһӣбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣ! бһҸбһҫбһ”бҹ’бһўбһјбһ“бһҳбһ¶бһ“бһҖбҹ’бһ”бһҪбһ“бһўбһёбһҮбһҪбһҷбһҹбҹ’бһҡбҹ„бһ…бһҹбҹ’бһҡбһ„бҹӢбһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһҷбһҫбһ„бһ‘бҹҒ?

**бһ”бҹҒбһҖбҹ’бһҒбһҮбһ“ (бһҹбһ»бһҒбһ¶ - бһҳбҹ„бҹҮбһҳбһ»бһҸ):** бһӣбҹ„бһҖбһ”бҹ’бһҡбһ’бһ¶бһ“! бһ”бһүбҹ’бһ бһ¶бһ“бҹҒбҹҮбһ”бҹҶбһ–бһ¶бһ“бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚ Open/Closed Principle (OCP) бһҷбҹүбһ¶бһ„бһ’бҹ’бһ„бһ“бҹӢбһ’бҹ’бһ„бһҡбҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ бһҒбҹ’бһүбһ»бҹҶбһ“бһ№бһ„бһ”бһүбҹ’бһ…бһ»бҹҮбһ”бһүбҹ’бһ…бһјбһӣ **Factory Method Pattern**бҹ” бһҒбҹ’бһүбһ»бҹҶбһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸ `AuthService` бһҮбһ¶ Class бһҳбҹҒ бһҠбҹӮбһӣбһҳбһ¶бһ“ Method бһҳбһҪбһҷбһҲбҹ’бһҳбҹ„бҹҮбһҗбһ¶ `createAuth()`бҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһҒбҹ’бһүбһ»бҹҶбһ“бһ№бһ„бһҳбһ¶бһ“ Class бһҖбһјбһ“бҹ—бһҠбһјбһ…бһҮбһ¶ `FacebookAuthService` бһ“бһ·бһ„ `GoogleAuthService` бһҮбһ¶бһўбҹ’бһ“бһҖ Implement бһңбһ¶бҹ”

**бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚ:** бһҷбһё! бһ”бҹ’бһўбһјбһ“бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бһёбһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһ–бһёбһҖбһ“бҹ’бһӣбҹӮбһ„бһҳбһҪбһҷбһ‘бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһҳбһҪбһҷбһ‘бҹҖбһҸбһҸбһҫ! бһ”бһҫбһ”бҹ’бһўбһјбһ“бһ”бҹҶбһ”бҹӮбһҖбһҮбһ¶ Class бһҸбһјбһ…бҹ—бһҡбһ¶бһ”бҹӢбһҹбһ·бһ”бһ”бҹӮбһ”бһ“бҹҒбҹҮ бһҸбһҫбһңбһ¶бһҳбһ·бһ“бһ–бһ·бһ”бһ¶бһҖбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ File бһҖбһјбһҠбһҮбһ¶бһ„бһҳбһ»бһ“бһ‘бҹҒбһўбһё? бһҳбҹүбҹҒбһ…бһҳбһ·бһ“бһҠбһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„ File бһҸбҹӮбһҳбһҪбһҷбһ‘бҹ… бһҹбҹ’бһҡбһҪбһӣбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһң (Search) бһғбһҫбһү!

**бһ”бҹҒбһҖбҹ’бһҒбһҮбһ“ (бһҹбһ»бһҒбһ¶ - бһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҷбҹүбһ¶бһ„бһҸбҹ’бһҡбһҮбһ¶бһҖбҹӢ):** бһ”бһҫбһҠбһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ“бҹ’бһӣбҹӮбһ„бһҸбҹӮбһҳбһҪбһҷ бһӣбҹ„бһҖбһ”бҹ’бһҡбһ’бһ¶бһ“бһ“бһ№бһ„бһҮбһҪбһ”бһ”бҹ’бһҡбһ‘бҹҮ "Merge Conflict" бһҮбһ¶бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһҗбҹ’бһ„бҹғ бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣ Developers бһ…бҹ’бһҡбһҫбһ“бһ“бһ¶бһҖбҹӢбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһҖбҹӮбһҖбһјбһҠбһҗбҹӮбһҳ Social Login бһ–бҹ’бһҡбһҳбһӮбҹ’бһ“бһ¶бҹ” бһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖбһҮбһ¶ Class бһҸбһ¶бһҳбһ”бҹӮбһ” Factory Method бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷ Developer бһҳбҹ’бһ“бһ¶бһҖбҹӢбҹ—бһҹбһҡбһҹбҹҒбһҡ Class бһҗбҹ’бһҳбһёбһҠбҹ„бһҷбһҒбҹ’бһӣбһҪбһ“бһҜбһ„ бһҠбҹ„бһҷбһҳбһ·бһ“бһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһҖбһјбһҠбһҹбҹ’бһ“бһјбһӣ (Core) бһҡбһ”бһҹбҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһЎбһҫбһҷбҹ” бһ”бһҫбһҖбһјбһҠ Apple Login бһҠбҹӮбһӣбһ‘бһҫбһ”бһҹбһҡбһҹбҹҒбһҡбһҗбҹ’бһҳбһёбһҳбһ¶бһ“ Bug бһңбһ¶бһ’бҹ’бһӣбһ¶бһҖбҹӢбһҸбҹӮ Apple Login бһ‘бҹҒ... Google бһ“бһ·бһ„ Facebook бһ“бҹ…бһҠбһҫбһҡбһҮбһ¶бһ’бһҳбҹ’бһҳбһҸбһ¶!

**бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚ:** бһ…бһ»бҹҮбһ”бһҫбһҒбҹ’бһүбһ»бҹҶбһ”бҹ’бһҡбһ¶бһ”бҹӢбһҗбһ¶ бһҒбҹ’бһүбһ»бҹҶбһҒбҹ’бһҮбһ·бһӣбһ”бһ„бҹ’бһҖбһҫбһҸ Subclass (Factory бһҖбһјбһ“бҹ—) бһ…бҹ’бһҡбһҫбһ“бһ–бҹҒбһҖбһҺбһ¶бһҹбҹӢ! бһҒбҹ’бһүбһ»бҹҶбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ…бһ„бҹӢбһ бҹ… `AuthFactory.create("facebook")` бһҳбһҪбһҷбһҳбҹүбһ¶бһҸбҹӢбһҖбһ¶бһҸбҹӢбһ“бҹҗбһҷбһ”бһ¶бһ“бһўбһҸбҹӢ?

**бһ”бҹҒбһҖбҹ’бһҒбһҮбһ“ (бһҹбһ»бһҒбһ¶):** бһўбһј! бһ”бһҫбһӣбҹ„бһҖбһ”бҹ’бһҡбһ’бһ¶бһ“бһ…бһ„бҹӢбһ”бһ¶бһ“бһ”бҹӮбһ”бһ бҹ’бһ“бһ№бһ„ бһ“бҹ„бҹҮбһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶ Factory Method Pattern бһҠбҹӮбһӣбһҳбһ¶бһ“бһ…бҹӮбһ„бһҖбҹ’бһ“бһ»бһ„бһҹбҹҖбһңбһ—бҹ… GoF бһ‘бҹҒ! бһ“бҹ„бҹҮбһӮбҹҒбһ бҹ…бһҗбһ¶ **Simple Factory (бһ¬ Static Factory)** бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫ `switch/case` бһҠбһҠбҹӮбһӣ бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһӣбһ¶бһҖбҹӢбһңбһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Class бһ•бҹ’бһҹбҹҒбһ„бҹ” бһңбһ¶бһ“бҹ…бһҸбҹӮбһ”бҹҶбһ–бһ¶бһ“ OCP бһҠбһҠбҹӮбһӣбһ–бҹ’бһҡбҹ„бҹҮбһ”бһҫбһҗбҹӮбһҳ TikTok бһӣбҹ„бһҖбһ”бҹ’бһҡбһ’бһ¶бһ“бһҸбҹ’бһҡбһјбһңбһ…бһјбһӣбһ‘бҹ…бһҖбҹӮбһҖбҹ’бһ“бһ»бһ„ `switch` бһ“бҹ„бҹҮбһҠбһҠбҹӮбһӣбҹ”
бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮ... бһ”бһҫбһӣбҹ„бһҖбһ”бҹ’бһҡбһ’бһ¶бһ“бһ…бһ„бҹӢбһ”бҹ’бһҡбһҫ Simple Factory бһ бһҫбһҷбһҳбһ·бһ“бһ…бһ„бҹӢбһ”бҹҶбһ–бһ¶бһ“ OCP бһҒбҹ’бһүбһ»бҹҶбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҖбҹ’бһ”бһ¶бһ…бҹӢ **Reflection Registry** бһ”бһ¶бһ“! бһұбҹ’бһҷбһҸбҹӮбһҳбһ¶бһ“ Class бһҗбҹ’бһҳбһёбһҖбһҫбһҸбһЎбһҫбһ„ бһҒбҹ’бһүбһ»бҹҶбһұбҹ’бһҷбһңбһ¶бһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮ (Register) бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„ Map (Dictionary) бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ” бһ–бҹҒбһӣбһ бҹ… бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ‘бһ¶бһүбһ…бҹҒбһүбһ–бһё Map бһҳбһҖбһҮбһ¶бһҖбһ¶бһҡбһҹбҹ’бһҡбҹҒбһ… бһҳбһ·бһ“бһ”бһ¶бһ…бҹӢбһҹбһҡбһҹбҹҒбһҡ `if/else` бһЎбһҫбһҷ!

**бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚ (бһ„бһҖбҹӢбһҖбҹ’бһ”бһ¶бһӣбһ–бҹҒбһүбһ…бһ·бһҸбҹ’бһҸ):** бһ–бһјбһҖбҹӮбһҺбһ¶бһҹбҹӢбһҖбҹ’бһҳбһҪбһҷ! бһҹбҹ’бһӮбһ¶бһӣбҹӢбһ‘бһ¶бҹҶбһ„бһ‘бҹ’бһҡбһ№бһҹбҹ’бһҸбһё (GoF Factory Method) бһ“бһ·бһ„бһҹбҹ’бһӮбһ¶бһӣбҹӢбһ‘бһ¶бҹҶбһ„бһҖбһ¶бһҡбһ”бһҸбҹӢбһ”бҹӮбһ“бһҖбҹ’бһ“бһ»бһ„бһ–бһ·бһ—бһ–бһ–бһ·бһҸ (Simple Factory with Registry)бҹ”

## бҹў. бһҖбһјбһҠбһҠбҹӮбһӣбһ”бҹҒбһҖбҹ’бһҒбһҮбһ“бһҹбһҡбһҹбҹҒбһҡ (Candidate's Code)

```java
// бҹЎ. Product Interface
public interface AuthProvider {
    boolean login(String token);
}

// бҹў. Concrete Products
public class FacebookAuth implements AuthProvider {
    public boolean login(String token) { return true; /* Facebook logic */ }
}
public class GoogleAuth implements AuthProvider {
    public boolean login(String token) { return true; /* Google logic */ }
}

// бҹЈ. Creator (Base Class бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһӮбҹ’бһ“бһ¶бһҸбҹ’бһҡбһјбһңбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖ)
public abstract class AuthService {
    // бһ“бҹҒбҹҮбһ бһҫбһҷбһӮбһә Factory Method!
    protected abstract AuthProvider createProvider();

    // Core Logic бһҠбҹӮбһӣбһўбһҸбҹӢбһҒбҹ’бһңбһӣбҹӢбһҗбһ¶бһҮбһ¶ FB бһ¬ Google бһ‘бҹҒ
    public void processLogin(String token) {
        AuthProvider provider = createProvider();
        if (provider.login(token)) {
            System.out.println("Login Success! Granting access...");
        }
    }
}

// бҹӨ. Concrete Creators (бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһўбһ¶бһ…бһ”бһ“бҹ’бһҗбҹӮбһҳ Class бһ“бҹҒбҹҮбһҠбҹ„бһҷбһҳбһ·бһ“бһ”бһ¶бһ…бҹӢбһҖбҹӮбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ…бһ¶бһҹбҹӢ)
public class FacebookAuthService extends AuthService {
    @Override
    protected AuthProvider createProvider() {
        return new FacebookAuth();
    }
}

public class GoogleAuthService extends AuthService {
    @Override
    protected AuthProvider createProvider() {
        return new GoogleAuth();
    }
}
```

## бҹЈ. бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚ (Interviewer's Evaluation)

вң… **бһ…бҹҶбһҺбһ»бһ…бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҡбһ”бһҹбҹӢбһ”бҹҒбһҖбҹ’бһҒбһҮбһ“:**
- бһҷбһӣбҹӢбһ…бҹ’бһ”бһ¶бһҹбҹӢбһ–бһё Open/Closed Principle бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶ `if/else` бһ¬ `switch` бһңбҹӮбһ„бһўбһ“бҹ’бһӣбһ¶бһҷбҹ”
- бһҷбһӣбҹӢбһ–бһёбһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„ **Factory Method Pattern (GoF)** бһ”бҹ’бһҡбһҫ Inheritance бһ“бһ·бһ„ **Simple Factory** (бһ”бҹ’бһҡбһҫ Parameter)бҹ”
- бһҷбһӣбҹӢбһ–бһёбһӮбһ»бһҺбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ·бһ“бҹғбһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖбһҖбһјбһҠ (Decoupling) бһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹҖбһҹбһңбһ¶бһ„ Merge Conflicts бһ“бҹ…бһ–бҹҒбһӣбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҖбҹ’бһҡбһ»бһҳбҹ”

вқҢ **бһ…бҹҶбһҺбһ»бһ…бһҒбҹ’бһҹбҹ„бһҷ (Trap Question):**
- бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһ—бһ¶бһҹбһ“бҹҚбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһ”бһүбҹ’бһ…бһ»бҹҮбһ”бһүбҹ’бһ…бһјбһӣбһұбҹ’бһҷбһ”бҹҒбһҖбҹ’бһҒбһҮбһ“бһ”бҹ’бһҡбһҫ Simple Factory бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷбһҖбһјбһҠбһҒбҹ’бһӣбһёбҹ” бһ”бҹҒбһҖбҹ’бһҒбһҮбһ“бһҶбҹ’бһӣбһ¶бһҸбһҠбҹӮбһӣбһҠбһ№бһ„бһҗбһ¶ Simple Factory бһ“бҹ…бһҸбҹӮбһ”бҹҶбһ–бһ¶бһ“ OCP бһ бһҫбһҷбһҗбҹӮбһҳбһ‘бһ¶бҹҶбһ„бһ•бҹ’бһҸбһӣбҹӢбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ Registry (Map) бһ‘бҹҖбһҸбһ•бһ„бҹ”

**бһӣбһ‘бҹ’бһ’бһ•бһӣ:** бһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢ! (Pass)
