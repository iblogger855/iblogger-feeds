## 🗺️ How to Use

Pick a topic, then choose the strategy that matches how you learn:

- **Abstract thinker?** → MIT Professor (first principles)
- **Need simplicity first?** → Feynman Technique
- **Remember through stories?** → Parable or Storyteller
- **Learn by overhearing conversation?** → Podcast
- **Preparing for a job interview?** → Interview

All strategy docs for the same topic cross-link each other — start anywhere.
