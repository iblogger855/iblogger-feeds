## бҹЎ. бһҸбһҫбһўбҹ’бһңбһёбһҮбһ¶ CIS Benchmarks? (What It Is)

CIS Benchmarks бһӮбһәбһҮбһ¶бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҺбҹӮбһ“бһ¶бҹҶбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҠбҹ„бһҷбһҹбһ бһӮбһҳбһ“бҹҚ бһ“бһ·бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҖбһ¶бһҡбһ–бҹ’бһҡбһҳбһ–бҹ’бһҡбҹҖбһ„бһӮбҹ’бһ“бһ¶ бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһңбһ·бһ‘бҹ’бһҷбһ¶бһ”бҹ’бһҡбһҖбһ”бһҠбҹ„бһҷбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бҹ” бһ–бһҪбһҖбһңбһ¶бһ•бҹ’бһҠбһӣбҹӢбһ“бһјбһңбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһӣбһҳбҹ’бһўбһ·бһҸбһҮбһ¶бһҮбҹҶбһ бһ¶бһ“ бҹ— бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһ№бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– (Hardening) бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҖбҹ’бһӣбҹ„бһҠ (Cloud Platforms) бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡ бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„ бҹ—бҹ”

CIS Benchmarks are community-developed, consensus-based configuration guidelines for securely configuring IT systems. They provide specific, step-by-step instructions for hardening operating systems, cloud platforms, databases, web servers, and applications.

бһҳбһ¶бһ“ **CIS Benchmarks бһҮбһ¶бһ„ бҹЎбҹ бҹ ** бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҠбһ”бҹӢбһӣбһҫбһ•бҹ’бһӣбҹҒбһҸбһ бҹ’бһңбһҳбһ–бҹҒбһүбһ“бһ·бһҷбһҳбһҹбҹ’бһ‘бһҫбһҡбһҸбҹӮбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ” бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһўбһ¶бһ…бһ‘бһ¶бһүбһҷбһҖбһҠбҹ„бһҷбһҹбҹҒбһҡбһё бһ“бһ·бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҷбҹүбһ¶бһ„бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҮбһ¶бһӮбҹ’бһҡбһ№бҹҮбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ SOC 2 бһ“бһ·бһ„ ISO 27001бҹ”

There are **100+ CIS Benchmarks** covering virtually every mainstream platform. They are free to download and widely used as the technical foundation for SOC 2 and ISO 27001 controls.

```
в–Ў бһӮбҹ„бһӣбһ“бһҷбҹ„бһ”бһ¶бһҷбһ”бһҺбҹ’бһҠбһ¶бһү вҖ” бһ”бһҠбһ·бһҹбҹҒбһ’бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҸбһ¶бһҳбһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳ, бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһҸбҹӮбһўбҹ’бһңбһёбһҠбҹӮбһӣбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһүбҹ’бһҮбһёбһҹ (Whitelist)
в–Ў бһҖбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһ•бһҸ (Pod Security Admission) вҖ” бһҡбһ№бһҸбһҸбҹ’бһ”бһ·бһҸбһ•бһҸбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ·бһ‘бҹ’бһ’бһ·бһҒбҹ’бһ–бһҹбҹӢ (Privileged Pods)
в–Ў бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбһјбһ“бһҹбҹ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһјбһ“бһҹбҹ„бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ… (бһҳбһ·бһ“бһҳбҹӮбһ“бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһҮбһ¶ k8s secrets бһ’бһҳбҹ’бһҳбһҸбһ¶бһЎбһҫбһҷ)
в–Ў бһҹбҹ’бһҖбҹҒбһ“бһҡбһјбһ”бһ—бһ¶бһ– (Image Scanning) бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹҶбһ–бһ„бҹӢбһ”бһ„бҹ’бһ бһјбһҡ CI/CD
в–Ў бһӮбҹ’бһҳбһ¶бһ“бһҖбһ»бһ„бһҸбһәбһ“бҹҗбһҡбһҺбһ¶бһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбҹ’бһ“бһ»бһ„бһ“бһ¶бһҳбһҮбһ¶ Root бһЎбһҫбһҷ
```
