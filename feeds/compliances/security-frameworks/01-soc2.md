## бҹЎ. бһҸбһҫбһўбҹ’бһңбһёбһҮбһ¶ SOC 2? (What It Is)

SOC 2 бһӮбһәбһҮбһ¶бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҹбһңбһ“бһҖбһҳбҹ’бһҳбһҳбһҪбһҷ бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһ…бҹ’бһ”бһ¶бһ”бҹӢбһ“бҹ„бҹҮбһЎбһҫбһҷбҹ” бһңбһ¶бһҮбһ¶бһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһҠбҹӮбһӣбһ…бҹҒбһүбһҠбҹ„бһҷбһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһӮбһҺбһ“бҹҒбһҷбҹ’бһҷбһҖбһҡбһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲбһҜбһҖбһҡбһ¶бһҮбҹ’бһҷ (CPA Firm) бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶бһҸбһҫбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҡбһ”бһҹбҹӢбһҹбҹ’бһҗбһ¶бһ”бҹҗбһ“бһ•бҹ’бһҠбһӣбҹӢбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ бһҹбҹ’бһҡбһ”бһ‘бҹ…бһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһңбһ·бһ“бһ·бһ…бҹ’бһҶбҹҗбһҷбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸ (Trust Services Criteria - TSC) бһҠбҹӮбһҡбһ¬бһ‘бҹҒбҹ” бһўбһҸбһ·бһҗбһ·бһҮбһ“ (бһҮбһ¶бһ–бһ·бһҹбҹҒбһҹбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһ’бҹҶ бҹ—) бһҸбҹӮбһ„бһҸбҹӮбһҸбһҳбҹ’бһҡбһјбһңбһұбҹ’бһҷбһҳбһ¶бһ“бһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚ SOC 2 бһҳбһ»бһ“бһ“бһ№бһ„бһ…бһ»бҹҮбһ бһҸбҹ’бһҗбһӣбҹҒбһҒбһ¶бһӣбһҫбһҖбһ·бһ…бҹ’бһ…бһҹбһ“бҹ’бһҷбһ¶бҹ”

SOC 2 is an auditing standard вҖ” not a law. It is a report issued by an independent CPA firm that attests whether a service organisation's controls meet the Trust Services Criteria (TSC). Customers (especially enterprises) frequently require a SOC 2 report before signing contracts.

бһҳбһ¶бһ“бһ–бһёбһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘бһӮбһәбҹ–

There are two types:

- **SOC 2 Type I** вҖ” бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҷбҹүбһ¶бһ„бһҹбһҳбһҹбҹ’бһҡбһ”бһ“бҹ…бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҺбһ¶бһҳбһҪбһҷбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ
- **SOC 2 Type I** вҖ” Controls are suitably designed at a point in time
- **SOC 2 Type II** вҖ” бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бҹ’бһҡбһҖбһ”бһҠбҹ„бһҷбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣбһҖбҹҶбһҺбһҸбҹӢбһҺбһ¶бһҳбһҪбһҷ (бһҮбһ¶бһ‘бһјбһ‘бҹ…бһ–бһё бҹҰ бһ‘бҹ… бҹЎбҹў бһҒбҹӮ) бһҠбҹӮбһӣбһҳбһ¶бһ“бһ—бһ¶бһ–бһҡбһ№бһ„бһҳбһ¶бҹҶ бһ“бһ·бһ„бһӮбһҪбһҡбһұбҹ’бһҷбһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһҮбһ¶бһ„бһҳбһ»бһ“
- **SOC 2 Type II** вҖ” Controls operated effectively over a period (usually 6вҖ“12 months) вҖ” much stronger and more trusted

## бҹў. бһӣбһҖбҹ’бһҒбһңбһ·бһ“бһ·бһ…бҹ’бһҶбҹҗбһҷбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ‘бһ¶бҹҶбһ„ бҹҘ (The 5 Trust Services Criteria - TSC)

| бһ”бҹ’бһҡбһ—бҹҒбһ‘ (Category) | бһңбһ·бһҹбһ¶бһӣбһ—бһ¶бһ–бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҠбһ”бҹӢ (What it covers) | бһҸбһҳбҹ’бһҡбһјбһңбһұбҹ’бһҷбһҳбһ¶бһ“бһҮбһ¶бһ“бһ·бһ…бҹ’бһ…? (Always required?) |
|:---|:---|:---|
| **бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒ (Security)** <br/> Security | бһҖбһ¶бһҡбһ–бһ¶бһҡбһ”бҹ’бһҡбһҶбһ¶бҹҶбһ„бһ“бһ№бһ„бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸ <br/> Protection against unauthorised access | бһ”бһ¶бһ‘ вҖ” бһҮбһ¶бһҖбһ¶бһҸбһ–бҹ’бһңбһҖбһ·бһ…бҹ’бһ… <br/> Yes вҖ” mandatory |
| **бһ—бһ¶бһ–бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһ¶бһ“ (Availability)** <br/> Availability | бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҳбһ¶бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбһ¶бһҳбһҖбһ¶бһҡбһ”бҹ’бһҠбҹҒбһҮбҹ’бһүбһ¶бһ…бһ·бһҸбҹ’бһҸ <br/> System is available as committed | бһҮбһҳбҹ’бһҡбһҫбһҹ <br/> Optional |
| **бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ“бҹғбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ (Processing Integrity)** <br/> Processing Integrity | бһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӮбһәбһ–бҹҒбһүбһӣбҹҒбһү бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ <br/> Processing is complete, valid, accurate | бһҮбһҳбҹ’бһҡбһҫбһҹ <br/> Optional |
| **бһ—бһ¶бһ–бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ (Confidentiality)** <br/> Confidentiality | бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбһ¶бһҡбһ–бһ¶бһҡ <br/> Confidential information is protected | бһҮбһҳбҹ’бһҡбһҫбһҹ <br/> Optional |
| **бһҜбһҖбһҮбһ“бһ—бһ¶бһ– (Privacy)** <br/> Privacy | бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бһ¶бһҸбҹӢбһ…бҹӮбһ„бһҸбһ¶бһҳбһҖбһ¶бһҡбһ”бҹ’бһҠбҹҒбһҮбҹ’бһүбһ¶бһ…бһ·бһҸбҹ’бһҸ <br/> Personal information is handled per commitments | бһҮбһҳбҹ’бһҡбһҫбһҹ <br/> Optional |

бһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҷбһҖ бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒ (Security) + бһ—бһ¶бһ–бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһ¶бһ“ (Availability) + бһ—бһ¶бһ–бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ (Confidentiality) бһҮбһ¶бһҖбһүбҹ’бһ…бһ”бҹӢбһўбһ”бҹ’бһ”бһ”бһҡбһҳбһ¶бҹ”

Most companies pursue Security + Availability + Confidentiality as a minimum set.

## бҹЈ. бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ‘бһјбһ‘бҹ… (Common Controls)

### бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒ (CC6 вҖ” бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҸбһ¶бһҳбһҸбһҖбҹ’бһҖбһңбһ·бһҮбҹ’бһҮбһ¶ бһ“бһ·бһ„бһҡбһјбһ”бһңбһ“бҹ’бһҸ) (Security (CC6 вҖ” Logical and Physical Access))

- бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ–бһ бһ»бһҖбһҸбҹ’бһҸбһ¶ (MFA) бһӣбһҫбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ (Production Systems) бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ
- Multi-factor authentication on all production systems
- бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҸбһҪбһ“бһ¶бһ‘бһё (RBAC) вҖ” бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҹбһ·бһ‘бҹ’бһ’бһ·бһўбһ”бҹ’бһ”бһ”бһҡбһҳбһ¶ (Least Privilege)
- Role-based access control вҖ” least privilege
- бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбһҖбһҹбһ·бһ‘бҹ’бһ’бһ·бһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖ (Offboarding Process) вҖ” бһҖбһ¶бһҡбһӣбһ»бһ”бһ…бҹ„бһӣбһҹбһ·бһ‘бҹ’бһ’бһ·бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣ бҹўбҹӨ бһҳбҹүбҹ„бһ„
- Offboarding process вҖ” access revoked within 24 hours
- бһҖбһ¶бһҡбһҖбһјбһҠбһ“бһёбһҷбһҖбһҳбҹ’бһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бҹҒбһӣбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ (Encryption at Rest) бһ“бһ·бһ„бһ–бҹҒбһӣбһ”бһүбҹ’бһҮбһјбһ“ (Encryption in Transit)
- Encryption at rest and in transit
- бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ—бһ¶бһ–бһ„бһ¶бһҷбһҡбһ„бһӮбҹ’бһҡбҹ„бҹҮ (Vulnerability Management) вҖ” бһҖбһ¶бһҡбһҹбҹ’бһҖбҹҒбһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„бһ”бҹҶбһҺбҹҮ (Patching) бһұбҹ’бһҷбһ”бһ¶бһ“бһ‘бҹҖбһ„бһ‘бһ¶бһҸбҹӢ
- Vulnerability management вҖ” regular scans and patching

### бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡ (CC8) (Change Management (CC8))

- бһҸбһҳбҹ’бһҡбһјбһңбһұбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһјбһҠбһЎбһҫбһ„бһңбһ·бһү (Code Review) бһҳбһ»бһ“бһ–бҹҒбһӣбһ”бһүбҹ’бһ…бһјбһӣбһӮбҹ’бһ“бһ¶ (Merge)
- Code review required before merge
- бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбҹҶбһҺбһҸбҹӢбһҖбҹ’бһ“бһ»бһ„бһ”бҹҶбһ–бһ„бҹӢбһ”бһ„бҹ’бһ бһјбһҡ CI/CD (CI/CD Pipeline)
- Automated testing in CI/CD pipeline
- бһ”бҹӮбһ„бһ…бҹӮбһҖбһҠбһ¶бһ…бҹӢбһҠбҹ„бһҷбһЎбҹӮбһҖбһҡбһңбһ¶бһ„бһҳбһҮбҹ’бһҲбһҠбҹ’бһӢбһ¶бһ“бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ (Development) бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ (Staging) бһ“бһ·бһ„бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ (Production)
- Separate development, staging, and production environments
- бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбһ“бһ»бһҳбҹҗбһҸбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡ (Change Approval Process)
- Change approval process

### бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ бһ¶бһ“бһ·бһ—бҹҗбһҷ (CC3) (Risk Management (CC3))

- бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ бһ¶бһ“бһ·бһ—бҹҗбһҷбһ”бҹ’бһҡбһ…бһ¶бҹҶбһҶбҹ’бһ“бһ¶бҹҶ
- Annual risk assessment
- бһ”бһүбҹ’бһҮбһёбһҲбҹ’бһҳбҹ„бҹҮбһ бһ¶бһ“бһ·бһ—бҹҗбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ‘бһ»бһҖбһҮбһ¶бһҜбһҖбһҹбһ¶бһҡ (Risk Register)
- Documented risk register
- бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ бһ¶бһ“бһ·бһ—бҹҗбһҷбһҡбһ”бһҹбҹӢбһҠбҹғбһӮбһјбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢ (Vendor Risk Assessments)
- Vendor risk assessments

### бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ“бһ№бһ„бһ§бһ”бҹ’бһ”бһҸбҹ’бһҸбһ·бһ бҹҒбһҸбһ» (CC7) (Incident Response (CC7))

- бһ•бҹӮбһ“бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ“бһ№бһ„бһ§бһ”бҹ’бһ”бһҸбҹ’бһҸбһ·бһ бҹҒбһҸбһ»бһҠбҹӮбһӣбһ”бһ¶бһ“бһ…бһ„бһҖбҹ’бһҡбһ„бһҮбһ¶бһҜбһҖбһҹбһ¶бһҡ
- Documented incident response plan
- бһҖбһ¶бһҡбһ”бҹ’бһҠбһјбһҡбһңбҹҒбһ“бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶ (On-call Rotation)
- On-call rotation
- бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһЎбһҫбһ„бһңбһ·бһүбһҖбҹ’бһҡбҹ„бһҷбһ§бһ”бҹ’бһ”бһҸбҹ’бһҸбһ·бһ бҹҒбһҸбһ» (Post-incident Review Process)
- Post-incident review process
- бһ“бһёбһҸбһ·бһңбһ·бһ’бһёбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһҠбһӣбҹӢбһўбһҸбһ·бһҗбһ·бһҮбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһӣбһҪбһ…бһ…бһјбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ (Breaches)
- Customer notification procedure for breaches

### бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ (CC7) (Monitoring (CC7))

- бһҖбһ¶бһҡбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ‘бһ»бһҖбһҖбһҺбҹ’бһҠбһ¶бһӣ (Centralised Logging - SIEM)
- Centralised logging (SIEM)
- бһҖбһ¶бһҡбһ•бҹ’бһҠбһӣбҹӢбһҠбҹҶбһҺбһ№бһ„бһўбҹҶбһ–бһёбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҳбһ·бһ“бһ”бҹ’бһҡбһҖбҹ’бһҡбһҸбһё (Anomalous Activity)
- Alerting on anomalous activity
- бһҖбһ¶бһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶ (Log) бһЎбһҫбһ„бһңбһ·бһүбһұбҹ’бһҷбһ”бһ¶бһ“бһ‘бҹҖбһ„бһ‘бһ¶бһҸбҹӢ
- Regular log review

## бҹӨ. бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбһңбһ“бһҖбһҳбҹ’бһҳ (The Audit Process)

бҹЎ. бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҖбһ¶бһҡбһҸбҹ’бһҡбҹҖбһҳбһҒбҹ’бһӣбһҪбһ“ (бһ•бҹ’бһ‘бҹғбһҖбҹ’бһ“бһ»бһ„)
   в””в”Җв”Җ бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһӮбһҳбҹ’бһӣбһ¶бһҸбһ’бҹҖбһ”бһ“бһ№бһ„бһӣбһҖбҹ’бһҒбһңбһ·бһ“бһ·бһ…бҹ’бһҶбҹҗбһҷ TSC
   в””в”Җв”Җ бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҠбҹӮбһӣбһҒбҹ’бһңбҹҮбһ…бһ“бҹ’бһӣбҹ„бҹҮ

бҹў. бһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһ—бһҹбҹ’бһҸбһ»бһҸбһ¶бһ„
   в””в”Җв”Җ бһҡбһјбһ”бһҗбһҸбһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһӮбҹ„бһӣбһ“бһҷбҹ„бһ”бһ¶бһҷ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶ (Logs)
   в””в”Җв”Җ бһҖбһ¶бһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҠбҹғбһӮбһјбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢ

бҹЈ. бһҖбһ¶бһҡбһ…бһјбһӣбһҡбһҪбһҳбһҡбһ”бһҹбҹӢбһҹбһңбһ“бһҖбһҡ
   в””в”Җв”Җ бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“ CPA бһҠбҹӮбһӣбһҳбһ¶бһ“бһўбһ¶бһҮбҹ’бһүбһ¶бһ”бҹҗбһҺбҹ’бһҺбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
   в””в”Җв”Җ бһ•бҹ’бһҠбһӣбҹӢбһ—бһҹбҹ’бһҸбһ»бһҸбһ¶бһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Type I (бһ…бҹҶбһҺбһ»бһ…бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҺбһ¶бһҳбһҪбһҷ) бһ¬ 
       бһ—бһҹбҹ’бһҸбһ»бһҸбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣ бҹҰ бһ‘бҹ… бҹЎбҹў бһҒбҹӮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Type II

бҹӨ. бһҖбһ¶бһҡбһ…бҹҒбһүбһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһҹбһңбһ“бһҖбһҳбҹ’бһҳ
   в””в”Җв”Җ "Unqualified" = бһҮбһ¶бһ”бҹӢ (бһӮбҹ’бһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢ)
   в””в”Җв”Җ "Qualified" = бһҳбһ¶бһ“бһҖбһ¶бһҡбһҖбһҸбҹӢбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһ–бһёбһ—бһ¶бһ–бһҳбһ·бһ“бһ”бҹ’бһҡбһҖбҹ’бһҡбһҸбһё (бһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢ)

бҹҘ. бһҖбһ¶бһҡбһ”бһ“бҹ’бһҸбһҹбһ»бһ–бһӣбһ—бһ¶бһ–бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҶбҹ’бһ“бһ¶бҹҶ
   в””в”Җв”Җ Type II бһҸбһҳбҹ’бһҡбһјбһңбһұбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҹбһңбһ“бһҖбһҳбҹ’бһҳбһЎбһҫбһ„бһңбһ·бһүбһҮбһ¶бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһҶбҹ’бһ“бһ¶бҹҶ

1. Readiness Assessment (internal)
   в””в”Җв”Җ Gap analysis against TSC criteria
   в””в”Җв”Җ Implement missing controls

2. Evidence Collection
   в””в”Җв”Җ Screenshots, configurations, policies, logs
   в””в”Җв”Җ User access reviews, vendor assessments

3. Auditor Engagement
   в””в”Җв”Җ Select a licensed CPA firm
   в””в”Җв”Җ Provide evidence for Type I (point in time) or
       evidence over 6вҖ“12 months for Type II

4. Audit Report Issued
   в””в”Җв”Җ "Unqualified" = passed
   в””в”Җв”Җ "Qualified" = exceptions noted

5. Annual Renewal
   в””в”Җв”Җ Type II requires annual re-audit

бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһҳбҹ’бһҳ SOC 2 (бһҠбһјбһ…бһҮбһ¶ Vanta бһ“бһ·бһ„ Drata) бһўбһ¶бһ…бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҡбҹҖбһ”бһ…бҹҶбһҹбһңбһ“бһҖбһҳбҹ’бһҳбһ–бһёбһ…бҹ’бһҡбһҫбһ“бһҒбҹӮбһҳбһҖбһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһҹбһ”бҹ’бһҠбһ¶бһ бҹҚбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ бһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһ—бһҹбҹ’бһҸбһ»бһҸбһ¶бһ„бһҮбһ¶бһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ”

SOC 2 automation platforms (Vanta, Drata) can reduce audit preparation time from months to weeks by continuously collecting evidence.

## бҹ§. бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҖбһҫбһҸ (What to Build)

```
бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһүбҹ’бһ…бһ”бҹӢбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһўбһ”бҹ’бһ”бһ”бһҡбһҳбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ SOC 2 Security TSCбҹ–

в–Ў MFA бһӣбһҫбһӮбҹ’бһҡбһ”бҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ (Admin Systems)
в–Ў RBAC вҖ” бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҹбһ·бһ‘бҹ’бһ’бһ·бһўбһ”бҹ’бһ”бһ”бһҡбһҳбһ¶ бһ“бһ·бһ„бһҖбһ¶бһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһЎбһҫбһ„бһңбһ·бһүбһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһҸбҹ’бһҡбһёбһҳбһ¶бһҹ
в–Ў бһ”бһүбҹ’бһҮбһёбһҖбһ¶бһҡбһ„бһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫбһ–бҹҒбһӣбһҠбһҖбһҹбһ·бһ‘бҹ’бһ’бһ·бһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖ вҖ” бһӣбһ»бһ”бһ…бҹ„бһӣбһҹбһ·бһ‘бҹ’бһ’бһ·бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣ бҹўбҹӨ бһҳбҹүбҹ„бһ„
в–Ў бһҖбһ¶бһҡбһҖбһјбһҠбһ“бһёбһҷбһҖбһҳбҹ’бһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бҹҒбһӣбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ (AES-256) бһ“бһ·бһ„бһ–бҹҒбһӣбһ”бһүбҹ’бһҮбһјбһ“ (TLS 1.2+)
в–Ў бһҖбһ¶бһҡбһҹбҹ’бһҖбҹҒбһ“бһҡбһҖбһ—бһ¶бһ–бһ„бһ¶бһҷбһҡбһ„бһӮбҹ’бһҡбҹ„бҹҮ вҖ” бһҷбҹүбһ¶бһ„бһ бҹ„бһ…бһҺбһ¶бһҹбҹӢбһҳбҹ’бһҠбһ„бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҒбҹӮ
в–Ў бһӮбҹ„бһӣбһ“бһҷбҹ„бһ”бһ¶бһҷбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бҹҶбһҺбҹҮ (Patch) вҖ” бһҖбһҳбҹ’бһҡбһ·бһҸбһ’бҹ’бһ„бһ“бҹӢбһ’бҹ’бһ„бһҡбҹ– бҹ§ бһҗбҹ’бһ„бҹғ, бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбҹ– бҹЈбҹ  бһҗбҹ’бһ„бҹғ
в–Ў бһҖбһ¶бһҡбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ‘бһ»бһҖбһҖбһҺбҹ’бһҠбһ¶бһӣ (Centralised Logging) вҖ” бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҷбҹүбһ¶бһ„бһҸбһ·бһ… бҹ©бҹ  бһҗбҹ’бһ„бҹғ
в–Ў бһ•бҹӮбһ“бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ“бһ№бһ„бһ§бһ”бҹ’бһ”бһҸбҹ’бһҸбһ·бһ бҹҒбһҸбһ» вҖ” бһ”бһ¶бһ“бһ…бһ„бһҖбҹ’бһҡбһ„бһҮбһ¶бһҜбһҖбһҹбһ¶бһҡ бһ“бһ·бһ„бһ”бһ¶бһ“бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„
в–Ў бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡ вҖ” бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһјбһҠбһЎбһҫбһ„бһңбһ·бһү + бһҖбһ¶бһҡбһўбһ“бһ»бһҳбҹҗбһҸ
в–Ў бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ бһ¶бһ“бһ·бһ—бҹҗбһҷбһ”бҹ’бһҡбһ…бһ¶бҹҶбһҶбҹ’бһ“бһ¶бҹҶ
в–Ў бһҖбһ¶бһҡбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣбһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“ вҖ” бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҶбҹ’бһ“бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ
в–Ў бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ бһ¶бһ“бһ·бһ—бҹҗбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹғбһӮбһјбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—
в–Ў бһ•бҹӮбһ“бһҖбһ¶бһҡбһ”бһ“бҹ’бһҸбһўбһ¶бһҮбһёбһңбһҖбһҳбҹ’бһҳ (Business Continuity Plan)
в–Ў бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһҖбһ¶бһҡбһ•бҹ’бһҠбһӣбҹӢбһҠбҹҶбһҺбһ№бһ„
```

```
Minimum SOC 2 Security TSC implementation:

в–Ў MFA on all production and admin systems
в–Ў RBAC вҖ” least privilege, access reviews quarterly
в–Ў Offboarding checklist вҖ” access revoked within 24h
в–Ў Encryption at rest (AES-256) and in transit (TLS 1.2+)
в–Ў Vulnerability scanning вҖ” at least monthly
в–Ў Patch management policy вҖ” critical: 7 days, high: 30 days
в–Ў Centralized logging вҖ” 90-day retention minimum
в–Ў Incident response plan вҖ” documented and tested
в–Ў Change management вҖ” code review + approval process
в–Ў Annual risk assessment
в–Ў Security awareness training вҖ” annual, all staff
в–Ў Vendor risk assessments for critical vendors
в–Ў Business continuity plan
в–Ў System monitoring and alerting
```
