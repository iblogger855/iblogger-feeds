**бһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһҮбһ¶бһ”бҹӢбһҖбһ¶бһҸбһ–бҹ’бһңбһҖбһ·бһ…бҹ’бһ… (Applies to)бҹ–** бһңбҹҒбһ‘бһ·бһҖбһ¶бһҗбҹӮбһ‘бһ¶бҹҶбһҹбһ»бһҒбһ—бһ¶бһ–, бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ EHR/EMR, бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбһ»бһҒбһ—бһ¶бһ–, бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҠбһјбһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӮбҹ’бһӣбһёбһ“бһ·бһҖ (Healthcare platforms, EHR/EMR systems, health apps, clinical data exchange)

## бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ•бҹ’бһҸбһҫбһҳ (What It Is)

FHIR (Fast Healthcare Interoperability Resources) бһӮбһәбһҮбһ¶бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҹбһҖбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбҹ„бҹҮбһҠбһјбһҡбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбһ»бһҒбһ¶бһ—бһ·бһ”бһ¶бһӣбһҸбһ¶бһҳбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһўбҹҒбһЎбһ·бһ…бһҸбҹ’бһҡбһјбһ“бһ·бһҖбҹ” бһңбһ¶бһ…бҹӮбһ„бһўбҹҶбһ–бһёбһӮбҹҶбһҡбһјбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһҪбһҳ бһ“бһ·бһ„бһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё RESTful API бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӮбҹ’бһӣбһёбһ“бһ·бһҖ бһ“бһ·бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһҠбҹ’бһӢбһ”бһ¶бһӣбһҹбһ»бһҒбһ¶бһ—бһ·бһ”бһ¶бһӣ вҖ” бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбһ»бһҒбһ¶бһ—бһ·бһ”бһ¶бһӣбһ•бҹ’бһҹбҹҒбһ„ бҹ— бһӮбҹ’бһ“бһ¶бһўбһ¶бһ…бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹ’бһ“бһҖбһҮбҹҶбһ„бһәбһ‘бҹ…бһңбһ·бһүбһ‘бҹ…бһҳбһҖбһҠбҹ„бһҷбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ“бһ·бһ„бһўбһҸбҹ’бһҗбһ“бҹҗбһҷбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбҹ”

FHIR (Fast Healthcare Interoperability Resources) is a standard for exchanging healthcare information electronically. It defines a common data model and RESTful API for clinical and administrative health data вҖ” enabling different healthcare systems to share patient information securely and meaningfully.

бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ FHIR бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһ”бһ‘бһ”бһүбҹ’бһүбһҸбҹ’бһҸбһ·бһ…бҹ’бһ”бһ¶бһ”бҹӢбһҠбҹ„бһҷбһҒбҹ’бһӣбһҪбһ“бһҜбһ„бһЎбһҫбһҷ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһӮбһ¶бһ”бҹӢбһұбҹ’бһҷбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҠбҹ„бһҷбһ…бҹ’бһ”бһ¶бһ”бҹӢбһҮбһ¶бһҸбһ·бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“ (бһҠбһјбһ…бһҮбһ¶ бһ…бҹ’бһ”бһ¶бһ”бҹӢ US 21st Century Cures Act бһҡбһ”бһҹбҹӢбһўбһ¶бһҳбҹҒбһҡбһ·бһҖ бһ“бһ·бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ My Health Record бһҡбһ”бһҹбҹӢбһўбһјбһҹбҹ’бһҸбҹ’бһҡбһ¶бһӣбһё) бһ бһҫбһҷбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһңбһ¶бһ”бһ¶бһ“бһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„ (de facto standard) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһ»бһҒбһ¶бһ—бһ·бһ”бһ¶бһӣбһҹбһҳбҹҗбһҷбһ‘бҹҶбһ“бһҫбһ”бҹ”

FHIR is not a regulatory requirement itself, but it is mandated by several regulations (US 21st Century Cures Act, Australian My Health Record) and is the de facto standard for modern healthcare integration.

## бҹЎ. бһӮбҹ„бһӣбһӮбҹҶбһ“бһ·бһҸбһҹбҹ’бһ“бһјбһӣ (Core Concepts)

### бҹЎ.бҹЎ бһ’бһ“бһ’бһ¶бһ“ (Resources)
бһ…бҹ’бһ”бһ¶бһ”бҹӢбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ FHIR бһҡбҹҖбһ”бһ…бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһ»бһҒбһ—бһ¶бһ–бһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘ В«бһ’бһ“бһ’бһ¶бһ“ (Resources)В» вҖ” бһҠбҹӮбһӣбһҮбһ¶бһңбһҸбҹ’бһҗбһ»бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбҹүбһјбһҢбһ»бһӣбһҠбһ¶бһ…бҹӢбһҠбҹ„бһҷбһЎбҹӮбһҖбһ–бһёбһӮбҹ’бһ“бһ¶ (modular, self-contained data objects)бҹ” бһ’бһ“бһ’бһ¶бһ“бһ“бһёбһҳбһҪбһҷ бҹ— бһҳбһ¶бһ“бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹҶбһҺбһ¶бһ„бһҮбһ¶ JSON ХҜХЎХҙ XML бһҹбҹ’бһҠбһ„бҹӢбһҠбһ¶бһҡбҹ”

FHIR organises health data into **Resources** вҖ” modular, self-contained data objects. Each resource has a standard JSON/XML representation.

| бһ’бһ“бһ’бһ¶бһ“ (Resource) | бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸбһҠбҹӮбһӣбһ•бҹ’бһ‘бһ»бһҖ (Contains) |
|:---------|:---------|
| **Patient** | бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ”бҹ’бһҡбһҮбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ, бһӣбҹҒбһҒбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһўбҹ’бһ“бһҖбһҮбҹҶбһ„бһә, бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„ (Demographics, identifiers, contact info) |
| **Practitioner** | бһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺбһҡбһ”бһҹбҹӢбһңбҹҒбһҮбҹ’бһҮбһ”бһҺбҹ’бһҢбһ·бһҸ бһ¬бһӮбһ·бһӣбһ¶бһ“бһ»бһ”бһҠбҹ’бһӢбһ¶бһҖ бһ“бһ·бһ„бһҹбһүбҹ’бһүбһ¶бһ”бһҸбҹ’бһҡбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– (Doctor/nurse identity & credentials) |
| **Encounter** | бһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһӮбҹ’бһӣбһёбһ“бһ·бһҖ (бһҖбһ¶бһҡбһҺбһ¶бһҸбҹӢбһҮбһҪбһ”, бһҖбһ¶бһҡбһ…бһјбһӣбһҮбһҪбһ”бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҮбҹҶбһ„бһә) (Clinical interaction вҖ” appointment, visit) |
| **Observation** | бһҹбһүбҹ’бһүбһ¶бһҮбһёбһңбһ·бһҸ, бһӣбһ‘бҹ’бһ’бһ•бһӣбһҳбһ“бҹ’бһ‘бһёбһҡбһ–бһ·бһҹбҹ„бһ’бһ“бҹҚ, бһҖбһ¶бһҡбһңбһ¶бһҹбҹӢбһңбҹӮбһ„бһҡбһ¶бһ„бһҖбһ¶бһҷ (Vitals, lab results, measurements) |
| **Condition** | бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҡбҹ„бһӮбһңбһ·бһ“бһ·бһ…бҹ’бһҶбҹҗбһҷ, бһ”бһүбҹ’бһ бһ¶бһҹбһ»бһҒбһ—бһ¶бһ– (Diagnoses, problems) |
| **MedicationRequest** | бһңбҹҒбһҮбҹ’бһҮбһ”бһүбҹ’бһҮбһ¶ (Prescriptions) |
| **AllergyIntolerance** | бһўбһ¶бһЎбҹӮбһ бҹ’бһҹбҹҠбһё бһ“бһ·бһ„бһ”бҹ’бһҡбһҸбһ·бһҖбһҳбҹ’бһҳбһ•бҹ’бһҹбҹҒбһ„ бҹ— (Allergies and adverse reactions) |
| **DiagnosticReport** | бһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһҳбһ“бҹ’бһ‘бһёбһҡбһ–бһ·бһҹбҹ„бһ’бһ“бҹҚ, бһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһҡбһјбһ”бһ—бһ¶бһ–бһңбҹҒбһҮбҹ’бһҮбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ (Lab reports, imaging reports) |
| **Procedure** | бһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһ”бһ¶бһӣ бһ“бһ·бһ„бһ“бһёбһҸбһ·бһңбһ·бһ’бһёбһңбҹҒбһҮбҹ’бһҮбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹӮбһӣбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸ (Treatments and procedures performed) |
| **Immunization** | бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ…бһ¶бһҖбҹӢбһңбҹүбһ¶бһҖбҹӢбһҹбһ¶бҹҶбһ„ (Vaccination records) |

```
  вҶ’ бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ EHR бһ”бһ„бҹ’бһ бһ¶бһүбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһҹбһ»бҹҶбһҖбһ¶бһҡбһҷбһӣбҹӢбһ–бҹ’бһҡбһҳбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһўбҹ’бһ“бһҖбһҮбҹҶбһ„бһә
  вҶ’ бһўбҹ’бһ“бһҖбһҮбҹҶбһ„бһәбһўбһ“бһ»бһҳбҹҗбһҸбһҷбһӣбҹӢбһ–бҹ’бһҡбһҳбһӣбһҫбһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ
  вҶ’ бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбһјбһҠ OAuth2 Access Token бһҠбҹӮбһӣбһҳбһ¶бһ“бһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһҹбһ·бһ‘бҹ’бһ’бһ·бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ
  вҶ’ бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ бҹ…бһ‘бҹ…бһҖбһ¶бһ“бҹӢ FHIR API бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Access Token бһ“бҹ„бҹҮ
```

бһ…бҹҶбһҺбһ»бһ…бһ“бҹҒбҹҮбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбһ»бһҒбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһ—бһ¶бһӮбһёбһ‘бһёбһ”бһё бһўбһ¶бһ…бһ…бһјбһӣбһҳбһҫбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбҹ’бһ“бһҖбһҮбҹҶбһ„бһәбһ–бһёбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ EHR (бһҠбһјбһ…бһҮбһ¶ Epic, Cerner) бһҠбҹ„бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһҷбһӣбҹӢбһ–бҹ’бһҡбһҳбһ–бһёбһўбҹ’бһ“бһҖбһҮбҹҶбһ„бһә вҖ” бһҠбҹӮбһӣбһҮбһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӮбҹ’бһҡбһ№бҹҮбһ“бҹғбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҒбһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһ»бһҒбһ—бһ¶бһ– (health data portability)бҹ”

This allows third-party health apps to access patient data from EHRs with patient consent вҖ” the foundation of health data portability.

## бҹҰ. бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҸбһҳбҹ’бһҡбһјбһңбһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ FHIR APIs (Security Requirements for FHIR APIs)

```
в–Ў бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ OAuth2 / SMART on FHIR бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺ (OAuth2 authentication)
в–Ў бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ TLS 1.2+ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһ¶бһӣбҹӢбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһјбһ“бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҸбһ¶бһҳбһҡбһҷбҹҲ FHIR API (TLS 1.2+)
в–Ў бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҹбһңбһ“бһҖбһҳбҹ’бһҳ вҖ” бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҡбһ¶бһӣбҹӢбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ’бһ“бһ’бһ¶бһ“ (бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ’бһ“бһ’бһ¶бһ“ AuditEvent)
в–Ў бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳбһҖбһ¶бһҡбһҷбһӣбҹӢбһ–бҹ’бһҡбһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбҹҶбһ„бһә вҖ” бһ•бҹ’бһҸбһӣбҹӢбһҮбһјбһ“бһҸбҹӮбһ’бһ“бһ’бһ¶бһ“бһҠбҹӮбһӣбһ”бһ¶бһ“бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ (Consent enforcement)
в–Ў бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһўбһҸбҹ’бһҡбһ¶бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ API (Rate Limiting) вҖ” бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһҠбһ»бҹҶбһ’бҹҶ (Rate limiting)
в–Ў бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһұбҹ’бһҷбһ“бҹ…бһҖбһҳбҹ’бһҡбһ·бһҸбһўбһ”бҹ’бһ”бһ”бһҡбһҳбһ¶ (Data minimisation)
в–Ў бһҮбһҳбҹ’бһҡбһҫбһҹбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһўбһ¶бһҹбһ“бҹ’бһ“ (Break-glass access) вҖ” бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶ logs бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ
в–Ў бһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢ FHIR (Capability Statement) вҖ” бһ…бһ„бһҖбҹ’бһҡбһ„бһҜбһҖбһҹбһ¶бһҡбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһ–бһёбһўбҹ’бһңбһёбһҠбҹӮбһӣ server бһӮбһ¶бҹҶбһ‘бҹ’бһҡ
```

## бҹ§. бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„ FHIR бһ“бһ·бһ„ HL7 v2 (FHIR vs HL7 v2)

| бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ– (Aspect) | бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ FHIR | бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ HL7 v2 |
|:-------|:-----|:-------|
| **бһ‘бһҳбҹ’бһҡбһ„бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Format)** | JSON / XML / Turtle | бһўбһҸбҹ’бһҗбһ”бһ‘бһҒбһҺбҹ’бһҢбһҠбҹ„бһҷбһҹбһүбҹ’бһүбһ¶бһ”бҹҶбһ–бһ„бҹӢ Pipe (`MSH\|^~\&\|...`) |
| **бһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢ API** | RESTful HTTP | бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһјбһ“бһҜбһҖбһҹбһ¶бһҡ / бһ•бҹ’бһүбһҫбһҹбһ¶бһҡ point-to-point (File transfer) |
| **бһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһўбһ¶бһ“** | бһҳбһ“бһ»бһҹбҹ’бһҹбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһўбһ¶бһ“ бһ“бһ·бһ„бһҷбһӣбҹӢ (Human-readable) | бһ–бһ·бһ”бһ¶бһҖбһҷбһӣбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһҮбҹҶбһ“бһ¶бһүбһҖбһ¶бһҡ (Cryptic) |
| **бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“** | бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҗбҹ’бһҳбһё бҹ— (New implementations) | бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҳбһ“бҹ’бһ‘бһёбһҡбһ–бҹҒбһ‘бҹ’бһҷбһ…бһ¶бһҹбҹӢ бҹ— (бһ“бҹ…бһҸбҹӮбһ–бҹҒбһүбһ“бһ·бһҷбһҳбһҒбҹ’бһӣбһ¶бҹҶбһ„) (Legacy) |
| **бһўбһ“бҹ’бһҸбһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ** | бһӣбҹ’бһўбһҘбһҸбһҒбҹ’бһ…бҹ„бҹҮбһҮбһ¶бһҳбһҪбһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ‘бҹҶбһ“бһҫбһ” бҹ— | бһ‘бһ¶бһҳбһ‘бһ¶бһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбҹҶбһ“бһҪбһҷбһ”бҹҶбһ”бҹ’бһӣбҹӮбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (бһҠбһјбһ…бһҮбһ¶ Mirth, Rhapsody) |

бһңбҹҒбһ‘бһ·бһҖбһ¶бһҗбҹӮбһ‘бһ¶бҹҶбһҹбһ»бһҒбһ—бһ¶бһ–бһҗбҹ’бһҳбһё бҹ— бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ FHIRбҹ” бһ‘бҹ„бҹҮбһҮбһ¶бһҷбҹүбһ¶бһ„бһҺбһ¶ бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҳбһ“бҹ’бһ‘бһёбһҡбһ–бҹҒбһ‘бҹ’бһҷбһ…бһ¶бһҹбҹӢ бҹ— бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһ“бҹ…бһҸбҹӮбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ HL7 v2 бһҠбһҠбҹӮбһӣбҹ” бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷбһҳбһ“бҹ’бһ‘бһёбһҡбһ–бҹҒбһ‘бҹ’бһҷбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҸбһҳбҹ’бһҡбһјбһңбһұбҹ’бһҷбһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ“бҹҒбҹҮбҹ”

Most new healthcare platforms use FHIR. Most existing hospital systems still use HL7 v2. Integrating with hospitals often means supporting both.
